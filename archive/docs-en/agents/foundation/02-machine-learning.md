## I. Essential Definition

> Machine learning is the process of setting a specific cognitive goal, allowing the machine to deduce independently through massive amounts of data, and align with this target cognition.
> — Brother Sheng's verified expression, 2026.03.01

> Machine learning is the way AI achieves intelligence. By feeding massive amounts of data, the machine continuously trials, induces, and extracts patterns to achieve expected cognition or goals. — Brother Sheng's verified expression, 2026.03.03

**Technical Language Comparison**:
Instead of humans writing rules for AI, AI induces patterns from data by itself.
This "self-induction" is the most essential step in machine learning.

## II. Three Learning Methods

| Method | Essence | Life Analogy | Yijing Correspondence |
|--------|---------|---------------|------------------------|
| Supervised Learning | Give data + answers, learn correspondence | Teacher sets questions with standard answers, students learn by comparison | Learning with a teacher, following rules |
| Unsupervised Learning | Only give data, classify and induce by itself | Give a pile of things to a child, let them categorize on their own | Self-taught, investigating things to gain knowledge |
| Reinforcement Learning | Give feedback, reward for correct actions, punish for mistakes | Learning to walk, adjust when falling, continue when walking steadily | Yao movement — adjust according to time and position |
