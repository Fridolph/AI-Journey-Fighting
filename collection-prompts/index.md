### 分析原则

这些内容由模型根据用户最后输入自动决定，不需要用户额外提供。

如果是城市方向：
自动补充该城市相关的别称、历史定位、文化标签、地理特征、代表性建筑、名胜注释、人文亮点、城市精神、典型符号、简明知识点等。

如果是对象方向：
自动补充其功能、材料、结构、来源、用途、文化背景、代表性特征、技术点等。

如果是抽象主题或灵感方向：
自动补充其概念提炼、风格关键词、象征含义、背景知识、延展性亮点、联想线索等。

这些知识点不要机械罗列成说明书，不要一股脑堆满边角。
必须以设计性的方式拆分、分组、穿插、整合、排布，成为画面的一部分。
它们应围绕对角线主体自然分布在四周留白区中，形成既有信息性、又有装饰性的版式结构。
四周信息应采用“避让式排版”，围绕主体边缘自然展开，形成环绕、陪衬、补充关系，而不是侵入主体中心通道。
信息块之间要有疏密变化和呼吸感，做到丰富但不挤。

七、署名
在画面右下角保留一个极小号署名字样：
“NAICHA DESIGN”
署名字体要克制、干净、低调，不喧宾夺主。

八、整体气质目标
整张海报应呈现以下综合观感：
主体舒展、版面开阔、对角线主结构完整铺开；
留白高级且有呼吸感；
信息丰富但不杂乱；
远看有鲜明主视觉与强烈记忆点；
近看有丰富的知识点与排版细节；
兼具绘画性、设计性、信息性、阅读性与收藏感；
像一张高端展览海报、城市视觉海报、设计年鉴封面或艺术书籍内页级别的作品。

九、避免的问题
不要把中间图示区域缩成一坨；
不要让主体面积过小；
不要让对角线主图示被大字或信息模块拦腰切断；
不要让主体中段被标题横插打断；
不要让拼音 / 英文大字像路障一样压断主体；
不要让版面显得狭隘、逼仄、拘谨；
不要模板化旅游宣传页感；
不要低级拼贴感；
不要普通地图截图感；
不要说明书式排版；
不要廉价插画感；
不要满版塞满元素；
不要让文字沦为附属说明；
不要让材质大字显得俗气、死板、单调或商业化过强。

十、最终目标
要让“沿对角线大幅展开的综合色带主体 + 极简俯视元素 / 图示系统 + 具有烫印 / 金属材质感且颜色灵活的拼音 / 英文大字 + 四周知识信息 + 高级衬线排版 + 大面积留白”共同形成一张完整而高级的作品。

最终让整张图：
远看有强烈主视觉冲击，
近看有丰富阅读内容与设计细节，
主体不断，文字不截流，版面大气，排版高级，
整体具有收藏价值、作者性与大师级审美完成度。

用户最终输入：
【城市 / 主题 / 对象 / 灵感】：_____
尺寸要求：_____ （比如3:4）
 
6.进化史信息
你要生成的不是普通插画，也不是把信息机械排成一列的时间轴，而是一张“高完成度、强组织性、强阅读价值、兼具知识含量与高级审美的进程演化信息图海报”。

核心任务：
用户只会提供一个主题、概念、命题、对象、类别，或一个需要被梳理的方向。你需要自动理解这个主题背后所包含的“演化过程 / 发展进程 / 时间序列 / 阶段变化 / 版本更替 / 文明推进 / 形态变迁 / 谱系延展 / 历史流变”，并把它转译成一张结构清晰、图文一体、具有专业设计感的流程化信息图。

这张图必须具备如下能力：
当用户只输入一个主题时，你要自动沿着该主题延展出完整的阶段结构、关键节点、代表性视觉对象、时间分段、核心说明与整体叙事逻辑。不要要求用户提供大量细节。你要主动补全最合理、最有知识性、最有可视化价值的内容，并将其转化为一张完整、优雅、可信、耐看的信息图。

适用范围必须非常广：
可以用于文明史、产品演化史、技术发展史、朝代更替、帝王谱系、物种进化、器物演变、服饰演变、饮食演变、品牌发展、某一物件的形态变迁、某一制度或思想的发展过程、某一行业的阶段性变化，或任何可以被整理成“阶段推进”的主题。
无论用户输入的是严肃学术主题，还是日常消费品主题，还是带趣味性的生活主题，你都要以同样专业、优雅、清晰、富有美感的方式，把它梳理成一张完整的信息图。

请严格遵守以下创作原则：

第一原则：先理解主题，再决定进程结构
在开始生成前，你必须先智能判断这个主题最适合被梳理成什么样的“演进结构”。
如果它是历史主题，就梳理其年代推进、关键时期、代表节点与典型物证。
如果它是产品品类，就梳理其形态、结构、材料、工艺、使用方式、审美风格、社会功能的演化。
如果它是生物或自然主题，就梳理其进化链条、关键分支、典型形态与核心特征。
如果它是抽象命题，就梳理它在时间、认知、制度、媒介、社会语境中的发展阶段。
总之，你必须自动找到这个主题最适合被呈现为“进程”的逻辑，而不是机械套模板。

第二原则：画面不是简单时间轴，而是“被空间化的进程”
整体构图优先采用具有强烈秩序感和视觉记忆点的“递进式台阶 / 阶梯式平台 / 斜向推进路径 / 分层式结构 / 展陈式轨迹 / 版块式进化路径”。
这个结构要像一条被实体化的演进路径，带有明确的起点、阶段节点、递进节奏与终点。
路径不应只是画一条线，而应具有平台感、层级感、空间感，让每个阶段都像站在一个清晰的知识台阶或展陈单元上。
但不要固化为同一种造型。你必须根据主题决定更适合用“阶梯、坡道、时间地层、谱系展开、版本阶层、器物陈列台、年代阶板、进化路径”等哪一种形式。重点是强组织性，而不是形式重复。

第三原则：信息图必须兼具知识性与审美性
这不是普通科普图，也不是纯装饰海报，而是一张“有内容密度、有知识价值、有设计美感”的高级信息图。
观者看完应该有“学到了东西”的满足感，同时也被优雅的组织方式打动。
画面必须体现出：内容被高度梳理、信息被优雅安放、视觉被克制组织、阅读路径非常清楚。

第四原则：整体视觉语言要高级、克制、优雅、专业
画面审美参考高端展览信息设计、博物馆展板、视觉百科、现代知识型海报、编辑设计系统、精致图录。
整体风格必须简洁、沉稳、理性、克制，不要网红化，不要廉价卡通化，不要花哨拼贴，不要过度装饰，不要俗气配色。
即使主题本身很日常，也必须被提升成一种优雅、专业、可收藏的视觉呈现。

第五原则：视觉方案必须根据主题智能变化，不能千篇一律
不要让所有主题都长得一样。
你需要根据用户输入的主题，智能决定：
应该强调年代，还是强调结构变化；
应该突出实物，还是突出人物、器官、服装、工具、器皿、产品、图标、轮廓、谱系分支；
应该更偏博物馆考古感，还是更偏现代工业设计感，还是更偏自然科学图鉴感，还是更偏时尚编辑感；
应该采用更严谨的纪年表达，还是阶段命名表达，还是世代版本表达；
应该做得厚重、理性、冷静，还是轻盈、诗性、柔和，还是锐利、现代、未来感。
每次都要重新建立视觉规则，不允许形成固定套路。

第六原则：配色必须真正跟随主题，不允许固定色板
配色必须根据主题语义、情绪气质、时代属性、材料属性、文化背景、行业特征、使用场景、用户指定偏好来智能决定。
绝对不要默认永远使用同一种暖棕、米白、博物馆色调。只有当主题本身确实适合这种气质时，才可以使用。
你必须为每个主题重新思考以下问题：
这个主题更适合浅色模式还是深色模式；
背景应该是纸感、石感、雾感、金属感、科技感、纤维感、自然感、夜色感还是极简纯色感；
主色应该传达什么气质；
辅助色如何区分阶段；
强调色应该突出哪些关键节点；
整体色域是克制单色系、低饱和多色系、冷暖对照系，还是高端中性色加一枚强强调色。
配色必须体现“因题而变”的智能性。

第七原则：深色模式与浅色模式要有明确判断逻辑
如果用户指定深色或浅色模式，必须严格遵守。
如果用户没有指定，你要根据主题智能判断：
偏考古、文献、教育、器物、历史、自然图录、生活方式、服饰演变等，通常更适合浅底高级配色，但不是绝对；
偏科技、宇宙、未来、电子、夜间场景、数字系统、赛博工业、深海、生物解剖等，通常更适合深色模式，但仍要以主题匹配为准；
偏时尚、奢侈品、品牌演化、潮流器物，也可能采用深浅双系统中的任一种，但必须有明确设计理由。
无论深色还是浅色，信息都必须清楚、层级明确、对比得当、阅读舒适。
背景色必须参与叙事，而不是简单填充。

第八原则：建立“主题驱动的配色系统”
每张图都必须构建一个完整的配色系统，而不是随便选几个颜色。
这个系统通常包括：
背景色；
结构色；
主标题色；
副标题与说明文字色；
阶段标签色；
时间标签色；
强调节点色；
引线或注释系统色；
必要时的分组色。
颜色之间必须有逻辑、有层级、有秩序。
可以控制在少量精炼颜色中完成，也可以做低饱和层次扩展，但必须高级、耐看、统一。
不要颜色过多，不要脏，不要杂，不要俗。
对于不同主题，应尽量呈现出不同的色彩人格，让人一看就知道这是为这个主题专门定制的，而不是拿模板换词。

第九原则：字体与排版不能过于中规中矩，要有“克制的创新”
文字排版不能只是把标题居中放上去、把说明整齐排成一列。你需要在保持清晰可读的前提下，进行有分寸的创新。
可以做的创新包括但不限于：
主标题与副标题形成更有节奏的比例关系；
主标题可以采用更有姿态的断行、错位、穿插、留白、纵横组合、局部叠压、局部嵌入结构；
大年份、大数字、大阶段编号可以成为画面的一部分，而不只是说明文字；
部分文字可以顺应路径、平台、引线、空间方向排布；
文字块可以像展签、目录条、标签、铭牌、注释条、信息胶囊那样被整合进画面；
中英文、数字、符号、年代信息之间可以有层级节奏，而不是一律同质；
字距、行距、对齐方式、留白、大小反差要体现编辑设计意识。
但这种创新必须是“微妙而聪明”的，不是哗众取宠。不能为了设计感而损害清晰度与可读性。

第十原则：字体气质必须根据主题智能匹配
如果主题偏历史、文明、考古、传统文化，可采用更有文化气息、展览气质、碑帖感或书卷气的标题处理，但仍需现代设计化，不可陈旧土气。
如果主题偏科技、工业、未来、产品演化，可采用更利落、理性、现代、精密的字体气质。
如果主题偏时尚、生活方式、消费品、服饰，可采用更编辑化、更时尚、更克制的字形秩序。
如果主题偏自然科学、图鉴、百科，可采用更清晰、严谨、学术化但不呆板的排版风格。
总之，字体是内容气质的一部分，不可以永远是同一种感觉。

第十一原则：版式结构必须非常强
整张图必须有清晰、成熟、专业的排版组织。建议包含但不限于以下模块：
顶部主标题：直接点明用户主题；
副标题：对主题的进程逻辑做简洁概括，例如“演化路径 / 发展脉络 / 进程总览 / 形态变迁 / 文明推进 / 谱系展开 / 时间纵览”等，但具体措辞要根据主题智能匹配；
主体路径：以台阶、层级平台、斜向递进结构或其他空间化系统承载时间与阶段；
阶段节点：每个阶段要清晰可辨，有对应的时间标记、阶段名称、代表性对象与简短说明；
注释系统：通过引线、点位、标签、小标题、年份、小注、区域说明等方式，把信息优雅展开；
底部或末端收束：以终点、当代阶段、总结性阶段或未来延展节点形成完整闭环。

第十二原则：时间与阶段必须清楚
如果主题适合做年代型信息图，就要明确标出关键年份、世纪、年代段、时代名称或阶段范围。
如果主题不需要精确年份，也要以“阶段一、阶段二、阶段三”这种清晰的结构来组织。
任何情况下都要让观者能看出进程的先后顺序与演变关系。
阶段数量通常控制在 5 到 12 个左右，具体根据主题复杂度智能调整。不要太少而显得空洞，也不要过多而拥挤失控。

第十三原则：每个阶段都要有“代表性视觉对象”
每个阶段应配一个或多个最具辨识度、最能说明该阶段特征的视觉对象。
这些对象可以是器物、产品、人物形象、服饰形态、化石、物种形态、结构剖面、工具、文物、轮廓图、标本、部件、容器、造型特征等。
对象必须足够典型，能让观众通过图像直观感知“这个阶段的差异是什么”。
每个对象都应尽量精致、清晰、质感良好，像被 carefully curated 的展品一样陈列在对应阶段。

第十四原则：文本说明要少而准
文字不是越多越好，而是要精准、简洁、清晰、有知识点。
每个阶段附近可以放：
阶段标题；
时间标签；
一句话特征说明；
必要时加一个更小的补充说明。
文字必须简练，不堆砌，不长篇大论，但要让人感觉专业而有收获。
说明内容优先提炼“该阶段最本质的变化、代表意义、标志特征、关键突破、社会用途、审美转折、技术变化”等。

第十五原则：图文关系要真正融合
所有文字、对象、节点、引线、标签、时间标记、阶段说明，都必须自然依附于主路径结构。
它们应共同组成一个整体，而不是“背景一张图，上面再贴几行字”。
图与文应该彼此解释、彼此支撑、彼此增强，让整张图成为一个完整的知识可视化系统。

第十六原则：阅读体验必须层层递进
观众第一眼先看到主标题与整体结构；
第二眼看到清晰的进程路径；
第三眼看到不同阶段的代表性对象；
第四眼逐步阅读每个阶段的说明；
第五眼能通过细节标签获得更细的知识点。
这张图必须让人感到“结构非常好”“被整理得非常舒服”“看起来很高级”“内容很值得看”。

第十七原则：重点避免的问题
不要生成普通 PPT 感；
不要做成廉价百科图；
不要把所有主题都做成相同的米白棕色模板；
不要颜色逻辑僵化；
不要把标题和文字排得过于保守、缺乏呼吸与层次；
不要使用杂乱无章的箭头堆叠；
不要把所有信息铺满整个页面导致失去呼吸感；
不要把图像和文字割裂成两个系统；
不要为了装饰而装饰；
不要出现与主题无关的花纹、图标或填充元素；
不要过度卡通化；
不要颜色过多或过艳；
不要让阶段之间差异不明显；
不要让观众看不出进程方向；
不要让信息图失去“优雅”“可信度”“可读性”。

第十八原则：智能细化内容
当用户输入主题后，请你自动完成以下工作并反映到画面中：
判断这个主题最合理的起点与终点；
梳理其中最关键的阶段；
为每个阶段匹配最有辨识度的代表性视觉对象；
总结每个阶段的核心变化；
决定整体更适合深色还是浅色模式；
建立一套与主题匹配的配色系统；
决定整体更适合什么样的字体气质与排版节奏；
决定整体适合用什么样的副标题与注释语言；
控制信息密度与视觉密度；
自动让图面既丰富又整洁，既有知识价值又有设计感。
你必须保证：换一个主题，就像换了一套专属视觉系统，而不是只换内容不换灵魂。

最终画面应该像一张“高端、优雅、极具知识含量、图文深度融合、配色与字体都因主题而生”的演化进程信息图海报。

7.博物馆国宝/文物解读图鉴
本次不是文本任务， 是生成图片任务，要求如下：
生成图片任务：
你要生成的不是普通插画，也不是标准化的信息板，而是一张“经过联网核实、知识准确、图文一体、兼具国际级海报设计水准、博物馆出版物气质、东方审美、研究感与强视觉记忆点的文物知识海报 / 文物美学卡片 / 展览级单页”。

你的目标不是把资料堆上去，而是把一件文物真正“设计出来、讲透出来、研究出来”。它必须同时满足以下标准：知识准确，不凭空编造；视觉大气，有留白，有巧思；排版灵动，不僵硬，不套路化；图文关系高级，不是图片加说明；第一眼就抓人，越看越有内容；既像顶级展览海报，也像一页被精心设计过的博物馆研究图录。

在正式生成之前，必须先执行联网检索与事实核验。优先检索博物馆官网、文博机构官网、官方展览资料、权威百科、考古报告、学术文章、出版图录等高可信来源。重点核实正式名称、别名、年代、朝代或时期、器类、材质、工艺、尺寸、重量、出土地、发现背景、现藏机构、用途、造型特征、纹饰特征、铭文信息、文化寓意、历史价值、艺术价值、工艺价值、学术意义、代表性细节、相关典故或文献出处、归属与流传等信息。若不同来源存在差异，应优先采用更权威、更广泛认可的说法，并对存疑内容谨慎表述，不得凭空编造。最终画面中的所有知识文字都必须建立在检索与判断基础上，而不是靠想象补写。

整张海报的信息量必须明显强于普通博物馆展签，至少包含十个不同层级的信息点，而且每个层级都要有真实的信息增量。除基础数据外，应尽量涵盖挖掘背景、考古过程、流传与修复经历、文献典故或出处、器物归属、时代背景、礼制与社会意义、工艺与结构分析、纹饰与造型解读、与同类器物的比较、地域文化特征、学术研究价值、展览与收藏意义、公众传播影响、历史趋势与文明脉络、价值总结等维度。观者读完后要有一种“比听讲解更透、更准、更有收获”的感觉。

整体版式必须彻底摆脱教科书式、表格式、规则分栏式的信息图逻辑。画面要更有灵气、更有节奏、更有松弛感，更像国际知名海报设计师为一件文物量身打造的视觉作品。必须具备强烈的整体构图意识、前景与后景的穿插关系、几何图形的设计感、留白的控制力，以及图文之间的巧妙遮挡与共生关系。

核心构图建议采用“主体文物 + 几何背景结构 + 嵌入式超大文字 + 一条完整的主笔刷动线 + 周边研究信息模块”的方式。主体文物是前景核心，几何图形是支撑整体版面的背景结构，大字嵌在几何图形之中，笔刷则负责建立画面的气脉与动态节奏。整张图要形成清晰的前景、中景、背景三层关系，彼此叠压、遮挡、呼应。

文物名称的最大文字不要再出现在画面顶部，也不要以传统标题方式直白地摆在那里。最大的文物名称，应当转化成一种“围绕文物而存在的背景结构文字”。做法是：将文物名称中的关键字拆解成超大号背景字，嵌入几何图形之中，让这些大字环绕在文物周边，部分被文物遮挡，部分被几何图形切割，部分被笔刷穿过，只露出一部分轮廓或局部笔画，让观者能够通过这些碎片化的大字去猜测文物名称。每一个字都必须被保留下来，绝对不能漏掉任何一个字。即使允许遮挡，也必须保证每个字都至少有足够清晰的一部分被看见，使观者仍能辨认或推断出该字，保证文字的可理解性与完整性。不能出现明明是四个字却少一个字、或者某个字被吞掉到完全认不出的情况。背景大字应有“局部遮挡但整体可读”的效果，而不是“为了形式感牺牲识别度”。真正完整、清晰的文物全名，还必须在比超大背景字小一级的文字层中被巧妙地再次体现一次，例如放在副标题、小标签、角落题签、信息卡标题、器物说明、边注信息或基础信息区中，用一种低调但清楚的方式明确写出完整文物名称，方便不熟悉该文物的观者快速知道它到底叫什么。也就是说，超大字负责气势与记忆点，次一级标题或说明负责完整识别，两者必须互相配合。

几何图形必须成为画面中的重要设计语言。可以使用矩形、弧形、圆形、方块、条带、框形、弧面切片、结构块面等作为背景承托物，把超大文字嵌入其中。几何图形不仅用于装饰，更用于组织空间、建立层次、承载大字、调节留白、衬托主体。几何图形的边界可以与文物轮廓发生压叠关系，也可以为局部图和信息模块提供落点。整体效果要体现一种现代平面设计与东方文物气质融合的感觉：理性结构与文化气韵并存。

画面中仍然需要那条完整而有势的主笔刷，但它要和几何背景、大字嵌入式结构融合在一起，而不是孤立存在。笔刷应像一条河流、一笔刷过的气脉，从画面的某一端自然流向另一端，具有起承转合、浓淡变化、飞白、渗化、轻微晕染与书写节奏。它必须连贯、舒展、自然、有姿态，不能僵硬、断裂、像贴上去的色块。笔刷可以穿过几何图形，扫过背景大字，压住一部分信息模块，也可以轻轻托起主体文物，成为整张画面的视觉动线和情绪动线。

色调不必固定使用浅色背景，而应根据文物自身的气质、材质、年代感与视觉重量来智能判断。可以使用浅背景，也可以使用深背景，但必须保证背景与文物主体之间有清晰而高级的层次关系。如果文物较深，可以使用更浅、更能托起主体的底色，也可以使用同色系中更特别、更透气的一种色相。如果文物较浅，可以使用更沉稳、更有包裹感的背景来衬托。整体配色要结合几何图形进行灵活处理，允许深浅对比、冷暖偏移、材质对照、矿物色延展、综合色协调，但必须统一、高级、克制，不可土气，不可僵硬，不可把背景、文物、笔刷和大字糊成一片。

色彩必须遵循“色从器出，但不死守器色”的原则。要从文物本身提炼核心色，再进行更聪明的延展、偏移、互补与层次拆分。背景、几何图形、笔刷、嵌入式大字、信息模块和主体文物之间，既要有统一关系，也要有明确区分。可以根据文物类型建立不同的综合色系统。青铜器可延展青铜绿、古铜褐、烟灰、暗金、松石灰、黛青、墨蓝灰等；瓷器可延展月白、釉青、青花蓝、米釉、暖灰白、浅矿物色等；玉器可延展脂白、青白、淡灰、温润浅青等；漆器、织物、佛造像、画像石、铜镜等也应依据材质和时代气质决定综合色逻辑。目标不是炫彩，而是建立一套真正高级、耐看、富有审美判断的系统。

背景中的超大文字不要机械地默认用烫金，也不要一律使用同一款字体。字体的选择必须跟着文物的气质走，做到真正的智能判断。不同类型的文物，应当匹配不同的字形性格和视觉语言。青铜器可以偏金石感、碑刻感、篆隶气、厚重铸铭气；铜镜可偏古雅、镜铭感、细致而有历史气息；瓷器可偏秀润、清雅、洁净而有流线感；佛像类可偏庄严、含蓄、静穆、有宗教气息；玉器可偏温润、克制、留白感强；漆器、织物、画像石、碑刻、书画类也都应根据文物的材质、用途、时代精神与文化属性来判断。可以在碑帖感、题签感、篆意、隶意、楷意、行书感、手写感、现代展览设计字之间灵活选择，但必须统一于文物气质，而不是模板化地使用同一款所谓高级字体。

是否使用金色，必须由文物本身决定，而不是预设。可以完全不用烫金，也可以只在小范围点到为止。标题和背景大字可以采用低饱和纯色、矿物色、压印感、朴素描边、留白、浅墨、柔和铜色、淡灰褐色、局部磨损感、轻微浮雕感、局部色块承托等多种方式来呈现。目标是让字成为文物精神的延伸，而不是影响整体美学观感的多余装饰。

主体文物必须保持高质量、清晰、完整和真实依据，但不应只是冷冰冰的产品图式摆拍。要在真实基础上融入适度的绘画化、图录插绘化、工笔设色化或研究图化处理，使其自然嵌入整体设计语言。主体文物应当像从几何结构、大字背景和笔刷动线中“生长出来”，而不是简单贴在版面中间。其边缘、压叠关系、阴影、与背景字的遮挡关系，都要处理得非常讲究，体现前景感和层次感。

信息模块必须更灵活，不要再整齐排成说明书。基础数据、挖掘背景、流传修复、文献典故、归属与礼制、工艺分析、纹饰观察、时代趋势、学术价值、展览意义、价值总结等，可以围绕主体与几何背景自由分布，形成疏密有致的阅读节奏。可以采用边栏式信息、研究卡片式小块、悬浮标签、题签、细线引导、编号系统、半透明叠层、边角批注、局部框选等方式。重要信息块更突出，次级信息更轻、更小、更克制。不要平均分配信息，要让版面有呼吸、有主次、有节奏。

局部细节观察区必须紧密围绕主图展开。可以安排纹饰放大图、器耳器足特写、结构分析图、线稿研究图、局部描摹图、铭文局部或工艺拆解图。这些细节图必须说明“为什么这个地方重要”，而不是只做装饰。它们可以压在几何图形或背景大字之上，也可以被笔刷轻轻带过，与主体形成自然的研究关系和视觉联系。

正文与图注系统必须清晰可读，但不能全部机械统一。可以让主要说明文字保持工整、克制、可读性高，同时让部分旁注、关键词、局部解说、观察提示、学术判断、研究批注具有更轻微的手写感、批注感或研究手记感。整体效果应像一页真正被设计师与研究者共同打磨过的图录页：正文有秩序，旁注有人的温度。

留白必须极其讲究。要像顶级海报设计那样，大胆留白，敢于空出空间，让主体、几何图形、大字、笔刷和信息块都能呼吸。留白不是浪费，而是画面气场的一部分。整个版面要有大片安静区域，也要有局部信息密度较高的区域，通过疏密对比建立真正高级的节奏感与大气感。

整体阅读路径必须自然。观者应先被文物主体、周边嵌入的巨型结构字和几何图形吸引，再顺着笔刷与视觉流线进入关键信息模块，然后通过局部观察图和研究批注逐层深入，最后读到更深层的背景知识与价值总结。整个过程要像被一位非常聪明、非常懂设计、也非常懂文物的策展人带着看懂这件文物，而不是被迫阅读一张课件。

最终生成目标不是把资料罗列出来，而是产出一张真正具备以下特质的高水平作品：知识准确，信息丰厚，至少十个层级的信息点清晰可见；文物名称的最大文字不在顶部，而是围绕文物嵌入几何背景之中，并以部分显露、部分遮挡的方式形成“可猜测”的高级视觉效果；所有背景大字一个都不能丢，必须保持整体可理解性；完整文物名称必须在次一级文字中被巧妙而清楚地再次体现；画面可浅可深，但配色必须智能而高级；几何图形与笔刷共同参与构图；字体选择根据文物气质灵活变化，不再千篇一律；排版大气、留白充足、非常有巧思；整体具备国际知名海报设计师级别的美学素质与排版素质，既震撼、又耐看、又有文化温度。

请严格围绕用户最终输入的对象，先检索，后判断，再提炼，再设计，确保内容准确，排版顶级，配色聪明，字体与文物气质匹配，几何背景与嵌入式大字逻辑清晰，所有大字都可被理解且不会缺字，完整文物名称在次一级文字中清楚出现，笔刷自然流动，留白大气，整体不套路、不重复、不僵硬，真正做到高级、灵动、聪明、有国际级完成度。

8.真实旁观抓拍
【核心目标】

这张图不是摆拍写真，不是广告大片，不是刻意设计的电影剧照，而是像在真实生活中，摄影者恰好从一个正常、合理、公开的观察位置，偶然捕捉到的一瞬间。
整张图需要具备以下气质：
- 自然
- 真实
- 非表演感
- 带距离感
- 有旁观视角
- 有现场感
- 有“刚好看到这一幕”的偶然性
- 像街拍、纪实摄影、通勤记录或生活观察照片
- 让观者产生“我仿佛正站在附近，恰好看见这一幕”的代入感
【内容生成逻辑】
请根据用户最后输入的“拍摄主题 / 要求元素 / 灵感来源”，智能决定：
- 最合适的场景类型
- 人物所处的位置
- 画面中的主次关系
- 前景、中景、背景的层次
- 哪些元素清晰，哪些元素虚化
- 哪些物体可以自然形成遮挡、切割、引导线和旁观感
- 最终整体氛围与色调

所有内容都必须围绕用户输入展开，不能脱离主题乱发挥。

【重要修正：主体不要太远，必须看得清】
这是重点要求，优先级很高：

1. 默认不要生成“大远景人物很小”的画面，除非用户明确要求远景。
2. 主体人物应当足够清晰可辨认，能看清：
- 基本面部轮廓
- 发型
- 穿着
- 手部动作
- 正在做什么
3. 画面景别优先使用：
- 中景
- 中近景
- 半身到膝上构图
- 或包含完整站姿但人物依然足够大的构图
4. 人物在画面中的视觉占比应合理：
- 人物通常占画面高度约 1/3 到 2/3
- 不要小到缩在画面角落
- 不要让观众需要放大后才能看清主体
5. 即使保留旁观感，也要让人物成为清楚的视觉焦点，而不是环境里的一个小点。
6. 如果场景较复杂，应优先保证“主体清楚”，环境作为辅助信息存在。
7. 避免过度宽广的全景镜头，除非用户特别要求“强环境叙事”。

【视角要求】
采用“合理、正常、公开空间中的第三视角观察构图”，重点是“旁观感”和“偶然抓拍感”，不是正面摆拍。

可以理解为：
- 从街边、通道、站台、商场、校园、公共休息区、开放式室内外空间等位置
- 以一种不打扰主体的方式
- 在正常视线高度或略带偏移的角度
- 捕捉人物处于自然状态中的一瞬间

画面不要像人物知道自己正在被拍摄。
人物通常不直视镜头，动作应自然、松弛、生活化。

【镜头语言与取景控制】
为了防止主体太远，请优先使用更适合人物观察的真实镜头语言：
- 更接近中焦或标准焦段的观感
- 避免极端广角导致人物缩小
- 画面像摄影者站在相对合理的观察距离内拍到
- 可以理解为“站得不算远，但也没有贴脸拍”

整体观感更接近：
- 街拍中近景
- 通勤纪实中景
- 生活观察式人物抓拍
而不是遥远的环境大全景。

【构图要求】
整体构图要像真实摄影，而非设计图。
优先体现以下视觉特征：

1. 画面具有明显“旁观感”
- 主体不要机械居中摆拍
- 可以偏左、偏右、偏下，留出呼吸空间
- 构图要像摄影者在真实环境中迅速捕捉，而不是精心摆位
