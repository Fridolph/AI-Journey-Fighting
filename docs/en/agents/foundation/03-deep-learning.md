## I. Essential Definition

> Deep learning is a type (subset) of machine learning. It aims to achieve preset goals, through orderly (logical) deduction during the learning process, progressing layer by layer according to results and correlations, and exhausting all possibilities.
> — Brother Sheng's verified expression, 2026.03.01

> Deep learning takes neural networks as its framework, through signals, activation functions, and other methods, similar to the pipe function in functional programming, passing stage outputs layer by layer, deducing step by step, and ultimately achieving machine learning goals. — Brother Sheng's understanding version, 0303

**Technical Language Comparison**:
Deep learning uses a multi-layer neural network structure,
allowing each layer to process the output of the previous layer.
The deeper you go, the more abstract the extracted features are, and the closer they are to the essence.
It does not rely on artificially set rules, but is induced by the machine itself.

## II. Relationship with Machine Learning

```
AI
└── Machine Learning
    └── Deep Learning  ← Subset of machine learning, but with more layers, goes further
```

Machine learning is the parent, deep learning is the child.
The child goes deeper than the parent.
Depth refers to both the depth of layers and the depth of breaking through human cognitive boundaries.

| Traditional Machine Learning | Deep Learning |
|-------------------------------|---------------|
| 1-2 layers of processing | Dozens or even hundreds of layers |
| Humans design features | Learns to extract features by itself |
| Processes structured data | Can process images, sounds, languages |
