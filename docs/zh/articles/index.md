## 阶段复盘

从已经发出的这些文章回头看，会很明显地看到一条学习主线：

```text
让 LLM 听话做事
  -> 让它做复杂的事
  -> 让它记住事、学会事
  -> 让多个它协作
  -> 让整个系统安全、可控、可观测
  -> 理解它完整的工作机制
```

这也是我后续继续补充 VitePress 文档时，会持续复用的整理方式。
