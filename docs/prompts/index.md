## 按分类浏览

### 创意海报
将文字、概念、诗句、城市主题转译为高级视觉海报。核心特征：语义理解先行 → 极简强概念 → 文字即骨架 → 高级平面设计感。

| 提示词 | 核心能力 | 输入 | 输出 |
|--------|---------|------|------|
| [任意词视觉化](./creative-posters/word-visualization) | 词语语义 → 视觉隐喻 | 任意词汇/短句 | 概念海报 |
| [城市宣传收藏级海报](./creative-posters/city-poster) | 城市元素提炼 → 对角线构图 | 城市/主题名称 | 油画风收藏级海报 |
| [名著经典海报](./creative-posters/classic-literature) | 名著世界 → 剪影叙事 | 名著名称 | 东方手绘/电影海报 |
| [国风唐诗宋词视觉化海报](./creative-posters/poetry-poster) | 诗词意境 → 视觉解析 | 诗句/诗题 | 文学解析海报 |

### 信息图
知识、数据、历史、建筑等信息图形化。核心特征：智能补全内容 → 强组织性结构 → 配色因题而变 → 图文深度融合。

| 提示词 | 核心能力 | 输入 | 输出 |
|--------|---------|------|------|
| [进化史信息图](./infographics/evolution-history) | 主题 → 阶段演进结构 | 任意主题 | 进程演化信息图 |
| [博物馆国宝/文物解读图鉴](./infographics/museum-artifact) | 文物检索+知识+设计 | 文物名称 | 展览级文物知识海报 |
| [科学百科信息图](./infographics/science-encyclopedia) | 百科知识 → 结构化图解 | 人/动物/植物名 | 百科风格信息图 |
| [景点建筑信息图](./infographics/landmark-architecture) | 建筑照片+蓝图标注 | 建筑/景点名 | 技术蓝图叠加图 |

### 人像诊断
基于人像照片进行分析、算命或专业诊断。核心特征：高情商表达 → 多维度分析 → 报告级排版 → 真实可信。

| 提示词 | 核心能力 | 输入 | 输出 |
|--------|---------|------|------|
| [掌纹算命](./portrait-analysis/palm-reading) | 掌纹 → 命运预测 | 手掌照片 | 10+要点分析报告 |
| [面相痣相算命](./portrait-analysis/face-reading) | 面相 → 综合命理 | 面部照片 | 10+要点分析报告 |
| [发型专属优化报告](./portrait-analysis/hair-optimization) | 人像 → 发型推荐 | 人物照片 | 10+发型方案+综合报告 |

### 纪实模拟
模拟真实世界摄影、纪录片、报纸风格。核心特征：真实感>美感 → 旁观视角 → 物理痕迹 → 非表演感。

| 提示词 | 核心能力 | 输入 | 输出 |
|--------|---------|------|------|
| [真实旁观抓拍](./documentary-sim/bystander-snapshot) | 场景描述 → 自然抓拍 | 主题+元素 | 纪实感街拍照片 |
| [伪纪录片老照片](./documentary-sim/mockumentary-photo) | 主题 → 现场证据感 | 主题内容 | 粗粝真实老照片 |
| [报纸头条假新闻](./documentary-sim/fake-news-headline) | 新闻+人脸 → 报纸头版 | 新闻理由+报名 | 复古报纸头版图 |

## 使用说明

- 每个提示词独立页面可直接复制全文使用
- `{{}}` 或 `［］` 标识的为**用户输入变量**，使用前替换为实际内容
- 部分提示词含多个版本（如"无地面版"/"有地面版本"），页面内已分段标注
- 截图栏暂标记为 `—`，后续将补充分类示例图片，方便快速预览效果
