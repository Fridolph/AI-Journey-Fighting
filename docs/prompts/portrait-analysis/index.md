# 人像诊断

基于人像照片进行分析、算命或专业诊断，生成具有报告/海报品质的输出。本分类下的提示词专注于"图像理解 + 专业分析 + 视觉输出"的复合能力。

## 包含提示词

| 编号 | 名称 | 核心能力 |
|------|------|---------|
| 01 | [掌纹算命](./palm-reading) | 掌纹分析、玄学算命报告 |
| 02 | [面相痣相算命](./face-reading) | 面相分析、命运预测报告 |
| 03 | [发型专属优化报告](./hair-optimization) | 发型适配分析与推荐方案 |

## 设计要点

- **高情商表达**：先肯定优势，再给出专业建议
- **多维度分析**：综合多套理论体系交叉验证
- **报告级排版**：海报级的视觉呈现，适合分享传播
- **真实可信**：分析要有依据，评分要明确，不可空泛
