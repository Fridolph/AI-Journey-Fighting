# 创意海报

将文字、概念、诗句、城市主题转译为具有高级设计感的视觉海报作品。本分类下的提示词专注于"从抽象到具象"的视觉转译能力。

## 包含提示词

| 编号 | 名称 | 核心能力 |
|------|------|---------|
| 01 | [任意词视觉化](./word-visualization) | 将任意词汇/短句转译为概念海报 |
| 02 | [城市宣传收藏级海报](./city-poster) | 城市主题的油画风格收藏级海报 |
| 03 | [名著经典海报](./classic-literature) | 东方叙事手绘风格的名著海报 |
| 04 | [国风唐诗宋词视觉化海报](./poetry-poster) | 古典诗词的视觉化解析海报 |

## 设计要点

- **语义理解先行**：先理解词语/诗句的含义、情绪、象征，再转译为视觉
- **极简强概念**：宁可少而准，不要多而散
- **文字即骨架**：大字作为画面核心结构，而非后期贴字
- **高级平面设计感**：克制配色、印刷品质感、展览级完成度
