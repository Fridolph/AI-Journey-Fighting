# 名著经典海报

> **类型**: 创意海报 &nbsp;|&nbsp; **标签**: `名著` `东方叙事` `手绘风格` `剪影轮廓` `收藏版`

## 概述

围绕经典名著主题创作具有收藏价值的东方叙事手绘海报。提供两个版本：虚版偏向纸本手绘、半写意东方风格；实版偏向电影海报与梦幻水彩融合风格。

---

## 版本一：虚版（东方叙事手绘）

请围绕最后输入的【主题 / 灵感 / 作品要求】，创作一张具有收藏价值的东方叙事手绘海报。整体风格必须接近高级国风纸本插画、东方水墨淡彩叙事绘画、收藏版书卷海报的质感，而不是写实电影概念海报，不是 CG 场景图，不是摄影式写实插画，也不是普通双重曝光拼贴。

画面主体为一个巨大、完整、优雅、极具识别度的人物侧脸剪影，但这个侧脸不是机械容器，也不是简单轮廓模板，而应像一笔气韵连贯的主形，轮廓与内部世界自然相生。剪影内部不是把元素塞满，而是从主题中提炼最核心、最有代表性的叙事场景、角色关系、象征符号、建筑、生物、道具与氛围线索，让它们像山海云气般自然浮现、层层展开，形成卷轴式、章节式、由上而下或由远及近的东方叙事节奏。整体结构要像同一位顶级插画师在同一张纸上统一构思、统一经营、统一绘制完成，不能像 photobash 合成，也不能像西式 montage 海报。

整体绘画语言必须偏东方手绘、半写意、非写实。重点不是逼真的体积和材质，而是气韵、虚实、留白、层次和诗性。山石、云雾、建筑、人物、神怪、路径与场景关系可以清晰，但不应过分写实，不应像电影分镜设定图。应保留明显的手绘概括感：有些地方精细，有些地方轻轻带过；有些地方轮廓明确，有些地方融入云气与纸面；让画面呈现"可读、可看、可品"的东方叙事绘画质感。

质感必须是纸本手绘感，而不是表面滤镜感。允许出现淡墨晕染、赭石淡彩、干刷痕迹、边缘飞白、轻微渗化、柔和颗粒、云烟化开的笔意、旧纸般的轻微底纹，但这些都必须克制，并服务于结构、空间、气氛和叙事节奏，不能均匀覆盖全图，不能做成廉价假水彩特效。整张画面要像高手画在高级宣纸、旧纸或手工纸上的成品海报，有真实的手绘呼吸感和高级完成度。

色彩系统必须高度克制，并从具体主题中提炼，而不是套用通用奇幻海报配色。整体以低饱和、纸本感、东方古意为主，可采用米白、淡墨灰、赭石、灰棕、烟金、浅雾色等为基础主调，再根据主题加入极少量点睛色。色彩关系应安静、统一、含蓄、雅致，像从旧卷轴和古画中生长出来。不要高饱和电影配色，不要紫蓝金奇幻模板，不要强烈商业 CG 光效。

版式必须具有明显的收藏版书卷海报气质。画面应保留大面积自然留白，主体与留白共同构成呼吸感。文字排版必须是整体设计的一部分，而不是后加贴字。标题可采用竖排中文书法或题签式排版，位置克制，具有古典书卷气；辅助文字可以极少量、小尺寸、细长排布，像书页边注或收藏版说明；可加入小型印章、题记或编号式装饰，但必须低调、精致、统一。整体版式应安静、讲究、克制，像一张可装裱的东方收藏版海报，而不是现代商业广告版面。

画面中的专属签名"xxd"要自然融入整体版式，像作者落款或小型题签，可放在左下角、右下角或留白区域，低调但清晰，字体气质应与整体统一，可带轻微手写感或设计签章感，不可过大，不可突兀，不可破坏画面气韵。

必须明确避免：写实摄影感、3D 渲染感、欧美奇幻概念图感、过度细节堆砌、强材质真实感、商业电影海报光效、机械双重曝光拼贴、现代潮流字体、杂乱信息排布、廉价水彩滤镜、满图刷痕、假纸纹覆盖。

最终效果应让人一眼感受到：这不是一张写实场景图，而是一张具有东方书卷气、纸本手绘感、半写意叙事性、收藏级版式审美的顶级插画海报。

## 版本二：实版（电影海报融合）

根据【主题】自动生成一张收藏版史诗叙事海报：巨大优雅的人物侧脸剪影作为外轮廓，剪影内部自动生长出最契合该主题的完整世界观、标志性场景、角色关系、象征符号、关键建筑、生物、道具与氛围。整体不是普通拼贴，而是高级的剪影轮廓填充式叙事合成，带有双重曝光式联想，但更偏电影海报与梦幻水彩插画融合风格；柔和空气透视，轻雾化过渡，纸张颗粒，边缘飞白与刷痕，大面积留白，版式克制高级，安静、宏大、神圣、怀旧、诗意、传说感强。风格、色彩、场景、材质全部根据主题自动适配，所有元素必须强绑定主题，一眼识别，不要杂乱，不要硬拼贴，不要模板化背景，不要廉价奇幻素材。画面中需自然加入专属签名"xxd"，作为海报设计的一部分，位置低调但清晰，可放在左下角、右下角或标题附近，风格需与整体版式统一，像收藏版海报的作者落款或设计签章；签名字体精致、克制、高级，不可过大，不可破坏主体构图，不可显得突兀廉价。
