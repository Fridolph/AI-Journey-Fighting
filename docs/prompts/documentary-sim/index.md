# 纪实模拟

模拟真实世界的摄影、纪录片、报纸等纪实风格图像。本分类下的提示词专注于"真实感优先于美感"的影像生成能力。

## 包含提示词

| 编号 | 名称 | 核心能力 |
|------|------|---------|
| 01 | [真实旁观抓拍](./bystander-snapshot) | 第三视角自然抓拍的生活瞬间 |
| 02 | [伪纪录片老照片](./mockumentary-photo) | 粗粝真实的现场感纪实图像 |
| 03 | [报纸头条假新闻](./fake-news-headline) | 复古报纸头版 + 新闻摄影 |

## 设计要点

- **真实感 > 美感**：宁可略丑、略粗糙，也不能假、空、飘
- **旁观视角**：人物不看镜头、不摆拍、自然松弛
- **物理痕迹**：折痕、磨损、灰尘、噪点等一切让画面真实的细节
- **非表演感**：像是被偶然拍下，而非精心设计
