# 伪纪录片老照片

> **类型**: 纪实模拟 &nbsp;|&nbsp; **标签**: `纪实摄影` `现场感` `粗粝` `冲击力` `老照片` `证据感`

## 概述

生成一张极度真实、真假难辨、带有强烈现实冲击力的现场感图像。真实感优先于美感——宁可粗糙、压抑，也不能假、空、飘。画面应具有"像真实世界上流传出来的一张现场照片"的气质。

---

## 提示词

你的任务不是生成一张"好看的图"，而是生成一张极度真实、真假难辨、带有强烈现实冲击力的现场感图像。

这张图必须满足以下核心目标：

1. 真实感极强，强到让人第一眼难以判断这到底是真实拍摄，还是AI生成。
2. 画面内容在视觉和心理上具有冲击力，甚至让人感到"这也太离谱了"，但仔细思考之后，又会觉得它并非纯幻想，而是现实中可能发生、可能存在、或至少逻辑上成立。
3. 不要生成"精致的写实插画"或"电影感剧照"，而是生成一种更接近现实世界偶然拍到的、粗粝的、未经修饰的、证据式的现场照片。
4. 真实感必须优先于美感。宁可略丑、略脏、略粗糙、略压抑，也不能显得假、空、飘、摆拍或过度设计。

### 一、智能理解用户输入

你需要根据用户最后提供的"主题 / 要素 / 要求"，智能判断最合理、最有冲击力、同时又最可信的呈现方式。
不要把画面机械地理解成字面插图，也不要生硬拼贴元素。
你要主动思考：
- 什么样的场景最可能承载这个主题
- 什么样的空间、人物、物件、痕迹、结构、状态，最能让这个主题既离谱又合理
- 什么样的视觉证据能让观众相信"这像是真的"
- 什么样的失控感、反差感、堆积感、痕迹感、偶然感，能让画面更有冲击力

允许你根据主题自动决定：
- 场景类型（室内 / 室外 / 半封闭空间 / 公共空间 / 私人空间 / 临时空间 / 功能空间等）
- 是否出现人物 / 人物数量与状态 / 道具与细节
- 混乱程度、异常程度、情绪方向

但前提是：一切都必须围绕"现实中可能成立"的逻辑展开，而不是变成纯超现实奇观。

### 二、真实感的底层原则

请不要只追求"高清"和"细节多"，而要追求真正的"真实照片感"。

#### 1. 成像逻辑真实

整张图要像现实设备拍摄出来的照片，而不是渲染图。
优先模拟：
- 普通手机拍摄 / 普通数码设备拍摄 / 非专业布光 / 自然环境光或混合光源 / 无意中拍下的现场感

成像上要具备真实摄影常见特征：
- 轻微广角透视或普通镜头视角
- 构图不完美，不要过于讲究
- 机位自然，像人站着、蹲着、探身、顺手拍到
- 局部清晰，局部略乱，不要全图过分锐利
- 高光可能轻微过曝 / 阴影可能略闷
- 白平衡可能不完全精准 / 室内外混合光可能带来轻微色偏
- 有轻微噪点、压缩痕迹、边缘锐化痕迹、传感器质感
- 不要有CG渲染那种"干净、均匀、漂亮得不真实"的质感

#### 2. 物理痕迹真实

所有东西都必须像被真实使用过，而不是模型摆放。请强化：
- 折痕 / 磨损 / 灰尘 / 污渍 / 压痕 / 拖拽痕迹
- 液体残留 / 受潮痕迹 / 破损 / 堆积关系
- 重力逻辑 / 接触关系 / 使用痕迹 / 时间留下的痕迹

#### 3. 空间关系真实

不要让场景像"摆出来给人看"的布景。必须让它像真实空间的一部分：
- 物体摆放要有生活逻辑 / 混乱要有来源和轨迹
- 密集区域与稀疏区域并存 / 不是平均铺开，而是自然堆积、散落、偏移、挤压、重叠
- 空间尺度要合理 / 动线要成立
- 人和环境之间要有互动痕迹 / 画面里应该有"发生过什么"的证据感

#### 4. 人物状态真实（如果画面适合出现人物）

人物不能像模特，不能摆拍，不能太干净，不能看起来在"配合拍摄"。
人物应更接近真实生活中的抓拍状态：
- 疲惫 / 放松过头 / 无表演意识 / 动作别扭但合理
- 姿势不优雅但真实 / 头发有自然凌乱感
- 衣服有穿着后的褶皱和堆积感 / 皮肤保留真实质感，不要磨皮，不要假脸
- 不看镜头，或对镜头没有明确反应
- 与周围环境融为一体，而不是突出成主角海报

### 三、冲击力的来源

这类图的冲击力，不来自猎奇特效，而来自"现实失控后的证据感"。

请优先制造以下类型的冲击：
- 数量异常，但分布合理
- 状态极端，但逻辑成立
- 细节粗糙，但非常真实
- 场面看似夸张，但又不是幻想
- 视觉上令人难以接受，但又找不到明显作假的地方
- 让人产生"太离谱了，可仔细想想又不是没可能"的感受

这种冲击最好来自：过度堆积 / 长期累积 / 使用失控 / 秩序崩坏 / 现实反差 / 生活痕迹被推到极端 / 平凡事物因为数量、状态、环境关系而显得异常震撼

不要把冲击力做成：血腥猎奇 / 恐怖片风格 / 纯粹恶心堆砌 / 魔幻超现实拼贴 / 低级奇观 / 明显摆拍 / 电影化灾难场景 / 艺术装置感

### 四、画面气质要求

整体气质应尽量接近：纪实摄影 / 现场记录 / 随手抓拍 / 社交媒体流出的原始照片 / 普通人无意拍下的震撼现场 / 看起来像真实世界中流传出来的一张图

画面不是为了"唯美"，而是为了"可信"。
可以略显压抑、局促、混乱、窒息、冷漠、疲惫、荒诞、麻木、失控，但都要克制，不要戏剧化过度。

### 五、严格压制的假感

请尽一切可能压制以下倾向：插画感 / CG感 / 3D渲染感 / 影视剧照感 / 海报感 / 网红摄影感 / 过度布光 / 过度磨皮 / 过于锐利 / 过于干净 / 过于对称 / 过于完美的构图 / 道具像复制粘贴 / 混乱缺乏逻辑 / 材质像塑料 / 人物像摆拍 / 场景为了"震撼"而硬编 / 明显AI痕迹 / 细节很多但没有生活证据链 / 一眼看上去像"AI在努力模仿真实"

### 六、输出策略

你需要根据用户输入，自主生成最适合的画面方案。
不要固定使用某一种环境。不要把所有主题都做成同一种房间、同一种脏乱、同一种构图。
而是要围绕用户给的主题，动态匹配最合理的场景、主体、叙事关系、视觉证据、情绪方向、混乱来源、异常程度、拍摄方式。

最终目标只有一个：生成一张"像真实世界里的现场照片一样成立"的图，让人第一眼怀疑真伪，第二眼细看仍然觉得真实，并且在视觉与心理上都感到强烈冲击。
