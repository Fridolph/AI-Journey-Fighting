# 真实旁观抓拍

> **类型**: 纪实模拟 &nbsp;|&nbsp; **标签**: `纪实摄影` `旁观视角` `自然抓拍` `生活感` `第三视角`

## 概述

模拟真实生活中偶然捕捉的一瞬间，具有自然、非表演感、带距离感的旁观视角。画面像是街拍、纪实摄影或通勤记录，让观者产生"刚好看见这一幕"的代入感。

**注意**：默认生成中近景，主体看得清（非远景），同时保留旁观感。

---

## 提示词

### 核心目标

这张图不是摆拍写真，不是广告大片，不是刻意设计的电影剧照，而是像在真实生活中，摄影者恰好从一个正常、合理、公开的观察位置，偶然捕捉到的一瞬间。

整张图需要具备以下气质：
- 自然 / 真实 / 非表演感 / 带距离感 / 有旁观视角 / 有现场感
- 有"刚好看到这一幕"的偶然性
- 像街拍、纪实摄影、通勤记录或生活观察照片
- 让观者产生"我仿佛正站在附近，恰好看见这一幕"的代入感

### 内容生成逻辑

请根据用户最后输入的"拍摄主题 / 要求元素 / 灵感来源"，智能决定：
- 最合适的场景类型 / 人物所处的位置 / 画面中的主次关系
- 前景、中景、背景的层次 / 哪些元素清晰，哪些元素虚化
- 哪些物体可以自然形成遮挡、切割、引导线和旁观感 / 最终整体氛围与色调

所有内容都必须围绕用户输入展开，不能脱离主题乱发挥。

### 重要修正：主体不要太远，必须看得清

这是重点要求，优先级很高：

1. 默认不要生成"大远景人物很小"的画面，除非用户明确要求远景。
2. 主体人物应当足够清晰可辨认，能看清：基本面部轮廓、发型、穿着、手部动作、正在做什么。
3. 画面景别优先使用：中景、中近景、半身到膝上构图、或包含完整站姿但人物依然足够大的构图。
4. 人物在画面中的视觉占比应合理：人物通常占画面高度约 1/3 到 2/3，不要小到缩在画面角落，不要让观众需要放大后才能看清主体。
5. 即使保留旁观感，也要让人物成为清楚的视觉焦点，而不是环境里的一个小点。
6. 如果场景较复杂，应优先保证"主体清楚"，环境作为辅助信息存在。
7. 避免过度宽广的全景镜头，除非用户特别要求"强环境叙事"。

### 视角要求

采用"合理、正常、公开空间中的第三视角观察构图"，重点是"旁观感"和"偶然抓拍感"，不是正面摆拍。

可以理解为：
- 从街边、通道、站台、商场、校园、公共休息区、开放式室内外空间等位置
- 以一种不打扰主体的方式
- 在正常视线高度或略带偏移的角度
- 捕捉人物处于自然状态中的一瞬间

画面不要像人物知道自己正在被拍摄。人物通常不直视镜头，动作应自然、松弛、生活化。

### 镜头语言与取景控制

为了防止主体太远，请优先使用更适合人物观察的真实镜头语言：
- 更接近中焦或标准焦段的观感
- 避免极端广角导致人物缩小
- 画面像摄影者站在相对合理的观察距离内拍到
- 可以理解为"站得不算远，但也没有贴脸拍"

整体观感更接近：街拍中近景、通勤纪实中景、生活观察式人物抓拍，而不是遥远的环境大全景。

### 构图要求

整体构图要像真实摄影，而非设计图。优先体现以下视觉特征：

1. **画面具有明显"旁观感"**：主体不要机械居中摆拍，可以偏左、偏右、偏下，留出呼吸空间；构图要像摄影者在真实环境中迅速捕捉，而不是精心摆位。

2. **前景遮挡要自然、多变、灵活，不死板**：不要总是只有窗框、门缝、树枝这几种。请根据具体场景，智能使用更生活化、更随机的前景元素来形成层次与旁观感，例如：
   - 路人的虚化肩膀、背影、手臂、衣角
   - 经过的车辆边缘、车窗倒影、后视镜边框
   - 公共设施边缘，如栏杆、广告牌、站牌、货架、扶手、玻璃隔断、座椅边缘
   - 近距离失焦物体，如植物、雨伞、袋子、栏杆、包、杯子、反光面、橱窗边缘
   - 光斑、反射、局部遮影、前景虚焦块面

   但必须注意：遮挡物不能严重破坏主体可见性，不要把人物挡得只剩一小块，不要让遮挡面积过大，遮挡的作用是增加层次和偶然感，不是让主体消失。

3. **画面要有空间层次**：前景可轻微虚化；主体相对清晰；背景略虚化或局部虚化；保持真实镜头语言，而非过度虚化到像商业写真。

4. **利用环境中的线条引导视线**：例如路灯杆、站牌、窗框、扶手、建筑边线、地面边界、雨伞边缘、走廊结构、灯带、橱窗框架等，这些线条可以形成竖向、横向或斜向节奏，引导观者自然看向主体。

### 人物状态

如果画面中有人物，人物应处于自然生活动作中，例如：等车、走路、看手机、发呆、低头整理物品、坐着休息、短暂停留、与环境发生轻微互动。

人物必须呈现：不看镜头、不刻意凹造型、不强烈表演、不夸张情绪、像在真实生活中被恰好记录到。

### 清晰度优先级

1. 主体人物清楚
2. 动作和状态清楚
3. 氛围真实
4. 环境辅助成立
5. 前景遮挡增加层次

不能为了氛围、遮挡、环境而牺牲人物辨识度。

### 真实感要求

真实感优先于美感，不要过分精致，不要"AI海报感"。

请强化以下真实摄影特征：
- 略带随机性的构图 / 适度的不完美 / 真实相机颗粒感
- 轻微运动模糊 / 局部失焦 / 自然光线过渡
- 真实环境纹理 / 真实反光、折射、阴影和空气感
- 不要磨皮，不要假脸，不要过度锐化
- 不要过度对称，不要像棚拍

### 光影要求

光线必须像真实环境中自然存在的光，而不是刻意打光。

请根据主题智能决定光影来源，例如：阴天自然光、路灯、霓虹反光、商场顶灯、公交站灯箱、车灯、玻璃反射光、窗边漫射光、夜景环境混合光。

整体光影应具备：真实、柔和但不虚假、有环境色温差、有局部明暗变化、不要过度戏剧化、不要商业广告式布光。

### 氛围要求

画面应带有一种：日常生活被偶然定格的感觉、安静的观察感、真实世界的空气感、似乎没有被打扰的自然流动感。

可以根据用户主题自动调整氛围倾向，例如：雨夜通勤感、清晨街头感、黄昏路边感、商场一隅感、校园偶遇感、城市生活切片感、安静等候感、一瞬间的孤独感或松弛感。

### 限制要求

- 必须是合规、公开、合理的观察视角
- 不表现违法偷拍、隐私侵犯、偷窥或私密空间窥视
- 不表现洗澡、更衣、厕所、卧室等私密场景中的观察
- 不渲染冒犯性或不适当的侵入感
- 核心是"纪实旁观感"和"偶然抓拍感"，不是侵犯感

### 输出重点

请最终生成一张：
- 高真实度 / 强纪实感 / 强第三视角旁观感 / 主体清晰可辨认
- 人物距离适中 / 具有遮挡层次 / 具有自然构图
- 像真实摄影抓拍 / 不摆拍 / 不广告化 / 不油腻 / 不刻意猎奇
- 无文字 / 无水印
