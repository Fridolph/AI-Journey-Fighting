# 科学百科信息图

> **类型**: 信息图 &nbsp;|&nbsp; **标签**: `科学百科` `解剖图` `知识图谱` `3D效果` `手写注释` `中文书法`

## 概述

生成一份高度详细、极其复杂且视觉效果惊艳的通用图解科学信息图，采用经典科学百科全书风格。支持人类、动物、植物三种学科模版，各有不同的知识模块结构。

---

## 提示词

职位：世界级科学百科全书插画师兼知识图谱架构师。

任务：生成一份高度详细、极其复杂且视觉效果惊艳的"通用图解科学信息图"，采用经典、无品牌（无标志）科学百科全书风格。

主题：从[人、植物或动物]中选择一个。

具体主题：[例如，巨型鱿鱼 / 达·芬奇 / 红杉树]。

风格：精美细致的科学插图，背景是复古的陈旧米色纸张。细腻的线条。极其复杂且专业。

### 关键视觉要求

1. **逼真的 3D 效果（核心主体）**：位于"C 位"的中心主体必须以非凡的真实感和动态性呈现。营造一种戏剧性的深度感，角色、植物或动物似乎破格而出，跳跃或爆裂，朝观众冲来（类似变形 3D 或动态弹出效果，具有高精度的真实感）。

2. **布局与战略空白**：
   - 中心主题：主导中心，周围有意设置"战略性空白"，以增强突出效果，使人物成为清晰的焦点。
   - 周边模块：周围区域（左、右、上、下和角落）必须根据学科不同，设置6-8个高度有序的知识模块。应该有有序的密度感，而不是随意的杂乱。模块本身必须有清晰的边界、页眉和详尽的内容。

3. **连接**：利用复杂且逻辑的细引线、箭头、括号、虚线和小连接点网络，将中心图形与周围所有模块连接起来，并将模块本身相互连接成一个连贯的知识网络。

4. **文本与注释（硬性要求——必须是清晰的中文）**：
   - 主标题：一大块、醒目且精美的书法体，写着特定主题的名称。
   - 书法口音：散布在主内容和模块标题中，使用清晰的中文书法表示重要术语。
   - 标准中文文本：所有其他描述性文本、手写笔记（大量清晰中文手写注释）、模块内容和注释必须为清晰易读的中文字符（简体中文），而非胡言乱语或不可辨认的符号。
   - 引导线注释：模块内的每一个小组件、细节、子模块、图表或插图都必须有详细的引导线注释（拟解剖图），以最大化专业性和教育价值。每个部件都应该有标签。

### 学科特定模块结构

**A. 对于人类：**
- 模块1：解剖与骨骼结构（含放大剖面）
- 模块2：生理过程（例如循环/神经系统）
- 模块3：历史背景与时间线（关键成就）
- 模块4：主要贡献图（详细分解）
- 模块5：认知过程/心理洞察
- 模块6：遗传特征/进化
- 模块7：全球影响力与文化影响
- 模块8：文化表征/遗产

**B. 对于动物：**
- 模块1：完整外部草图与解剖（含显微镜放大细节圆形窗）
- 模块2：行为模式与生命周期（例如，交配/迁移，流程图风格）
- 模块3：消化与骨骼系统
- 模块4：栖息地与分布图（含环境细节）
- 模块5：独特适应（如伪装、狩猎工具）
- 模块6：进化历史与亲属
- 模块7：共生关系/生态系统角色
- 模块8：保护状况与人类互动

**C. 植物：**
- 模块1：完整植物素描与解剖（附放大叶片/根部细节）
- 模块2：光合作用与生命周期流程（附环境图标）
- 模块3：细胞结构（放大圆形视图）
- 模块4：药用特性/实用应用
- 模块5：环境适应/独特特征
- 模块6：分布图与环境背景
- 模块7：遗传变异与培养
- 模块8：历史用法与民间传说

整体构图：信息极为密集，分为 6-8 个结构化模块，但中央留出战略性空白，使主体的超现实人物得以突出。硬核、专业、学术，但由于动态的 3D 中心人物，视觉上极具吸引力。没有任何特定百科全书的品牌标识。所有注释必须清晰可读。所有手写笔记必须清晰。中国书法的主标题。
