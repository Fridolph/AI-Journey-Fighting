# 涣卦（䷺）· 六爻提示词

> 卦序：第 59 卦 &nbsp;|&nbsp; 卦象：巽上坎下 &nbsp;|&nbsp; 风水涣 &nbsp;|&nbsp; 卦德：散

涣卦，是易经最「涣散消解」的一卦——风行水上，是风吹水面、涣散流动的卦象。「涣，亨」——涣，是亨通。「刚来而不穷，柔得位乎外而上同」——刚柔相济，是涣亨最深的根基。「王假有庙」——王至宗庙，是涣散中凝聚人心最深的象征。「先王以享于帝立庙」——涣，是享帝立庙最深的精神。六爻是涣散消解的六种面貌：**用拯马壮→涣奔其机→涣其躬→涣其群→涣汗大号→涣其血**，是一部从顺吉到远害无咎的完整涣散图鉴。

## 涣卦「王假有庙乘木有功」的深层结构

> 用拯马壮（初六）→涣奔其机（九二）→涣其躬（六三）→涣群光大（六四）→涣汗大号（九五）→涣血远害（上九◯）
> 涣散中凝聚——涣有丘匪夷所思；涣汗大号正位无咎；远害无咎是涣散最深的终结智慧。

## 兑与涣的深层对照

| | 兑卦 | 涣卦 |
|--|------|------|
| **核心** | 说之大民劝矣哉，刚中而柔外 | 王假有庙，乘木有功，涣有丘匪夷所思 |
| **最深一句** | 说之大，民劝矣哉 | 涣有丘，匪夷所思 |

兑是悦说感召，涣是涣散凝聚。悦说而后能涣散群党——兑的感召与涣的破除私党，是聚合与消解的两个面向。

## 初六 · 用拯马壮，吉

```text
易经爻辞知识图鉴
涣卦 · 初六 · 用拯马壮，吉

【画面主体意象】顺也，拯马壮吉——画面以「风行水上」为整体背景，风吹水面涣散流动是涣卦最根本的气息。

初六，「用拯马壮，吉」——用壮马拯救；「初六之吉，顺也」——顺是吉最深的根基。

画面主体：壮马拯救顺吉——「用拯马壮」涣散之初用壮马拯救，是顺从最深的行动；「顺也」——顺从则用拯马壮，用拯马壮则吉。

【几何爻位结构】涣卦六爻从下到上（巽上坎下）：初六 ━ ━  高亮深赭拯马 ← 当前爻 九二 ━━ 六三 ━ ━ 六四 ━ ━ 九五 ━━ 上九 ━━
标注：「巽上坎下，风行水上，涣」

【嵌入式大字】「初六」嵌入拯马壮顺几何结构，字色用深赭，被风行水上的动势部分遮挡——
「用拯马壮，吉」，字，拯马，顺吉，有涣卦最初的顺吉智慧。

【信息模块】① 涣卦 · 初六（右上角）② 爻辞：「用拯马壮，吉。」③ 小象传：「初六之吉，顺也。」
④ 彖传节录：「涣，亨。王假有庙，利涉大川，乘木有功也。」
⑤ 核心义理：涣的起点是「顺」——涣散之初用壮马拯救；「顺也」——顺则用拯马壮，用拯马壮则吉。
⑧ 收尾精神之语：「涣散之初用壮马拯救——吉；初六之吉顺也——顺是吉最深的根基。」

【配色】深赭·拯马为主，用拯马壮用深赭，顺吉用暖金，吉用素白，背景用深蓝水。
16:9横幅，宋代风水涣散拯马意境，现代平面设计结构。

## 涣卦六爻 · 涣散图鉴总结

```

> ——涣卦的终极义理，不是一味涣散，而是「涣中有聚」——涣散是为了凝聚；
> 涣有丘匪夷所思——涣散中有聚合是非常人所能思及的智慧；涣汗大号正位无咎——大号令是涣散最深的力量。
