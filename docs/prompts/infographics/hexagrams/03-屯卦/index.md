# 屯卦（䷂）· 六爻提示词

> 卦序：第 3 卦 &nbsp;|&nbsp; 卦象：坎上震下 &nbsp;|&nbsp; 水雷屯 &nbsp;|&nbsp; 卦德：难

屯卦和乾坤完全不同——乾是天道的磅礴，坤是地德的深厚，屯是**刚柔始交、难中求生**的挣扎感。雷在地下，云在天上，雨还没下来，万物在混沌中艰难破土。六爻的情感弧线是：**徘徊→困守→莽进→求合→积而不施→泣血**，是一部难中求通的众生相。

## 初九 · 磐桓，利居贞，利建侯

```
易经爻辞知识图鉴
屯卦 · 初九 · 磐桓，利居贞，利建侯

【画面主体意象】
雷雨前的磐石——
画面以一块巨大的磐石
为绝对主体，
石面宽阔，沉稳，
纹理深刻，
有久经风雨的质感。

磐石的状态：
不是死寂的石，
是有内力的石——
石缝间，
细小的草芽正在破缝而出，
极细，极嫩，
但方向是向上的，
「志行正也」。

磐石周围：
天空低沉，云雷积聚，
雨尚未落，
但空气中已有雷电的气息——
这是屯卦的天地背景，
「刚柔始交而难生」，
难，但生。

磐石下方：
隐约可见民众聚集的轮廓，
极淡，极小，
暗示「以贵下贱，大得民也」——
初九阳爻居下，
以贵下贱，
是建侯之基。

画面远处：
一面旗帜，
在风雨中立着，
「利建侯」——
不是已经建成，
是立旗的那一刻。

【几何爻位结构】
六爻从下到上排列：
初九 ━━  高亮 ← 当前爻
六二 ━ ━ （淡）
六三 ━ ━ （淡）
六四 ━ ━ （淡）
九五 ━━  （淡）
上六 ━ ━ （淡）
标注：「震下坎上，动乎险中」

【嵌入式大字】
「初九」嵌入磐石的几何块面，
字体有石刻感、碑铭感，
被石缝纹理部分切割，
字色用深墨，
与磐石融为一体，
局部可见，整体可读。

【笔刷动线】
一条从左下角向右上方
缓慢上升的笔刷，
模拟雷气在地下积聚之势，
在磐石底部最浓，
向上渐淡，
在旗帜处化为飞白——
是「磐桓」中蓄积的力量。

【信息模块】
① 屯卦 · 初九（右上角）
② 卦象标注：水雷屯 · 坎上震下
③ 爻辞：「磐桓；利居贞，利建侯。」
④ 小象传：「虽磐桓，志行正也。
   以贵下贱，大得民也。」
⑤ 爻位说明：阳爻居初，
   震卦之主，
   处屯之始，
   盘旋不进，非无力，
   是知时守正。
⑥ 核心义理：磐桓，不是停滞，
   是在最难的时候，
   把根扎进去。
   建侯，从这里开始。
⑦ 角落批注：
   「动乎险中，大亨贞」
⑧ 收尾精神之语：
   「盘旋，是因为知道
    这一步踩下去，
    要踩得稳。」

【配色】
深墨磐石灰为主，
石缝草芽用极嫩的冷绿，
天空用低沉的雷云深灰，
旗帜用点赭，
整体沉稳、有力、蓄势，
是屯卦六爻中
最「稳」的一张，
难中有志，
磐而不散。

现代平面设计结构。

## 六二 · 屯如邅如，乘马班如，十年乃字

```
易经爻辞知识图鉴
屯卦 · 六二 · 屯如邅如，乘马班如，十年乃字

【画面主体意象】
困顿中的守候——
画面以泥泞的道路为主体，
雨正在下，
路面积水，
马蹄印深陷泥中，
「乘马班如」——
马在原地踏步，
不是不想走，
是路太难，走不了。

画面中心：
一匹马，侧面，
四蹄深陷泥泞，
头微低，
不是绝望，
是在等——
「十年乃字」，
等，是这一爻的核心动作。

马的旁边：
一个计时的意象——
沙漏，或刻有年份的木桩，
暗示「十年」的漫长，
但木桩还立着，
没有倒。

天空：
阴雨连绵，
但不是暴雨，
是那种
看不见尽头的阴天——
「六二之难，乘刚也」，
难，是因为下面踩着刚（初九），
处境本就不易。

画面远处：
隐约可见晴天的光，
极淡，极远，
「反常也」——
十年之后，
终于回到正常，
但那个光，现在还很远。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
六二 ━ ━  高亮冷蓝 ← 当前爻
六三 ━ ━  （淡）
六四 ━ ━  （淡）
九五 ━━   （淡）
上六 ━ ━  （淡）
标注：「乘刚也，十年乃字，反常也」

【嵌入式大字】
「六二」嵌入阴雨几何层，
字色用冷灰蓝，
被雨丝纹理部分遮挡，
有一种「屯如邅如」
的迟滞感——
字在，但走得很慢。

【笔刷动线】
一条几乎水平、
极缓慢移动的笔刷，
模拟「班如」——
来回踏步，
进退两难，
笔触有重复叠压的痕迹，
如马蹄在泥中反复踩踏。

【信息模块】
① 屯卦 · 六二（右上角）
② 爻辞：「屯如邅如，乘马班如。
   匪寇婚媾，女子贞不字，
   十年乃字。」
③ 小象传：「六二之难，乘刚也。
   十年乃字，反常也。」
④ 爻位说明：阴爻居二，
   乘初九之刚，
   处境困顿，
   守贞不从，
   待时而动。
⑤ 核心义理：困，不是因为弱，
   是因为时机未到。
   十年，是守贞的代价，
   也是守贞的意义。
⑥ 角落批注：「反常也」——
   十年后回到正常，
   说明守贞是对的。
⑦ 收尾精神之语：
   「十年，不是惩罚，
    是时间在验证
    你当初的那个'不'。」

【配色】
阴雨冷灰为主，
泥路用深赭褐，
马用深墨，
雨丝用极淡冷银，
远处晴光用极淡暖黄，
整体压抑、湿冷、漫长，
是六爻中最「等」的一张，
困而不乱，
守而有节。

16:9横幅，宋代雨景意境，
现代平面设计结构。
```

## 六三 · 既鹿无虞，惟入于林中，往吝

```
易经爻辞知识图鉴
屯卦 · 六三 · 既鹿无虞，惟入于林中，往吝

【画面主体意象】
无向导的深林——
这是屯卦最具警示性的一爻，
「既鹿无虞」——
追鹿却没有虞人（向导），
「惟入于林中」——
只是莽撞地冲进了密林。

画面主体：
一片密林，
林木高大，
枝叶交错，
遮天蔽日，
深处黑暗，
看不见出路。

林中：
一条鹿的蹄印，
通向林深处，
越走越深，
越走越暗——
鹿在里面，
但没有向导，
追进去，
只会迷失。

林口：
一个岔路口，
一条路通向林内（暗），
一条路通向开阔地（淡光）——
「君子几不如舍」——
聪明的人，
在这里停下来，
选择那条有光的路。

但画面的重心在林内——
因为这一爻的警示，
是已经走进去了，
或者正在走进去的那一刻。

林口地面：
散落的猎具，
或一顶被遗落的帽子，
暗示「往吝」——
有人已经进去了，
留下了这些。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  高亮暗绿 ← 当前爻
六四 ━ ━  （淡）
九五 ━━   （淡）
上六 ━ ━  （淡）
标注：「以纵禽也，往吝穷也」

【嵌入式大字】
「六三」嵌入密林的几何暗块，
字色用深林墨绿，
几乎被林木遮没，
只在林隙透光处
隐约可辨——
这是「无虞」的视觉化，
方向不明，字也难辨。

【笔刷动线】
一条从林口向林深处
急速冲入的笔刷，
模拟「莽进」之势，
笔触急促，
在林深处戛然而止——
「往吝」，
走到这里，停了，
但已经进去了。

【信息模块】
① 屯卦 · 六三（右上角）
② 爻辞：「既鹿无虞，惟入于林中，
   君子几不如舍，往吝。」
③ 小象传：「既鹿无虞，以纵禽也。
   君子舍之，往吝穷也。」
④ 爻位说明：阴爻居三，
   处下卦之上，
   无应无援，
   莽进则吝，
   知舍则智。
⑤ 核心义理：没有向导的路，
   不是勇敢，是莽撞。
   「舍」，有时候
   比「往」需要更大的勇气。
⑥ 角落批注：
   「君子几不如舍」——
   几，是几乎，是差一点，
   是还来得及的那一刻。
⑦ 收尾精神之语：
   「林很深，鹿很快，
    但没有向导，
    追进去的，
    是你自己。」

【配色】
深林墨绿·暗褐为主，
林外用极淡的冷光，
林内用最深的墨绿黑，
鹿蹄印用深赭，
遗落猎具用暗金，
整体压抑、幽暗、有警示感，
是六爻中最「暗」的一张，
暗，是因为没有向导，
不是因为没有路。

16:9横幅，宋代山林意境，
现代平面设计结构。
```

## 六四 · 乘马班如，求婚媾，无不利

```
易经爻辞知识图鉴
屯卦 · 六四 · 乘马班如，求婚媾，无不利

【画面主体意象】
主动求合的转机——
六四是屯卦的转折点，
同样是「乘马班如」，
但这一爻不是困守，
是主动出发去求——
「求而往，明也」，
主动，是这一爻的核心。

画面主体：
一匹马，正在出发，
不是六二那匹困在泥中的马，
这匹马蹄子已经抬起，
方向明确，
向着画面右侧的光走去。

马的前方：
晨光初现，
道路逐渐清晰，
远处有隐约的人影或灯火，
暗示「婚媾」——
合，是这条路的终点。

马的状态：
步伐还不是飞奔，
仍有「班如」的迟缓，
但头是抬着的，
方向是对的——
「求而往」，
主动迈出这一步，
已经是「明」了。

画面地面：
从泥泞渐变为干燥，
从阴暗渐变为晨光，
是屯卦六爻中
第一次出现明显转暖的地面——
困，开始松动了。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  （淡）
六四 ━ ━  高亮晨光赭 ← 当前爻
九五 ━━   （淡）
上六 ━ ━  （淡）
标注：「求而往，明也」

【嵌入式大字】
「六四」嵌入晨光几何层，
字色用暖赭金，
被马身和晨雾部分遮挡，
比六三明显更亮，
整体可读，
有一种「困中见光」的感觉。

【笔刷动线】
一条从左向右
明确前进的笔刷，
不再是六二的来回踏步，
是有方向的流动，
从左侧深色渐变为
右侧暖金，
模拟「求而往」的动势——
方向对了，
笔就走了。

【信息模块】
① 屯卦 · 六四（右上角）
② 爻辞：「乘马班如，
   求婚媾，无不利。」
③ 小象传：「求而往，明也。」
④ 爻位说明：阴爻居四，
   与初九相应，
   主动下求，
   柔顺得正，
   无不利之象。
⑤ 核心义理：同样是「乘马班如」，
   六二是被困，
   六四是主动求——
   处境相似，心态不同，
   结果天壤之别。
⑥ 角落批注：
   「求，是这一爻
    和六二最大的区别。」
⑦ 收尾精神之语：
   「主动走出去，
    就是明。
    困，有时候只差
    那个'求'字。」

【配色】
晨光赭暖为主，
马用深墨，
地面从深赭渐变为暖土，
远处灯火用点金，
天空用晨雾淡蓝渐暖，
整体是屯卦六爻中
第一张有明显暖意的图，
困，开始松动，
光，开始出现。

16:9横幅，宋代行旅意境，
现代平面设计结构。
```

## 九五 · 屯其膏，施未光也

```
易经爻辞知识图鉴
屯卦 · 九五 · 屯其膏，小贞吉，大贞凶

【画面主体意象】
积而不施的困局——
「屯其膏」是这一爻最深刻的意象，
膏，是膏泽，是恩泽，
屯，是积聚，是囤积——
有恩泽，但施不出去，
「施未光也」。

画面主体：
一盏灯，或一个装满油的器皿，
油满，灯芯也有，
但灯，没有点亮——
不是没有光，
是光还没有出来。

灯的周围：
黑暗，
不是敌意的黑暗，
是等待的黑暗——
周围的人需要光，
但光被困在灯里，
出不来。

灯的上方：
厚重的云层，
坎卦（水、险）的意象——
险，是九五被困的原因，
不是九五自己不愿施，
是险阻让恩泽无法普照。

画面一角：
隐约可见远处的黑暗中
有人在等待，
极小，极淡，
「施未光也」——
他们需要光，
但光还没到。

灯与黑暗之间：
一道极细的光缝，
暗示「小贞吉」——
小处守正，
还有一线可为，
但大处，
险阻太深，
「大贞凶」。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  （淡）
六四 ━ ━  （淡）
九五 ━━   高亮暗金 ← 当前爻
上六 ━ ━  （淡）
标注：「施未光也，坎险在上」

【嵌入式大字】
「九五」嵌入灯与黑暗的
几何交界处，
字色用暗金——
不是九五飞龙的亮金，
是被压住的金，
有力量，但出不来，
局部遮挡，整体可读。

【笔刷动线】
一条从灯芯向外扩散
但被截断的笔刷，
模拟「施未光」之势——
力道有，
但在中途被厚重的
云层几何块压住，
扩散到一半，停了，
是这一爻最准确的视觉化。

【信息模块】
① 屯卦 · 九五（右上角）
② 爻辞：「屯其膏，
   小贞吉，大贞凶。」
③ 小象传：「屯其膏，施未光也。」
④ 爻位说明：阳爻居五，
   君位，有德有位，
   但坎险在上（坎为上卦），
   恩泽被困，
   施而不广。
⑤ 核心义理：九五不是无能，
   是被险阻困住了。
   屯其膏，是时势的困，
   不是人的错——
   但困在君位，
   代价最重。
⑥ 角落批注：
   「小贞吉——
    在能做的范围内守正，
    仍有可为。」
⑦ 收尾精神之语：
   「满灯的油，
    点不亮，
    不是油的问题，
    是这个时候，
    风太大。」

【配色】
深金·暗郁为主，
灯用暗琥珀金，
灯油用深赭金，
周围黑暗用最深的墨，
那道光缝用极细的冷白，
远处等待者用极淡墨，
整体压抑、郁结、有重量，
是六爻中最「憋」的一张——
有力，但使不出来。

16:9横幅，宋代灯火意境，
现代平面设计结构。
```

## 上六 · 乘马班如，泣血涟如

```
易经爻辞知识图鉴
屯卦 · 上六 · 乘马班如，泣血涟如

【画面主体意象】
困极无出的泣血——
上六是屯卦的极处，
同样是「乘马班如」，
这是第三次出现这个意象——
初见是困顿，
再见是求合，
三见是绝望。

画面主体：
一匹马，
站在原地，
不动——
不是六二的等待，
不是六四的出发，
是已经走不动了，
或者，已经不知道往哪里走了。

马的状态：
头深低，
鬃毛凌乱，
四蹄沾满泥泞，
身上有雨，
有风，
「泣血涟如」——
不画血，
用雨代血，
雨从马身滑落，
在地面形成涟漪，
每一圈涟漪，
都是一声无声的哭泣。

天空：
最深的夜，
或最深的阴云，
没有月，没有星，
没有那道光缝——
「何可长也」，
这种状态，
不能持续，
但现在，
还没有出路。

地面：
积水，
反射着极暗的天色，
马的倒影在水中，
模糊，破碎——
是上六最准确的自画像。

画面极远处：
极极淡的一线晨光，
几乎看不见——
不是希望的光，
是「何可长也」的提示——
这个状态，
终究会结束，
但那个结束，
现在还很远。

【几何爻位结构】
六爻从下到上：

标注：「何可长也，屯之极」

【嵌入式大字】
「上六」嵌入最深的夜色几何块，
字色用极深的玄黑，
几乎与背景融为一体，
只在积水反光处
隐约可见字的轮廓——
「泣血涟如」的字体处理，
存在，但已经快看不见了。

【笔刷动线】
一条从上向下
垂落的笔刷，
模拟雨落、泪落之势，
笔触断续，
有涟漪扩散的痕迹，
在地面积水处
形成同心圆的墨晕——
「涟如」，
就是这个形状。

【信息模块】
① 屯卦 · 上六（右上角）
② 爻辞：「乘马班如，泣血涟如。」
③ 小象传：「泣血涟如，何可长也。」
④ 爻位说明：阴爻居极上，
   屯之终，
   困之极，
   无应无援，
   动则无功，
   止则泣血。
⑤ 核心义理：「何可长也」
   不是绝望，
   是易经的提示——
   极处必变，
   但变之前，
   要先经历这个泣血。
⑥ 角落批注：
   「三见乘马班如：
    六二——困守；
    六四——求合；
    上六——泣血。
    同一句话，三种命运。」
⑦ 收尾精神之语：
   「泣血，
    不是终点，
    是屯卦在说：
    这里，已经到底了，
    该变了。」

【配色】
玄黑·血色为主，
天空用最深的玄墨，
积水用深墨蓝，
雨丝用极细冷银，
涟漪用深墨晕，
极远处晨光用极极淡的暖白，
整体是屯卦六爻中
最沉、最暗、最悲的一张，
但「何可长也」四字，
是黑暗中唯一的方向——
极处，就是转机的前夜。

## 「乘马班如」三现对照

| 爻位 | 同一句话 | 不同命运 |
|------|---------|---------|
| **六二** | 乘马班如 | 困守·等待·十年 |
| **六四** | 乘马班如 | 主动·求合·无不利 |
| **上六** | 乘马班如 | 穷极·泣血·何可长 |

同一句「乘马班如」，六二是困，六四是明，上六是悲——屯卦最深的哲学，藏在这三次重复里。
