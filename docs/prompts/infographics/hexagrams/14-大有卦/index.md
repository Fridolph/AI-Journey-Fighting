# 大有卦（䷍）· 六爻提示词

> 卦序：第 14 卦 | 卦象：离上乾下 | 火天大有 | 卦德：有

大有是易经最「富」的一卦——火在天上，光照四方，一阴居尊，五阳应之。六爻藏着持盈守正的深刻智慧：**无交害→大车载→公亨天子→匪其彭→孚威相济→自天佑**。

## 同人与大有的深层对照

| | 同人卦 | 大有卦 |
|--|--------|--------|
| **卦象** | 天火同人（火在下） | 火天大有（火在上） |
| **君子之道** | 类族辨物 | 竭恶扬善，顺天休命 |
| **核心** | 通天下之志 | 持盈守正 |
| **最深一句** | 先号啕而后笑 | 自天佑之，吉无不利 |

同人是求同，大有是持有。先通天下之志，才能真正大有；大有而顺天，才能自天佑之。

> 「竭恶扬善，顺天休命」——大有不是终点，是责任。有了大有，竭尽恶，扬尽善，顺着天道，安享天命，这才是「吉无不利」的真正原因。
