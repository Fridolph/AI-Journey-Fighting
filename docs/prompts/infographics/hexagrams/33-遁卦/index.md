# 遁卦（䷠）· 六爻提示词

> 卦序：第 33 卦 &nbsp;|&nbsp; 卦象：乾上艮下 &nbsp;|&nbsp; 天山遁 &nbsp;|&nbsp; 卦德：退

## 遁卦「遁之时义大矣哉」的深层结构

> 遁尾不往何灾（初六）→黄牛革固志（六二）→系遁有疾不大事（九三）→好遁君子吉（九四）→嘉遁以正志（九五）→肥遁无所疑无不利（上九◯）
> 从被迫遁尾到从容肥遁——遁之六层递进；尾→固→系→好→嘉→肥，是君子隐退之道完整修炼。

## 恒与遁的深层对照

| | 恒卦 | 遁卦 |
|--|------|------|
| **核心** | 立不易方，圣人久於其道天下化成 | 遁而亨，与时行也，君子远小人不恶而严 |
| **最深一句** | 悔亡，能久中也 | 肥遁，无不利 |

恒是恒久守持，遁是从容隐退。恒而后知遁——知守持而后知进退之时，恒与遁是进与退的两种智慧。

## 初六 · 遁尾，厉，勿用有攸往

```text
易经爻辞知识图鉴
遁卦 · 初六 · 遁尾，厉，勿用有攸往

【画面主体意象】遁尾之厉，不往何灾也——初六是「遁尾」的一爻，「遁尾」——退隐于尾，「厉」——危厉，「勿用有攸往」——不可前往；
「不往何灾也」——不前往则无灾。遁尾厉不可往不往则无灾——不往是遁道最初智慧。

【几何爻位结构】遁卦从下到上（乾上艮下）：初六 ━ ━  高亮暗褐遁尾 ← 当前爻 六二 ━ ━ 九三 ━━ 九四 ━━ 九五 ━━ 上九 ━━
标注：「天下有山，遁；君子以远小人，不恶而严」

【嵌入式大字】「初六」嵌入遁尾勿往几何结构，字色用暗褐，被遁尾厉不往的沉重部分遮挡——
「遁尾，厉，勿用有攸往」，字，遁尾，不往，有遁卦最初遁尾厉勿往不往何灾。

【信息模块】① 遁卦 · 初六（右上角）② 爻辞：「遁尾，厉，勿用有攸往。」③ 小象传：「遁尾之厉，不往何灾也。」
④ 彖传节录：「遁之时义大矣哉！」⑤ 核心义理：遁道起点——遁尾厉不可往不往则无灾。
⑨ 收尾精神之语：「退隐于尾有危厉——不可前往；不往何灾也——不往则无灾。」

【配色】暗褐·遁尾为主，遁尾用暗褐，厉用深灰，勿往用暗金，不往何灾用深赭，背景用深蓝天山。
16:9横幅，宋代遁尾勿往意境，现代平面设计结构。
```
