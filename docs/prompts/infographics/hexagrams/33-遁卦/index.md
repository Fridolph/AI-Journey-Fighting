# 遁卦（䷠）· 六爻提示词

> 卦序：第 33 卦 | 卦象：乾上艮下 | 天山遁 | 卦德：退

遁卦是易经最「退」的一卦——天高山止，是退隐、是遁去。「遁亨，遁而亨也，与时行也」——退而亨通，顺时而退。六爻：**遁尾厉→黄牛固志→系遁疾厉→好遁君子→嘉遁正志→肥遁无疑**。

## 遁卦「与时行」的深层结构

> 遁尾迟危（初六）→固志执守（六二◎）→被系疾惫（九三↓）→君子好遁（九四↑）→嘉美正志（九五☀️）→肥遁无疑（上九☀️）
> 遁要与时行，要固志，要无所疑。肥遁，是遁的最自在形态。

## 恒与遁的深层对照

| | 恒卦 | 遁卦 |
|--|------|------|
| **核心** | 久于其道 | 遁而亨，与时行 |
| **最深一句** | 天地万物之情可见矣 | 遁之时义大矣哉 |

恒是久于正道，遁是顺时而退。知久知退，是易经最深的时义。

> 「肥遁，无不利，无所疑也」——退，退得自在，退得肥美。遁之时义大矣哉。
