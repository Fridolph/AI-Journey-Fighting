# 履卦（䷉）· 六爻提示词

> 卦序：第 10 卦 &nbsp;|&nbsp; 卦象：乾上兑下 &nbsp;|&nbsp; 天泽履 &nbsp;|&nbsp; 卦德：行

履卦的气质极其独特——上天下泽，天高泽低，君子行走在礼与位之间。「履虎尾，不咥人」——踩着老虎的尾巴，却没有被咬，是礼的智慧，是知位而行的艺术。六爻是行履的六种面貌：**素履→坦道→眇跛→愬愬→夬履→视履考祥**，是一部关于「行」与「位」的礼义图鉴。

## 初九 · 素履，往无咎

```
易经爻辞知识图鉴
履卦 · 初九 · 素履，往无咎

【画面主体意象】
朴素而行，独行其愿——
画面以一双朴素的草履
为绝对主体，
「素履」——
素，是朴素，
是没有装饰的，
是本来的样子。

草履的质感：
简单，朴实，
有走过路的痕迹，
但不破败——
是一个人
安安静静走自己的路
留下的痕迹。

草履旁：
一条清晰的道路，
向前延伸，
「往无咎」——
就这样走，
没有问题。

「独行愿也」的意象：
道路上，
只有一行脚印，
不是孤独，
是专注——
独行，
是因为知道
自己要去哪里。

背景：
上天下泽，
天高在上，
泽在下方，
位分清晰，
但初九，
只是朴素地走着，
不去想那么多——
「素」，
是最轻盈的行走方式。

天色：
清晨，
光刚刚透出，
是出发的时刻，
素履，
在清晨的光里，
往，无咎。

【几何爻位结构】
履卦六爻从下到上
（乾上兑下）：
初九 ━━  高亮素白 ← 当前爻
九二 ━━
六三 ━ ━
九四 ━━
九五 ━━
上九 ━━
标注：「兑下乾上，上天下泽」

【嵌入式大字】
「初九」嵌入草履
几何朴素块，
字色用素白清赭，
被草履纹理
部分遮挡，
字，像草履一样，
朴素，不张扬——
「素」，
是字的气质。

【笔刷动线】
一条从草履处
向前方
朴素延伸的笔刷，
笔触轻盈，
有飞白，
不急，不重——
「素履之往」，
就是这条线
走得最轻的样子。

【信息模块】
① 履卦 · 初九（右上角）
② 卦象标注：天泽履 · 乾上兑下
③ 爻辞：「素履，往无咎。」
④ 小象传：「素履之往，独行愿也。」
⑤ 象传节录：「上天下泽，履；
   君子以辨上下，安民志。」
⑥ 彖传节录：「履，柔履刚也。
   说而应乎乾，
   是以履虎尾，不咥人，亨。」
⑦ 爻位说明：阳爻居初，
   乾之始，
   以素朴之心行走，
   不求华丽，
   独行其愿，
   无咎。
⑧ 核心义理：最好的行走，
   是朴素的行走。
   「素」，
   不是贫乏，
   是不被外物所累的
   轻盈。
⑨ 角落批注：「独行愿——
   知道自己要去哪里，
   就够了。」
⑩ 收尾精神之语：
   「不需要华丽的鞋，
    不需要人陪，
    知道方向，
    朴素地走，
    就无咎。」

【配色】
素白·清赭为主，
草履用温润素赭，
道路用淡清赭，
天空用清晨淡蓝，
脚印用深赭，
整体朴素、清朗、轻盈，
是履卦六爻中
最「素」的一张，
素，是最轻盈的出发。

16:9横幅，宋代平远意境，
现代平面设计结构。
```

## 九二 · 履道坦坦，幽人贞吉

```
易经爻辞知识图鉴
履卦 · 九二 · 履道坦坦，幽人贞吉

【画面主体意象】
坦道中行，中不自乱——
九二是履卦最「安静」的一爻，
「履道坦坦」——
走在平坦宽阔的道上，
「幽人贞吉」——
幽静之人，
守正则吉，
「中不自乱也」——
内心不自乱，
是这一爻最深的功夫。

画面主体：
一条极宽阔、极平坦的道，
向远方延伸——
不是山路，
不是险道，
是「坦坦」的道，
平，是这条路
最重要的质感。

道路两侧：
幽静的树木，
或幽静的山影，
「幽人」的气质——
不是热闹的地方，
是安静的、
可以守正的地方。

道上的人：
一个幽静的背影，
不急，不慢，
走在道路正中——
「中」，
是这个背影
最重要的位置。

天色：
平和，
不是清晨的出发，
是白日的行走，
光，均匀地照着，
没有阴影，
「坦坦」，
是光也是道。

画面整体：
有一种
「世界很安静，
我在走自己的路」
的幽静感——
「幽人贞吉」，
幽，不是隐遁，
是内心的安静。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
九二 ━━  高亮深赭坦 ← 当前爻
六三 ━ ━ （淡）
九四 ━━  （淡）
九五 ━━  （淡）
上九 ━━  （淡）
标注：「幽人贞吉，中不自乱也」

【嵌入式大字】
「九二」嵌入坦道
几何宽阔层，
字色用深赭，
被道路的宽阔感
部分遮挡，
字，在道路正中——
「坦坦」，
字，也是平的。

【笔刷动线】
一条极平稳、极宽阔
的横向笔刷，
不急，不偏，
在道路正中
匀速流动——
「坦坦」，
就是这条线
走得最平的样子。

【信息模块】
① 履卦 · 九二（右上角）
② 爻辞：「履道坦坦，幽人贞吉。」
③ 小象传：「幽人贞吉，中不自乱也。」
④ 爻位说明：阳爻居二，
   得中，
   以幽静之心
   行走坦道，
   中不自乱，
   贞吉。
⑤ 核心义理：坦道，
   不是没有挑战，
   是内心足够安静，
   所以路，
   看起来坦坦。
   「中不自乱」——
   乱，往往从内部开始。
⑥ 角落批注：
   「中不自乱——
    外面再乱，
    中心不乱，
    道，就是坦的。」
⑦ 收尾精神之语：
   「道坦不坦，
    不在道，
    在走道的人
    心里，
    乱不乱。」

【配色】
深赭·坦道为主，
道路用宽阔温赭，
两侧树影用深墨绿，
天空用均匀淡蓝，
背影用深墨，
整体幽静、平和、宽阔，
是履卦六爻中
最「坦」的一张，
坦，是中正的样子。

16:9横幅，宋代平远幽道意境，
现代平面设计结构。
```

## 六三 · 眇能视，跛能履，履虎尾，咥人，凶

```
易经爻辞知识图鉴
履卦 · 六三 · 眇能视，跛能履，履虎尾，咥人，凶

【画面主体意象】
自不量力，位不当也——
六三是履卦最具警示性的一爻，
「眇能视」——
眼睛看不清，
却以为自己能看；
「跛能履」——
脚有残疾，
却以为自己能走；
「履虎尾，咥人，凶」——
这样的状态，
踩上了虎尾，
被咬了，
凶。

画面主体：
一只老虎的尾巴，
占据画面中心，
一只脚，
正踩在上面——
不是勇敢，
是不知道自己在踩什么。

「眇能视」的意象：
画面一侧，
一双眼睛，
视线模糊，
却睁得很大——
「不足以有明也」，
看，但看不清。

「跛能履」的意象：
一行歪斜的脚印，
不稳，
却走得很快——
「不足以与行也」，
走，但走不稳。

老虎的反应：
虎尾被踩，
虎头回转——
「咥人」，
就是这个
回头的瞬间，
凶，是这个瞬间的结果。

画面整体：
不是恐怖，
是警示——
不是老虎的错，
是那只脚，
不该踩在那里。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
九二 ━━  （淡）
六三 ━ ━ 高亮暗灰裂 ← 当前爻
九四 ━━  （淡）
九五 ━━  （淡）
上九 ━━  （淡）
标注：「位不当也，咥人之凶」

【嵌入式大字】
「六三」嵌入虎尾
几何危险块，
字被虎纹
深度切割，
字色用暗灰，
有一种
「看不清自己在哪里」
的模糊感——
「眇能视」，
连字，都看不清。

【笔刷动线】
一条歪斜、不稳
却急速向前的笔刷，
在虎尾处
骤然停顿，
然后有一个
被咬的反弹——
「咥人」，
就是这个
停顿和反弹。

【信息模块】
① 履卦 · 六三（右上角）
② 爻辞：「眇能视，跛能履，
   履虎尾，咥人，凶。
   武人为于大君。」
③ 小象传：「眇能视；
   不足以有明也。
   跛能履；不足以与行也。
   咥人之凶；位不当也。
   武人为于大君；志刚也。」
④ 爻位说明：阴爻居三，
   失位不中，
   以柔居刚位，
   自不量力，
   强行而凶。
⑤ 核心义理：履卦最深的警示——
   不是老虎太危险，
   是自己不知道
   自己的能力在哪里。
   「位不当」，
   是凶的根本原因。
⑥ 角落批注：
   「不足以有明，
    不足以与行——
    两个「不足以」，
    是易经最诚实的评语。」
⑦ 收尾精神之语：
   「眼睛模糊，
    脚步不稳，
    却去踩虎尾——
    凶，不是命运，
    是自己选的。」

【配色】
暗灰·裂警为主，
虎尾用深虎纹赭黄，
那只脚用深墨，
眼睛意象用模糊冷灰，
歪斜脚印用暗褐，
背景用压抑深灰，
整体警示、危险、
有一种「看不清」的模糊感，
是履卦六爻中
最「危」的一张，
危，是自不量力的代价。

16:9横幅，宋代警示意境，
现代平面设计结构。
```

## 九四 · 履虎尾，愬愬终吉

```
易经爻辞知识图鉴
履卦 · 九四 · 履虎尾，愬愬终吉

【画面主体意象】
惕惧而行，志行终吉——
九四与六三同样是「履虎尾」，
但结果完全不同——
「愬愬」——
惕惧，警惕，
知道自己在踩虎尾，
所以每一步，
都走得极其谨慎，
「终吉」——
因为这份惕惧，
终究吉祥。

画面主体：
同样是老虎的尾巴，
但画面的气质
与六三完全不同——
这里，
是一双极其谨慎的脚，
踩在虎尾上，
每一步，
都轻，都稳，
「愬愬」的感觉。

「愬愬」的意象：
不是恐惧到不敢动，
是在恐惧中
依然前行——
脚步轻，
但方向坚定，
「志行也」——
志，在行走中。

老虎的状态：
虎，没有回头——
因为那双脚，
走得太轻，
太谨慎，
虎，感受不到威胁。

画面光线：
比六三暗，
但有一道
从前方透来的光——
「终吉」，
光，在前方，
惕惧，是走向那道光
的方式。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
九二 ━━  （淡）
六三 ━ ━ （淡）
九四 ━━  高亮深墨虎纹 ← 当前爻
九五 ━━  （淡）
上九 ━━  （淡）
标注：「愬愬终吉，志行也」

【嵌入式大字】
「九四」嵌入虎尾
几何结构，
字色用深墨，
被虎纹
部分遮挡，
但字，清晰可读——
「愬愬」，
字，走得很稳，
不像六三那样
模糊裂散。

【笔刷动线】
一条极轻、极谨慎
的笔刷，
沿虎尾方向
缓慢前行，
每一笔，
都有停顿和确认——
「愬愬」，
就是这条线
走得最轻、最慎的样子，
但，一直在走。

【信息模块】
① 履卦 · 九四（右上角）
② 爻辞：「履虎尾，愬愬终吉。」
③ 小象传：「愬愬终吉，志行也。」
④ 爻位说明：阳爻居四，
   近君位，
   知险而惕，
   以谨慎之心
   行险道，
   志不改，
   终吉。
⑤ 核心义理：同样是履虎尾，
   六三凶，九四吉——
   区别只在一个字：
   「愬愬」。
   知道危险，
   还是走，
   但走得很轻——
   这是勇气，
   不是莽撞。
⑥ 角落批注：
   「志行——
    惕惧，不是退缩，
    是走得更认真。」
⑦ 收尾精神之语：
   「同样的虎尾，
    六三被咬，
    九四终吉——
    差别不在虎，
    在那双脚，
    知不知道
    自己在哪里。」

【配色】
深墨·虎纹为主，
虎尾用深赭虎纹，
谨慎的脚用深墨，
前方终吉之光用冷白，
背景用深沉暗色，
整体谨慎、深沉、
有惕惧中前行的张力，
是履卦六爻中
最「慎」的一张，
慎，是终吉的原因。

16:9横幅，宋代险道意境，
现代平面设计结构。
```

## 九五 · 夬履，贞厉

```
易经爻辞知识图鉴
履卦 · 九五 · 夬履，贞厉

【画面主体意象】
果决而履，位正须警——
九五是履卦最「刚」的一爻，
「夬履」——
果决地行走，
「夬」，是决断，
是毫不犹豫，
「贞厉」——
守正，但须警惕，
「位正当也」——
位置正，
但正因为位置正，
更须警惕。

画面主体：
一条笔直向前的道，
没有弯曲，
没有犹豫——
「夬履」，
就是这条道的样子，
果决，直。

道的质感：
不是素履的朴素，
不是坦坦的宽阔，
是「夬」的果决——
窄，但直，
有一种
「我知道我要去哪里，
我就去」
的决断感。

天空：
乾卦之天，
高远，
「位正当也」——
九五，
站在最正的位置，
天，在上方，
是见证。

「贞厉」的意象：
道路两侧，
有极淡的险阻，
提醒着——
果决，
不等于无险，
「厉」，
是果决之人
最需要的警醒。

画面整体：
刚健，果决，
但有一丝
「须警」的张力——
不是凶，
是「贞厉」——
守正，
并且，警醒。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
九二 ━━  （淡）
六三 ━ ━ （淡）
九四 ━━  （淡）
九五 ━━  高亮天青刚金 ← 当前爻
上九 ━━  （淡）
标注：「位正当也，夬履贞厉」

【嵌入式大字】
「九五」嵌入果决道路
几何直线结构，
字体有刚健感，
字色用天青刚金，
被笔直道路
正中穿过——
「夬」，
字，也是果决的。

【笔刷动线】
一条完全笔直、
果决向前的笔刷，
没有弯曲，
没有停顿，
在道路正中
一笔而就——
「夬履」，
就是这条线
走得最直的样子。

【信息模块】
① 履卦 · 九五（右上角）
② 爻辞：「夬履，贞厉。」
③ 小象传：「夬履贞厉，位正当也。」
④ 彖传节录：「刚中正，
   履帝位而不疚，光明也。」
⑤ 爻位说明：阳爻居五，
   刚健中正，
   果决而行，
   位正当，
   但须守正警惕，
   贞厉。
⑥ 核心义理：果决，
   是九五的力量，
   也是九五的风险。
   「贞厉」——
   越是位正，
   越须警惕，
   因为，
   位正的人，
   一旦失正，
   代价最大。
⑦ 角落批注：
   「位正当——
    正，是力量，
    也是责任，
    更是须警的原因。」
⑧ 收尾精神之语：
   「果决地走，
    是九五的气质；
    守正而警，
    是九五的功课。
    光明，
```
