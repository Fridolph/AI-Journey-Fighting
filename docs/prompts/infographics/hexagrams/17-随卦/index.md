# 随卦（䷐）· 六爻提示词

> 卦序：第 17 卦 | 卦象：兑上震下 | 泽雷随 | 卦德：随

随卦是易经最「活」的一卦——泽中有雷，雷入泽中而静，是刚来下柔、动而说的智慧。「随时」——随，是知道什么时候该随什么。六爻：**官渝从正→系小子→系丈夫→随有获→孚于嘉→拘系维之**。

## 随卦「随时」的四层智慧

> 随什么？→ 随嘉（九五：孚于嘉）
> 随谁？→ 随正（初九：从正吉）
> 随到哪？→ 随时（彖传：天下随时）
> 随的极处？→ 亨于西山（上六）

## 豫与随的深层对照

| | 豫卦 | 随卦 |
|--|------|------|
| **卦象** | 雷出地奋 | 泽中有雷 |
| **核心** | 顺以动 | 随时 |
| **最深一句** | 豫之时义大矣哉 | 随之时义大矣哉 |

豫是「顺以动」，随是「动而说」。两卦都以「时义大矣哉」结彖，是易经对「时」的最深致敬。
