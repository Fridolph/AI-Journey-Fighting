# 咸卦（䷞）· 六爻提示词

> 卦序：第 31 卦 &nbsp;|&nbsp; 卦象：兑上艮下 &nbsp;|&nbsp; 泽山咸 &nbsp;|&nbsp; 卦德：感

咸——感也。柔上而刚下，二气感应以相与。「天地感而万物化生，圣人感人心而天下和平」——这是易经中最「感应相与」的一卦。

## 咸卦「观其所感而天地万物之情可见矣」的深层结构

> 咸拇志外（初六）→咸腓居吉（六二）→咸股执随（九三）→憧憧往来心感（九四）→咸脢志末（九五）→滕口说最浅（上六◯）
> 感应从拇趾到口舌从深到浅——九四憧憧往来是心感最真；上六滕口说是口舌最浅；真正的感应在心不在口舌。

## 离与咸的深层对照

| | 离卦 | 咸卦 |
|--|------|------|
| **核心** | 重明以丽乎正，日月丽乎天 | 天地感万物化生，圣人感人心天下和平 |
| **最深一句** | 黄离，元吉 | 憧憧往来，朋从尔思 |

离是光明附丽，咸是感应相与。上经以离终——光明附丽而后感应相与，下经以咸始。二卦是上下经的过渡桥梁。

## 初六 · 咸其拇

```text
易经爻辞知识图鉴
咸卦 · 初六 · 咸其拇

【画面主体意象】咸其拇，志在外也——初六是「初感」的一爻，「咸其拇」——感应于拇趾；「志在外也」——志向在外。
感应从拇趾开始从身体最低处萌动。六爻感应部位：初六拇趾→六二小腿→九三大腿→九四心→九五脊背→上六口舌。

【几何爻位结构】咸卦从下到上（兑上艮下）：初六 ━ ━  高亮深赭初感 ← 当前爻 六二 ━ ━ 九三 ━━ 九四 ━━ 九五 ━━ 上六 ━ ━
标注：「山上有泽，咸；君子以虚受人」

【嵌入式大字】「初六」嵌入咸其拇志外几何结构，字色用深赭，被初感志外的萌动部分遮挡——
「咸其拇」，字，咸拇，志外，有咸卦最初拇趾初感志在外。

【信息模块】① 咸卦 · 初六（右上角）② 爻辞：「咸其拇。」③ 小象传：「咸其拇，志在外也。」
④ 彖传节录：「观其所感，而天地万物之情可见矣！」⑤ 核心义理：咸道起点——拇趾初感志在外；感应从最低处开始。
⑨ 收尾精神之语：「感应于拇趾——志在外；咸其拇是感应最初萌动；感应从最低处开始。」

【配色】深赭·初感为主，咸拇用深赭，志外用暗金，背景用深蓝山泽。
16:9横幅，宋代初感萌动意境，现代平面设计结构。
```
