# 蛊卦（䷑）· 六爻提示词

> 卦序：第 18 卦 &nbsp;|&nbsp; 卦象：艮上巽下 &nbsp;|&nbsp; 山风蛊 &nbsp;|&nbsp; 卦德：事

蛊卦，是易经最「振民育德」的一卦——山下有风，风蚀山根，是蛊最深的意象。「刚上而柔下，巽而止，蛊」——刚上柔下，顺而止，是蛊最深的根基。「终则有始，天行也」——终而复始，是蛊最深的宇宙观。「先甲三日，后甲三日」——是改革前后最深的谋虑。六爻是蛊的六种面貌：**干父之蛊→干母之蛊→干父小有晦→裕父之蛊→干父用誉→不事王侯**，是一部从意承考到志可则的完整蛊卦图鉴。

## 蛊卦「终则有始天行也」的深层结构

> 干父意承考（初六）→干母中道（九二）→干父小晦（九三）→裕父往未得（六四）→干父承以德（六五）→不事高尚志可则（上九◯）
> 干与裕的对比——干是担当厉终吉，裕是放任往未得；从意承考到志可则是蛊最完整的精神历程。

## 随与蛊的深层对照

| | 随卦 | 蛊卦 |
|--|------|------|
| **核心** | 天下随时，动而说 | 终则有始，天行也 |
| **最深一句** | 随之时义大矣哉 | 终则有始，天行也 |

随是随时而动，蛊是积弊待治。随时而后知积弊——随与蛊，是从顺动到改革的两个阶段。

## 初六 · 干父之蛊，有子，考无咎，厉终吉

```text
易经爻辞知识图鉴
蛊卦 · 初六 · 干父之蛊，有子，考无咎，厉终吉

【画面主体意象】意承考，厉终吉——画面以「山下有风」为整体背景，风蚀山根是蛊卦最根本的气息。

初六，「干父之蛊，有子，考无咎，厉终吉」——整治父亲积弊有子承担则父无咎艰厉终吉；「意承考也」。

画面主体：干父意承考厉终吉——「干父之蛊」是蛊卦最初的担当；「意承考也」心意承担父考是干父之蛊最深的根基。
厉终吉——艰厉终吉，是意承考最深的结果。

【几何爻位结构】蛊卦六爻从下到上（艮上巽下）：初六 ━ ━  高亮深赭意承 ← 当前爻 九二 ━━ 九三 ━━ 六四 ━ ━ 六五 ━ ━ 上九 ━━
标注：「艮上巽下，山下有风，蛊」

【嵌入式大字】「初六」嵌入干父意承考几何结构，字色用深赭，被意承考的厚重部分遮挡——
「干父之蛊，有子，考无咎，厉终吉」，字，干父，意承，有蛊卦最初的意承考厉终吉。

【信息模块】① 蛊卦 · 初六（右上角）② 爻辞：「干父之蛊，有子，考无咎，厉终吉。」③ 小象传：「干父之蛊，意承考也。」
④ 彖传节录：「终则有始，天行也。」⑤ 核心义理：蛊的起点是「意承考」——心意承担是蛊最初的担当智慧。
⑨ 收尾精神之语：「整治父亲积弊——有子承担父考无咎；艰厉终吉；意承考也——心意承担是蛊最初的担当。」

【配色】深赭·意承为主，干父之蛊用深赭，意承考用暗赭，厉终吉用暖金，背景用深绿山风。
16:9横幅，宋代山风蛊意承意境，现代平面设计结构。
```

## 九二 · 干母之蛊，不可贞

```text
易经爻辞知识图鉴
蛊卦 · 九二 · 干母之蛊，不可贞

【画面主体意象】得中道，干母不可贞——九二是蛊卦最「中道」的一爻，「干母之蛊」——整治母亲积弊，
「不可贞」——不可过于贞正；「干母之蛊，得中道也」。

画面主体：干母中道不可贞——整治母亲积弊与整治父亲不同——母柔顺，需柔顺中道不可过于贞正。
与初六对比——干父需刚健担当，干母需柔顺中道，是蛊卦最深的刚柔对比。

【几何爻位结构】六爻：初六 ━ ━（淡）九二 ━━  高亮暖赭中道 ← 当前爻 九三 ━━（淡）六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━（淡）
标注：「干母之蛊，得中道也」

【嵌入式大字】「九二」嵌入干母中道几何结构，字色用暖赭，被中道的温暖部分遮挡——
「干母之蛊，不可贞」，字，干母，中道，有蛊卦中最深的中道不可贞。

【信息模块】① 蛊卦 · 九二（右上角）② 爻辞：「干母之蛊，不可贞。」③ 小象传：「干母之蛊，得中道也。」
④ 核心义理：蛊中最「中道」的干母——干母需柔顺中道不可过于贞正；中道是蛊最深的处世智慧。
⑥ 收尾精神之语：「整治母亲积弊——不可过于贞正；得中道也——干母需柔顺中道不可贞。」

【配色】暖赭·中道为主，干母之蛊用暖赭，得中道用深赭，不可贞用暗灰，背景用深绿山风。
16:9横幅，宋代干母中道意境，现代平面设计结构。
```

```text

从意承考到志可则，是蛊最完整的历程——振民育德，终于高尚其事。

【几何爻位结构】六爻：初六 ━ ━（淡）九二 ━━（淡）九三 ━━（淡）六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━  高亮暗金志可则 ← 当前爻

【嵌入式大字】「上九」嵌入不事高尚志可则几何结构，字色用暗金，被志可则的高远部分遮挡——
「不事王侯，高尚其事」，字，不事，志可则，有蛊卦终结最深的志可则高尚。
```
