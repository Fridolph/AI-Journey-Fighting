# 蛊卦（䷑）· 六爻提示词

> 卦序：第 18 卦 | 卦象：艮上巽下 | 山风蛊 | 卦德：治

蛊卦是易经最「深沉」的一卦——山下有风，风在山下回旋，积久成蛊。蛊是腐败，但蛊卦「元亨」——正因为有蛊，才有整治，才有新生。六爻：**干父→干母→干父小晦→裕父→干父用誉→高尚其事**。

## 「先甲后甲」的深层结构

> 先甲三日 → 甲（新始）→ 后甲三日
> 终则有始，天行也——蛊是终，整治是始。

## 随与蛊的深层对照

| | 随卦 | 蛊卦 |
|--|------|------|
| **核心** | 随时 | 终则有始 |
| **君子之道** | 晦入宴息 | 振民育德 |
| **最深一句** | 随之时义大矣哉 | 先甲三日，后甲三日 |

随是「顺时而随」，蛊是「知终而始」。有蛊，才有整治；有终，才有新始。
