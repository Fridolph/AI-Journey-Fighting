# 需卦（䷄）· 六爻提示词

> 卦序：第 5 卦 &nbsp;|&nbsp; 卦象：坎上乾下 &nbsp;|&nbsp; 水天需 &nbsp;|&nbsp; 卦德：等

需卦的气质非常独特——不是难，不是昧，是**等**。云在天上，雨还没下，刚健之人面对前方的险阻，不是莽进，是等待时机。六爻是等待的六种处境：**郊野→沙滩→泥泞→血地→酒食→穴中**，距离险阻越来越近，等待的代价越来越重，直到最后意外的转机。

---

## 总体设计逻辑

| 爻位 | 爻辞 | 等待处境 | 情感质地 | 主色调 |
|------|------|---------|---------|--------|
| 初九 | 需于郊 | 郊野远候，从容守常 | 从容·远观·无咎 | 旷野赭·淡云 |
| 九二 | 需于沙 | 沙滩近险，小有言 | 宽裕·稍近·终吉 | 沙黄·水蓝 |
| 九三 | 需于泥 | 泥泞贴险，致寇至 | 危迫·自招·警慎 | 泥褐·险灰 |
| 六四 | 需于血 | 血地深陷，顺以听 | 险极·顺从·出穴 | 玄黑·血色 |
| 九五 | 需于酒食 | 中正待时，贞吉 | 从容·中正·安然 | 暖金·酒红 |
| 上六 | 入于穴，不速之客 | 穴中意外，敬之吉 | 意外·转机·敬慎 | 深穴墨·点金 |

## 初九 · 需于郊，利用恒，无咎

```
易经爻辞知识图鉴
需卦 · 初九 · 需于郊，利用恒，无咎

【画面主体意象】
郊野远候——
画面以开阔的郊野为主体，
天地辽阔，
云在天上积聚，
雨尚未落，
人在郊外，
远远地等着。

画面构图：
前景是郊野的草地，
平旷，有风，
草随风微动，
是「恒」的感觉——
不急，不躁，
只是在这里，
持续地在这里。

中景：
一条通向远方的路，
路的尽头，
是险阻的方向——
云层在那里积聚，
但还很远，
「不犯难行也」——
知道险在前，
所以在郊野停下来，
不是退缩，是智慧。

天空：
云层在远处积聚，
但郊野这里，
天还是开阔的，
有一种
「险在前，我在后」
的从容距离感。

画面一角：
一个背影，
站在郊野，
面向远方，
姿态平静——
「利用恒」，
恒，是这个背影
最准确的注解。

【几何爻位结构】
需卦六爻从下到上
（坎上乾下）：
初九 ━━  高亮旷野赭 ← 当前爻
九二 ━━
九三 ━━
六四 ━ ━
九五 ━━
上六 ━ ━
标注：「乾下坎上，险在前也」

【嵌入式大字】
「初九」嵌入郊野天际线
的几何层，
字色用淡赭，
被草地纹理和远云
部分遮挡，
整体可读，
有一种「郊」的开阔感——
字，和郊野一样，
不紧张。

【笔刷动线】
一条从左至右
极缓慢流动的横向笔刷，
模拟郊野的风，
笔触轻，
有飞白，
在背影处微微停顿，
然后继续向远方延伸——
「恒」，就是这条线
一直走下去的样子。

【信息模块】
① 需卦 · 初九（右上角）
② 卦象标注：水天需 · 坎上乾下
③ 爻辞：「需于郊。
   利用恒，无咎。」
④ 小象传：「需于郊，不犯难行也。
   利用恒，无咎；未失常也。」
⑤ 彖传节录：「需，须也；
   险在前也。刚健而不陷，
   其义不困穷矣。」
⑥ 爻位说明：阳爻居初，
   乾卦之始，
   距险最远，
   守常待时，
   是需卦最从容的处境。
⑦ 核心义理：等待，
   不是无所作为，
   是在最远的地方，
   把自己守得最稳。
   恒，是等待的根基。
⑧ 角落批注：「未失常也」
⑨ 收尾精神之语：
   「险在前，
    我在郊野，
    不是因为怕，
    是因为时候还没到。」

【配色】
旷野赭·淡云为主，
草地用温润赭黄，
天空用淡墨蓝，
远处云层用深灰，
背影用深墨，
整体开阔、从容、有风，
是需卦六爻中
距离险阻最远、
最「安静」的一张。

16:9横幅，宋代平远郊野意境，
现代平面设计结构。
```

## 九二 · 需于沙，小有言，终吉

```
易经爻辞知识图鉴
需卦 · 九二 · 需于沙，小有言，终吉

【画面主体意象】
沙滩近险，宽裕中等——
画面以水边沙滩为主体，
沙，是水与陆的交界，
比郊野更近险（坎水），
但还没有陷入。

前景是宽阔的沙滩，

「小有言」——
有一点争议，
有一点不安，
但终究，
方向是对的，
「终吉」。

天空：
云层比初九更厚，
更近，

「终吉」的光，

九五 ━━  （淡）

   小挫不改其志，
   终吉。
⑤ 核心义理：沙，比郊野软，
   比泥还稳。
   小有言，是等待中
   难免的摩擦，
   但终吉，
   是中道的保证。
⑥ 角落批注：
   「虽小有言，以终吉——
    过程有波折，
    方向没有错。」
⑦ 收尾精神之语：
   「沙地会陷，
    但还没陷，
    中道在，
    终究会吉。」

【配色】
沙黄·水蓝为主，
沙滩用温润沙黄，
水波用淡墨蓝，
天空用厚云深灰，
云隙光用冷白，
整体比初九更近险，
但仍有宽裕感，
沙的温度，
是这一爻的情绪底色。
```

## 九三 · 需于泥，致寇至

```
易经爻辞知识图鉴
需卦 · 九三 · 需于泥，致寇至

【画面主体意象】
泥泞贴险，自招其祸——
九三是需卦最具警示性的一爻，
「需于泥」——等在泥泞里，
「致寇至」——
自己招来了祸患，
「自我致寇」。

画面主体：
深陷的泥地，
脚印深深陷入，
泥浆从四周涌来，
险（坎水）就在眼前——
不是险来找他，
是他走得太近了。

泥地中：
一个人的轮廓，
半陷在泥中，
姿态狼狈，
但眼睛还看着前方——
「敬慎不败也」，
还没有败，
但必须警醒了。

远处：
「寇」的意象——
不画具体的敌人，
用逼近的阴云，
或快速移动的暗影，
代表「致寇至」——
是自己的急进，
把险阻引来的。

地面：
泥与水的交界，
越陷越深，
但还有一块干地，
在画面边缘——
「敬慎不败」，
那块干地，
还在。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
九二 ━━  （淡）
九三 ━━  高亮泥褐 ← 当前爻
六四 ━ ━ （淡）
九五 ━━  （淡）
上六 ━ ━ （淡）
标注：「灾在外也，自我致寇」

【嵌入式大字】
「九三」嵌入泥泞几何块，
字体被泥浆纹理
深度切割，
字色用深泥褐，
有一种陷入感——
字，也在泥里。

【笔刷动线】
一条急速向前冲
却在泥中骤然停顿的笔刷，
笔触从左侧干燥处
急速冲入右侧泥泞，
在泥中扭曲、停滞——
「致寇至」，
就是这个
冲过头的动势。

【信息模块】
① 需卦 · 九三（右上角）
② 爻辞：「需于泥，致寇至。」
③ 小象传：「需于泥，灾在外也。
   自我致寇，敬慎不败也。」
④ 爻位说明：阳爻居三，
   乾卦之上爻，
   最近险阻，
   刚而不中，
   急进自招其祸。
⑤ 核心义理：寇，不是从外来的，
   是自己走太近招来的。
   「自我致寇」——
   最诚实的四个字。
   但敬慎，还能不败。
⑥ 角落批注：
   「敬慎不败——
    知道是自己招来的，
    才能真正警醒。」
⑦ 收尾精神之语：
   「泥，是险的边缘。
    寇，是自己请来的。
    知道这一点，
    还不算太晚。」

【配色】
泥褐·险灰为主，
泥地用深赭褐，
泥浆用暗灰褐，
逼近的阴云用深冷灰，
那块干地用极淡暖赭，
整体压迫、危迫、
是需卦六爻中
最「险」的一张，
险，是自己走进去的。

现代平面设计结构。
```

## 六四 · 需于血，出自穴

```
易经爻辞知识图鉴
需卦 · 六四 · 需于血，出自穴

【画面主体意象】
血地深陷，顺以听也——
六四是需卦最险的处境，
「需于血」——等在血地，
已经深陷险中（坎卦之初），
但「出自穴」——
能从穴中出来，
「顺以听也」——
顺从，是这一爻的出路。

画面主体：
一个洞穴的出口，
从内向外看——
外面是险地，
有血色的光，
但穴口透出光，
是出路的方向。

穴内：
深暗，
但不是绝望的暗，
是「顺以听」的暗——
在这里等，
听从时机，
等到可以出去的那一刻。

穴口：
光从外面透入，
血色与暗金混合，
「需于血」的光——
不是美丽的光，
是险中的光，
但是光。

穴外地面：
有血色的痕迹，
但穴口的方向，
地面渐渐干净——
「出自穴」，
出去，就有路。

画面构图：
用穴内视角，
让观者感受
「顺以听」的等待——
不是被困，
是主动选择在这里，
等到出去的时机。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
九二 ━━  （淡）
九三 ━━  （淡）

九五 ━━  （淡）

   穴，有入口，就有出口。
⑥ 角落批注：

    柔顺，是最有力的武器。」

穴口用血色暗金，
穴外地面用深赭，
出路方向用极淡冷白，
整体深沉、险峻，
但有方向感——
那道穴口的光，
是这张图唯一的温度。
```

## 九五 · 需于酒食，贞吉

```
易经爻辞知识图鉴
需卦 · 九五 · 需于酒食，贞吉

【画面主体意象】
中正安然，以待其时——
九五是需卦最「安」的一爻，
「需于酒食」——
在酒食中等待，
「以中正也」——
不是享乐，
是中正之人
在等待中
保持从容的姿态。

画面主体：
一张简朴的案几，
上面摆着酒器和食物——
不是奢华的宴席，
是「饮食宴乐」的象传意象，
简单，从容，
有一种「等待本身也是生活」
的智慧。

酒器：
古朴的陶器或青铜器，
酒色深沉，
暗示「有孚」——
内有诚信，
外有从容。

案几周围：
云在天上（坎卦），
但这里是天位（九五），
云，是等待的背景，
不是威胁——
「位乎天位，以正中也」，
在最正的位置，
等待，是最有力量的行动。

窗外或背景：
云层依然厚重，
雨还没下，
但这里的光，
是全卦最稳的光——
中正之光，
不因外险而动摇。

【几何爻位结构】
六爻从下到上：
初九 ━━  （淡）
九二 ━━  （淡）
九三 ━━  （淡）
六四 ━ ━ （淡）
九五 ━━  高亮暖金 ← 当前爻
上六 ━ ━ （淡）

「九五」嵌入案几背景
的几何暖块，

整体温暖、从容、有生活气息，
是需卦六爻中
最「安」的一张，
```

## 上六 · 入于穴，有不速之客三人来，敬之终吉

```
易经爻辞知识图鉴
需卦 · 上六 · 入于穴，有不速之客三人来，敬之终吉

【画面主体意象】
穴中意外，敬之终吉——
上六是需卦最出人意料的结尾，
「入于穴」——已经退入穴中，
「有不速之客三人来」——
没有邀请，三人突然到来，
「敬之终吉」——
以敬待之，终究吉祥。

画面主体：
一个洞穴内部，
与六四的穴不同——
六四是险中之穴，
上六的穴是退守之穴，
是主动退入的，
有一种「山穷水尽」
之后的意外转机。

穴口：
三个身影，
逆光而来，
轮廓清晰，
但面目不明——
「不速之客」，
不知道是谁，
不知道来意，

穴内的人：
面对三个来客，
姿态是「敬」的——
不是惊慌，
不是拒绝，
是起身，
是以礼相待。

穴内的光：
三个来客带来了光，
或者说，
他们的到来，
让穴内有了光——
「终吉」，
就是这道光。

画面氛围：
意外，但不惊慌；
陌生，但有礼；
困境，但有转机——
这是上六最准确的情绪。
```
