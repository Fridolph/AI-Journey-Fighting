# 谦卦（䷎）· 六爻提示词

> 卦序：第 15 卦 | 卦象：坤上艮下 | 地山谦 | 卦德：谦

谦卦是易经最「深」的一卦——地中有山，高山藏于大地之下，不显其高。彖传说：天道、地道、鬼神、人道，四者皆「亏盈益谦」——谦，是宇宙最普遍的规律。六爻是谦的六种面貌：**谦谦自牧→鸣谦中得→劳谦万服→撝谦不违→侵伐征不服→鸣谦行师**。

## 谦卦彖传四道（宇宙最普遍的规律）

> 天道亏盈而益谦 → 地道变盈而流谦 → 鬼神害盈而福谦 → 人道恶盈而好谦

四道皆谦——谦，是宇宙最普遍的规律。

## 大有与谦的深层对照

| | 大有卦 | 谦卦 |
|--|--------|------|
| **卦象** | 火在天上 | 地中有山 |
| **君子之道** | 顺天休命 | 裒多益寡，称物平施 |
| **核心** | 持盈守正 | 卑以自牧 |
| **最深一句** | 自天佑之 | 谦尊而光，卑而不可踰 |

大有是「有」的极致，谦是「有」之后的智慧。大有而能谦，才是「自天佑之」最深的原因。

> 彖传：「谦尊而光，卑而不可踰」——谦让尊者更光明，卑让人无法逾越。山，藏在地下，不是因为山不高，是因为山，知道真正的高，不需要显出来。
