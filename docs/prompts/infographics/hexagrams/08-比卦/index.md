# 比卦（䷇）· 六爻提示词

> 卦序：第 8 卦 &nbsp;|&nbsp; 卦象：坎上坤下 &nbsp;|&nbsp; 水地比 &nbsp;|&nbsp; 卦德：附

比卦的气质和师卦形成美妙的对照——师是众人听令于一，比是众人亲附于上。地上有水，水流亲地，自然相附。六爻是亲附的六种面貌：**诚附→内附→附匪人→外附贤→显比王道→无首而凶**，是一部关于「亲」与「附」的人心图鉴。

## 初六 · 有孚比之，有孚盈缶，终来有他，吉

```
易经爻辞知识图鉴
比卦 · 初六 · 有孚比之，有孚盈缶，终来有他，吉

【画面主体意象】
诚信盈满，自然亲附——
画面以一只朴素的陶缶
为绝对主体，
「有孚盈缶」——
缶，是最朴素的器皿，
不是华丽的青铜，
是民间日常的陶罐，
但它满了，
满的是「孚」——诚信。

陶缶的状态：
满而不溢，
缶口微微透出光——
不是刻意展示，
是诚信本身
自然流露的光。

缶的周围：
「终来有他」——
有人循着这道光走来，
不是一个人，
是「他」——
意外的、
不期而至的亲附，
因为诚信，
所以有人来。

地面：
水流亲地的意象，
极淡的水纹，
从四方向缶的方向
自然汇聚——
「地上有水，比」，
水，自然流向低处，
诚信，自然吸引亲附。

背景：
简朴，开阔，
是「比之初六」的气质——
不华丽，
但真实，
真实，所以吉。

【几何爻位结构】
比卦六爻从下到上
（坎上坤下）：
初六 ━ ━  高亮陶土赭 ← 当前爻
六二 ━ ━
六三 ━ ━
六四 ━ ━
九五 ━━
上六 ━ ━
标注：「坤下坎上，地上有水」

【嵌入式大字】
「初六」嵌入陶缶
几何圆弧结构，
字色用温润陶土赭，
被缶身和水纹
部分遮挡，
有一种「孚」的
朴素温度——
字，像缶一样，
不华丽，但实在。

【笔刷动线】
一条从四方向中心
自然汇聚的笔刷，
模拟水流亲地之势，
在陶缶处汇聚，
形成温暖的中心——
「有孚」，
就是这个
让水自然流来的东西。

【信息模块】
① 比卦 · 初六（右上角）
② 卦象标注：水地比 · 坎上坤下
③ 爻辞：「有孚比之，无咎。
   有孚盈缶，终来有他，吉。」
④ 小象传：「比之初六，有他吉也。」
⑤ 象传节录：「地上有水，比；
   先王以建万国，亲诸侯。」
⑥ 彖传节录：「比，吉也，
   比，辅也，下顺从也。」
⑦ 爻位说明：阴爻居初，
   比之始，
   以诚信为本，
   不求华丽，
   终有意外之吉。
⑧ 核心义理：亲附的根基，
   不是利益，
   是诚信。
   缶，是最朴素的容器，
   但满了孚，
   比任何华器都吉。
⑨ 角落批注：「有他吉——
   意外来的，
   才是真的。」
⑩ 收尾精神之语：
   「不是你去找人，
    是你的诚信，
    让人找到你。
    缶满了，
    人自然来。」

【配色】
陶土赭·温暖为主，
陶缶用温润赭黄，
缶口光用暖白，
水纹用极淡冷蓝，
背景用简朴深赭，
整体朴素、温暖、真实，
是比卦六爻中
最「朴」的一张，
朴，是诚信的颜色。

现代平面设计结构。
```

## 六二 · 比自内，贞吉

```
易经爻辞知识图鉴
比卦 · 六二 · 比自内，贞吉

【画面主体意象】
由内而附，不自失——
六二是比卦最「正」的亲附，
「比自内」——
亲附从内心出发，
不是外力驱使，
「不自失也」——
在亲附中，
没有失去自己。

画面主体：
一个内室，
或一个人的内心空间——
不是外面的广场，
是里面的庭院，
安静，有光，
光从内部透出，
不是外面照进来的。

内室中心：
一盏灯，
自己点亮的，
「比自内」——
亲附，
从这盏灯开始，
从内心的诚意开始。

内室的门：
半开，
朝向外面，
暗示「贞吉」——
内守而外应，
门开着，
但根在里面。

画面构图：
内外两层——
内层温暖，有光，有序；
外层隐约可见
外面的世界，
但内层的光，
不因外面而动摇——
「不自失」，
就是这个
内外分明的结构。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  高亮内光赭 ← 当前爻
六三 ━ ━  （淡）
六四 ━ ━  （淡）
九五 ━━   （淡）
上六 ━ ━  （淡）
标注：「比之自内，不自失也」

【嵌入式大字】
「六二」嵌入内室几何结构，
字色用深内光赭，
被灯光和内室轮廓
部分遮挡，
字在内室里，
不在外面——
「自内」，
字，也从内部透出。

【笔刷动线】
一条从内室中心
向外缓慢扩散的笔刷，
模拟内光向外渗透之势，
在灯处最浓，
向外渐淡——
「比自内」，
光，从里面出来。

【信息模块】
① 比卦 · 六二（右上角）
② 爻辞：「比自内，贞吉。」
③ 小象传：「比之自内，不自失也。」
④ 爻位说明：阴爻居二，
   得中得正，
   亲附九五，
   由内而发，
   守正不失己，
   贞吉。
⑤ 核心义理：亲附，
   最怕失去自己。
   「不自失」——
   在靠近别人的过程中，
   还知道自己是谁，
   这是比卦最难的功课。
⑥ 角落批注：
   「不自失——
    附，但不消失。
    这才是真正的比。」
⑦ 收尾精神之语：
   「从内心出发的亲附，
    不会让你失去自己。
    灯，是自己点的，
    光，才是自己的。」

【配色】
深赭·内光为主，
内室用深赭褐，
灯光用温润暖金，
内室门用深墨，
门外用极淡冷灰，
整体温暖、内敛、有自持，
是比卦六爻中
最「内」的一张，
内，是不自失的根基。

## 六三 · 比之匪人

```
易经爻辞知识图鉴
比卦 · 六三 · 比之匪人

【画面主体意象】
亲附非人，不亦伤乎——
六三是比卦最具警示性的一爻，
「比之匪人」——
亲附了不该亲附的人，
象传只说了四个字：
「不亦伤乎！」——
一个感叹，
比任何解释都重。

画面主体：
两个身影，
一个在前，一个在后，
后者紧紧跟随前者——
看起来是亲附，
但前者的方向，
是错的——
走向黑暗，
走向悬崖，
或走向迷雾。

跟随者的状态：
还没有意识到，
或者刚刚意识到——
脚步微微迟疑，
但还在跟着。

前方的意象：
不画具体的「匪人」，
用错误的方向来表达——
一条路，
看起来有光，
但那光，
是虚假的光，
「匪人」，
就是那个
带你走向虚假之光的人。

画面背景：
正确的方向，
在画面另一侧，
极淡，极远，
「不亦伤乎」——
伤，是因为
正确的路，
就在那里，
但选错了人。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  高亮暗灰 ← 当前爻
六四 ━ ━  （淡）
九五 ━━   （淡）
上六 ━ ━  （淡）
标注：「比之匪人，不亦伤乎」

【嵌入式大字】
「六三」嵌入错误方向
的几何暗块，
字色用冷暗灰，
被虚假之光
和错误身影遮挡，
字，有一种
被带偏的感觉——
跟错了人，
连字，都走歪了。

【笔刷动线】
一条本来方向正确
却在中途急转
走向错误方向的笔刷，
转折处有犹豫的痕迹，
但最终还是转了——
「比之匪人」，
就是这个
转错了的方向。

【信息模块】
① 比卦 · 六三（右上角）
② 爻辞：「比之匪人。」
③ 小象传：「比之匪人，不亦伤乎！」
④ 爻位说明：阴爻居三，
   失位不中，
   上下皆阴，
   无正应，
   所比非人，
   伤也。
⑤ 核心义理：比卦最短的爻辞，
   却是最重的警示。
   「匪人」，
   不一定是坏人，
   是不该亲附的人——
   方向错了，
   越亲近，越伤。
⑥ 角落批注：
   「不亦伤乎——
    易经少有的感叹句，
    是真的心疼。」
⑦ 收尾精神之语：
   「跟对人，
    是一生最重要的选择。
    比之匪人，
    不是罪，
    是伤——
    易经，心疼你。」

【配色】
暗灰·冷警为主，
错误方向用虚假暗金，
跟随者用深墨，
正确方向用极淡暖光，
背景用冷深灰，
整体警示、冷峻、有伤感，
是比卦六爻中
最「警」的一张，
警，是因为心疼。

16:9横幅，宋代人物叙事意境，
现代平面设计结构。
```

## 六四 · 外比之，贞吉

```
易经爻辞知识图鉴
比卦 · 六四 · 外比之，贞吉

【画面主体意象】
外附于贤，顺上而吉——
六四是比卦最「顺」的亲附，
「外比之」——
向外亲附，
「外比于贤，以从上也」——
附的是贤者（九五），
方向向上，
贞吉。

画面主体：
一条向上的山路，
或一条朝向高处的台阶，
画面中心是一个身影，
正在向上走——
不是攀爬，
是从容地走，
方向清晰，
步伐稳定。

上方：
九五的光，
从高处透下，
「外比于贤」——
那道光，
是贤者的光，
值得走向它。

身影的姿态：
不是仰望，
是行走——
「以从上也」，
从，是主动的选择，
不是被迫的跟随。

画面构图：
有明显的
由下向上的方向感，
与六三的
被带向错误方向
形成鲜明对比——
同样是「外比」，
方向对了，
就是贞吉。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  （淡）
六四 ━ ━  高亮清青 ← 当前爻
九五 ━━   （淡金，在上方）
上六 ━ ━  （淡）
标注：「外比于贤，以从上也」

【嵌入式大字】
「六四」嵌入向上山路
的几何层，
字色用清青，
被上方透下的光
和山路轮廓
部分遮挡，
字，在向上走的路上——
「外比」，
字，也在往上走。

【笔刷动线】
一条从下向上
清晰延伸的笔刷，
方向明确，
不急不缓，
在九五光源处
自然汇入——
「以从上」，
就是这条线
走得最正的样子。

【信息模块】
① 比卦 · 六四（右上角）
② 爻辞：「外比之，贞吉。」
③ 小象传：「外比于贤，以从上也。」
④ 爻位说明：阴爻居四，
   近九五，
   外附于贤，
   顺上而行，
   贞吉。
⑤ 核心义理：外比，
   不是失去自己，
   是找到值得跟随的方向。
   六二是比自内，
   六四是比自外——
   内外皆正，
   皆吉。
⑥ 角落批注：
   「以从上——
    从，是主动的，
    不是被动的。
    选择跟随，
    本身就是智慧。」
⑦ 收尾精神之语：
   「向上走，
    不是因为上面有权力，
    是因为上面有贤，
    值得走过去。」

【配色】
清青·向上为主，
山路用深青灰，
上方光源用冷白金，
身影用深墨，
天空用清透深蓝，
整体清朗、向上、有方向，
是比卦六爻中
最「顺」的一张，
顺，是找到了正确方向
之后的从容。

## 九五 · 显比，王用三驱，失前禽，邑人不诫，吉

```
易经爻辞知识图鉴
比卦 · 九五 · 显比，王用三驱，失前禽，邑人不诫，吉

【画面主体意象】
王道显比，舍逆取顺——
九五是比卦最磅礴的一爻，
「显比」——
光明正大地亲附，
「王用三驱」——
王者田猎，
三面驱赶，
留一面让禽兽逃走，
「失前禽」——
故意放走迎面而来的，
「舍逆取顺」——
只取顺从的，
不强求逆走的。

画面主体：
王者田猎的场景——
三面的驱赶者，
围成一个开口的阵势，
前方留有一个缺口——
那是给「前禽」的出路，
「失前禽」，
不是失误，
是王道的胸怀。

画面中心：
那个开口，
是整张图最重要的地方——
不是围死，
是留路，
「舍逆取顺」，
王道，从这个开口开始。

邑人：
散落在田野各处，
没有戒备，
「邑人不诫」——
王的亲附，
让百姓安心，
不需要戒备。

天空：
开阔，
比任何一爻都开阔——
「显比」，
是光明正大的，
天地都看得见。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  （淡）
六四 ━ ━  （淡）
九五 ━━   高亮天青王金 ← 当前爻
上六 ━ ━  （淡）
标注：「位正中也，舍逆取顺」

【嵌入式大字】
「九五」嵌入田猎开阔
的几何天地结构，
字色用王金，
被三驱阵势
和那个开口
部分遮挡——
「显比」，
字，是光明正大的，
遮挡，也遮不住。

【笔刷动线】
三条笔刷，
从三个方向
向中心汇聚，
但在中心留出一个缺口——
「三驱」，
就是这三条线，
「失前禽」，
就是那个缺口，
王道，在那个缺口里。

【信息模块】
① 比卦 · 九五（右上角）
② 爻辞：「显比，王用三驱，
   失前禽。邑人不诫，吉。」
③ 小象传：「显比之吉，位正中也。
   舍逆取顺，失前禽也。
   邑人不诫，上使中也。」
④ 彖传节录：「原筮元永贞，无咎，
   以刚中也。」
⑤ 爻位说明：阳爻居五，
   刚健中正，
   比卦之主，
   王道显比，
   舍逆取顺，
   天下归附。
⑥ 核心义理：王道的亲附，
   不强求，
   不围死，
   留一个缺口，
   让不愿来的走——
   这才是「显比」，
   光明正大的亲附。
⑦ 角落批注：
   「失前禽——
    故意放走的，
    是王道最大的胸怀。」
⑧ 收尾精神之语：
   「真正的王道，
    不是把所有人都围住，
    是留一个口，
    让愿意来的来，
    让不愿意的走。
    邑人不诫，
    才是真的吉。」

【配色】
天青·王金为主，
田野用开阔赭黄，
三驱阵势用深墨，
那个缺口用冷白开阔，
天空用最开阔的天青，
邑人用极淡暖赭，
整体开阔、王道、
有天地大气之感，
是比卦六爻中
最「显」的一张，
显，是光明正大，
无所遮蔽。

## 上六 · 比之无首，凶

```
易经爻辞知识图鉴
比卦 · 上六 · 比之无首，凶

【画面主体意象】
无所归附，无所终也——
上六是比卦最孤绝的结尾，
「比之无首」——
亲附却没有首领，
没有方向，
「无所终也」——
没有归宿，
没有终点，
凶。

画面主体：
一片散乱的水，
没有方向，
没有归处——
「地上有水，比」，
但这里的水，
失去了地的承载，
四散流淌，

一个空位——

极淡，极远——

上六，
离它们，
太远了。

最深的玄黑，
没有光，
「无所终」——

## 师卦与比卦的深层对照

| | 师卦 | 比卦 |
|--|------|------|
| **象传** | 地中有水 | 地上有水 |
| **水的位置** | 在地中，藏 | 在地上，显 |
| **关系** | 众听令于一 | 众亲附于上 |
| **核心** | 律与正 | 孚与中 |
| **警示** | 小人勿用 | 比之无首 |

地中有水，是师——水藏于地，兵藏于民，需要律来统领；地上有水，是比——水流亲地，民附于君，需要孚来凝聚。比卦最深的智慧，在九五那个缺口——「失前禽」，故意放走不愿来的，才是真正的王道亲附。
