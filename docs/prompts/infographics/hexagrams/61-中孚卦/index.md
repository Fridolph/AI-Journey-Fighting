# 中孚卦（䷼）· 六爻提示词

> 卦序：第 61 卦 &nbsp;|&nbsp; 卦象：巽上兑下 &nbsp;|&nbsp; 风泽中孚 &nbsp;|&nbsp; 卦德：信

中孚卦，是易经最「诚信」的一卦——泽上有风，是风吹泽面、诚信感化的卦象。「中孚，柔在内而刚得中」——柔在内，刚得中，是中孚最深的根基。「信及豚鱼」——诚信，连豚鱼都能感化，是中孚最深的力量。「孚，乃化邦也」——诚信，是化邦最深的精神。「君子以议狱缓死」——中孚，是议狱缓死最深的精神。六爻是诚信的六种面貌：**虞吉志未变→鸣鹤和子→得敌鼓罢→月几望马亡→有孚挛如→翰音登天**，是一部从志未变到翰音凶的完整中孚图鉴。

## 中孚卦「信及豚鱼孚乃化邦」的深层结构

> 虞吉志未变（初九）→鹤鸣和子（九二）→得敌鼓罢（六三）→月望马亡（六四）→孚挛位正（九五）→翰音登天凶（上九◯）
> 信及豚鱼孚乃化邦——鹤鸣中心愿是真实的诚信；翰音虚声是虚浮的表演；真实与虚浮是中孚最深的分野。

## 节与中孚的深层对照

| | 节卦 | 中孚卦 |
|--|------|------|
| **核心** | 天地节而四时成，甘节居中 | 信及豚鱼，孚乃化邦 |
| **最深一句** | 苦节不可贞，其道穷也 | 鸣鹤在阴，其子和之 |

节是节制有度，中孚是诚信感化。节制而后能中孚——节而制与信而孚，是制度与诚信的两个维度。

## 初九 · 虞吉，有他不燕

```text
易经爻辞知识图鉴
中孚卦 · 初九 · 虞吉，有他不燕

【画面主体意象】志未变，虞吉不燕——画面以「泽上有风」为整体背景，风吹泽面是中孚卦最根本的气息。

初九，「虞吉，有他不燕」——虞吉是安虑则吉专一；有他不燕是有他念则不安；「志未变也」——志向未变。

画面主体：专一安虑志未变——「虞吉」是专一安虑，专一是中孚最初的根基；「有他不燕」是他念是专一最深的威胁；
「志未变也」——志向未变则专一，专一则虞吉。志未变是中孚最初的智慧。

【几何爻位结构】中孚卦六爻从下到上（巽上兑下）：初九 ━━  高亮深赭虞吉 ← 当前爻 九二 ━━ 六三 ━ ━ 六四 ━ ━ 九五 ━━ 上九 ━━
标注：「巽上兑下，泽上有风，中孚」

【嵌入式大字】「初九」嵌入虞吉志未变几何结构，字色用深赭，被泽上有风的诚信部分遮挡——
「虞吉，有他不燕」，字，虞吉，志未变，有中孚卦最初的专一诚信。

【信息模块】① 中孚卦 · 初九（右上角）② 爻辞：「虞吉，有他不燕。」③ 小象传：「初九虞吉，志未变也。」
④ 彖传节录：「信及豚鱼也。中孚以利贞，乃应乎天也。」
⑤ 核心义理：中孚的起点是「志未变」——专一安虑则虞吉；有他念则不安；志未变是诚信最初的根基。
⑨ 收尾精神之语：「专一安虑则吉——有他念则不安；初九虞吉志未变也——志向未变是诚信最初的根基。」

【配色】深赭·虞吉为主，虞吉专一用深赭，有他不燕用暗灰，志未变用暖金，背景用深蓝绿。
16:9横幅，宋代泽上有风诚信意境，现代平面设计结构。
```

## 九二 · 鸣鹤在阴，其子和之，我有好爵，吾与尔靡之

```text
易经爻辞知识图鉴
中孚卦 · 九二 · 鸣鹤在阴，其子和之，我有好爵，吾与尔靡之

【画面主体意象】中心愿，鹤鸣子和——九二是中孚卦最「感应」的一爻，「鸣鹤在阴」——鹤在阴处鸣叫，
「其子和之」——其子自然应和，「我有好爵」——我有美好爵位，「吾与尔靡之」——与你共享；「其子和之，中心愿也」。

画面主体：鹤鸣于阴子应和之——「鸣鹤在阴」诚信是不需外显，不在阳处不表演；「其子和之」是自然应和不是强求；
「我有好爵吾与尔靡之」——共享是诚信最深的感召。诚信不需外显自然感召。

与初九的对比：初九志未变专一；九二鹤鸣子和中心愿共享——从志未变专一到鹤鸣感召是升华：诚信自然感召是中孚最深的力量。

【几何爻位结构】六爻：初九 ━━（淡）九二 ━━  高亮纯金鹤鸣 ← 当前爻 六三 ━ ━（淡）六四 ━ ━（淡）九五 ━━（淡）上九 ━━（淡）
标注：「其子和之，中心愿也」

【嵌入式大字】「九二」嵌入鹤鸣子和几何结构，字色用纯金，被鹤鸣感召的光芒部分遮挡——
「鸣鹤在阴，其子和之，我有好爵，吾与尔靡之」，字，鹤鸣，中心愿，有中孚卦中最深的诚信感召。

【信息模块】① 中孚卦 · 九二（右上角）② 爻辞：「鸣鹤在阴，其子和之，我有好爵，吾与尔靡之。」
③ 小象传：「其子和之，中心愿也。」
④ 核心义理：中孚中最「感应」的鹤鸣——「鸣鹤在阴其子和之」是诚信自然感召；「中心愿也」——共享是诚信最深的表现。
⑥ 收尾精神之语：「鹤在阴处鸣叫——其子自然应和；我有好爵吾与尔靡之——诚信自然感召共享。」

【配色】纯金·鹤鸣为主，鹤鸣在阴用纯金，子和回响用暖金，好爵共享用素白，背景用深蓝绿。

```

> **「信及豚鱼，孚乃化邦」**
> ——中孚卦的终极义理，不是表面诚信，而是「中心之孚」——
> 鸣鹤在阴其子和之——诚信自然感召是最深的象征；志未变是诚信最初的根基；
> 翰音登天何可长——虚浮则贞凶；真实与虚浮是中孚最深的分野。
