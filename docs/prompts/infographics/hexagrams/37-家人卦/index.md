# 家人卦（䷤）· 六爻提示词

> 卦序：第 37 卦 &nbsp;|&nbsp; 卦象：巽上离下 &nbsp;|&nbsp; 风火家人 &nbsp;|&nbsp; 卦德：齐

## 家人卦「正家而天下定矣」的深层结构

> 闲家志正（初九）→中馈顺巽（六二）→嗃嗃未失嘻嘻失节（九三）→富家顺位（六四）→王假交相爱（九五）→有孚威如反身（上九◯）
> 严正是家道根基，交相爱是家道精神——两者缺一不可。反身是家道最终圆满。

## 明夷与家人的深层对照

| | 明夷卦 | 家人卦 |
|--|------|------|
| **核心** | 明入地中用晦而明，内难而能正其志 | 正家而天下定，父父子子夫夫妇妇 |
| **最深一句** | 箕子之贞，明不可息也 | 王假有家，交相爱也 |

明夷是晦暗中守正，家人是家道齐正。蒙大难而后知家道之重——明夷与家人，是晦明与齐家的两种坚守。

## 初九 · 闲有家，悔亡

```text
易经爻辞知识图鉴
家人卦 · 初九 · 闲有家，悔亡

【画面主体意象】闲有家，志未变也——初九是「志正」的一爻，防闲于家志未变则悔亡。
「志未变也」——志向未曾改变是防闲最深的精神。防闲是家道最初的根基。

【几何爻位结构】家人卦从下到上（巽上离下）：初九 ━━  高亮深赭志正 ← 当前爻 六二 ━ ━ 九三 ━━ 六四 ━ ━ 九五 ━━ 上九 ━━
标注：「风自火出，家人；君子以言有物而行有恒」

【嵌入式大字】「初九」嵌入闲家志正几何结构，字色用深赭，被志未变悔亡的沉稳部分遮挡——
「闲有家，悔亡」，字，闲家，志正，有家人卦最初闲家志未变。

【信息模块】① 家人卦 · 初九（右上角）② 爻辞：「闲有家，悔亡。」③ 小象传：「闲有家，志未变也。」
④ 彖传节录：「正家而天下定矣。」⑤ 核心义理：家道起点——防闲于家志未变是根基。
⑨ 收尾精神之语：「防闲于家——悔亡；志未变也——志未变是家道最初精神。」

【配色】深赭·志正为主，闲家用深赭，志未变用暖赭，悔亡用深金，背景用深红风火。
16:9横幅，宋代闲家志正意境，现代平面设计结构。
```
