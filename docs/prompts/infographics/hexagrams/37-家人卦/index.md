# 家人卦（䷤）· 六爻提示词

> 卦序：第 37 卦 | 卦象：巽上离下 | 风火家人 | 卦德：家

家人是易经最「温暖」的一卦——风从火中出，是家的气息向外弥漫。「正家而天下定矣」——正家不是管理技术，而是从自身出发、以德感化的完整修身图鉴。六爻：**闲有家→在中馈→家人嗃嗃→富家→王假有家→有孚威如**。

## 家人卦「正家而天下定」的深层结构

> 闲有家防患（初九）→在中馈柔守（六二）→嗃嗃严正（九三）→富家丰盈（六四）→王假感化（九五）→有孚威如反身（上九◯）
> 九三最深：宁可嗃嗃过严，不可嘻嘻失节。上九最深：威如之吉，反身之谓也。

## 明夷与家人的深层对照

| | 明夷卦 | 家人卦 |
|--|--------|--------|
| **核心** | 晦其明，内难正志 | 正家而天下定 |
| **最深一句** | 明不可息 | 威如之吉，反身之谓也 |

明夷是晦中守贞，家人是正家修德。风从火出，是家的温度向外弥漫——从自己出发，以德感化，以孚威终。
