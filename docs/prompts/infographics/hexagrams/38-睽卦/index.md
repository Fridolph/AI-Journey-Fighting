# 睽卦（䷥）· 六爻提示词

> 卦序：第 38 卦 &nbsp;|&nbsp; 卦象：离上兑下 &nbsp;|&nbsp; 火泽睽 &nbsp;|&nbsp; 卦德：异

睽——乖也，异也。火炎上，泽润下，二气相背，是为睽。然「睽之时用大矣哉」——异中有同，分中有合，是睽卦最深的智慧。

## 卦辞总图 · 睽·小事吉

```text
易经爻辞知识图鉴
睽卦 · 卦辞总图 · 小事吉

【画面主体意象】
上火下泽，睽；君子以同而异——

画面分为上下两部分：
上半部：离火，火焰向上燃烧，橙红烈焰炎炎向上，是离卦之象；
下半部：兑泽，水泽向下流动，深蓝澄澈润泽向下，是兑卦之象。
火与泽之间，一道无形的分界，二者背道而驰，却共存于同一画面——这就是睽。

画面中央：两个女子的背影，并肩而立，却各望一方——
一望火一望泽，「二女同居，其志不同行」。

画面右侧竖排大字：「睽，小事吉」
副文：「睽之时用大矣哉」字色用烟金篆书。

画面左下角卦象图示：离上兑下，六爻排列。

【彖传节录·画面底部横排】
「天地睽而其事同也；男女睽而其志通也；万物睽而其事类也」

【构图】横版 16:9，上火下泽中分画面，两女背影居中，火泽各占半壁。
【色调】上半火域：烈橙、赤红、烟金；下半泽域：深蓝、青碧、冷银。两女背影：一着暖色一着冷色。背景用深墨黑。
【风格】宋代水墨意境，现代平面构成，意象化，是符号是哲学。
```

## 初九 · 悔亡，丧马勿逐，自复；见恶人无咎

```text
易经爻辞知识图鉴
睽卦 · 初九 · 悔亡，丧马勿逐，自复；见恶人无咎

【画面主体意象】
见恶人，以辟咎也——初九是睽卦最「辟咎」的一爻，也是睽卦之始。

「悔亡」——悔恨消失，是睽之初处置得当则悔自亡。
「丧马勿逐自复」——马走失不必追逐，它自会回来。丧是失去，勿逐是不必追，自复是自然归来。
这是睽之初最深的智慧：不强求不执着，顺其自然。
「见恶人无咎」——主动去见那些与自己不合的人，无咎。「以辟咎也」——是为了避开更大的祸患。
在睽异之时，主动沟通异己是化解睽的第一步。

画面主体：一匹马在旷野中自由奔跑，背向画面渐行渐远，却在远处自然转身回望。
马的主人立于画面左侧没有追逐，只是静静看着那匹马神情平静。
画面右侧一个「恶人」剪影面目模糊，但主人正转身向其走去姿态从容。
画面右侧竖排文字：「丧马勿逐自复」「见恶人以辟咎也」字色用深赭。

【几何爻位结构】睽卦六爻从下到上（离上兑下）：初九 ━━  高亮深赭辟咎 ← 当前爻 九二 ━━ 六三 ━ ━ 九四 ━━ 六五 ━ ━ 上九 ━━
标注：「上火下泽，睽；君子以同而异」

【信息模块】① 睽卦 · 初九（右上角）② 卦象标注：火泽睽 · 离上兑下
③ 爻辞：「悔亡，丧马勿逐自复；见恶人无咎。」④ 小象传：「见恶人以辟咎也。」
⑤ 核心义理：睽之初——不强求失去的，主动面对异己，是化睽的第一步。

【配色】深赭·辟咎为主，旷野用深赭暖土，马影用深墨，人影用纯白，背景用深蓝墨色。16:9横幅。
```

## 九二 · 遇主于巷，无咎

```text
易经爻辞知识图鉴
睽卦 · 九二 · 遇主于巷，无咎

【画面主体意象】
遇主于巷，未失道也——九二是睽卦最「曲道遇主」的一爻。

「遇主于巷」——在小巷中遇见主君。巷是小路是曲径，不是正道，却是睽异之时唯一可行之路。
遇主是遇见那个能与自己相应的人。在睽异之时，正道不通曲径偶遇反而无咎。
「未失道也」——虽走小巷虽非正途，但心中之道从未偏离。形式可以曲折，道不能丢失。

画面主体：一条幽深的小巷，青石板路两侧高墙，墙上苔痕斑斑，巷道深处有光透入。
巷道中央两个人影迎面相遇——一人来自巷的这端一人来自巷的那端，在最窄处相遇相认相视而立。
两人之间那一点相遇的空间是画面的光源——从两人之间透出一线暖金光。
画面右侧竖排文字：「遇主于巷无咎」「未失道也」字色用暗金。

【几何爻位结构】初九 ━━（淡）九二 ━━  高亮暗金遇主 ← 当前爻 六三 ━ ━（淡）九四 ━━（淡）六五 ━ ━（淡）上九 ━━（淡）

【信息模块】① 睽卦 · 九二（右上角）② 爻辞：「遇主于巷无咎。」③ 小象传：「遇主于巷未失道也。」
④ 核心义理：睽中曲道——正道不通时曲径相遇，心中之道未失是最深智慧。

【配色】暗金·遇主为主，青石巷道用深灰青，高墙用深赭，两人影用深墨，相遇处光源用暖金。16:9横幅。
```

## 六三 · 见舆曳，其牛掣，其人天且劓，无初有终

```text
易经爻辞知识图鉴
睽卦 · 六三 · 见舆曳，其牛掣，其人天且劓，无初有终

【画面主体意象】
见舆曳位不当也，无初有终遇刚也——六三是睽卦最「艰难」的一爻。

「见舆曳其牛掣」——车被拖拽牛被拉扯。车向前却被从后拖拽，牛向前却被从旁拉扯，一切都在对抗。
「其人天且劓」——天是黥刑在额头刺字，劓是割鼻之刑。受尽屈辱遍体鳞伤，是睽之极难。
「无初有终」——没有好的开始却有好的结局。「遇刚也」——遇到了刚强之人终得相助转危为安。

画面主体：一辆沉重的古车车轮深陷泥泞，车身被绳索从后拖拽。拉车的牛被另一根绳索从侧面拉扯，牛颈弓张四蹄挣扎。
车旁一个人影身形狼狈额有刺字面有伤痕，却仍向前挣扎没有放弃。画面远处一个刚强的人影正向车的方向走来，是援手是转机。
画面右侧竖排文字：「无初有终遇刚也」「位不当终遇援」字色用深墨。

【几何爻位结构】初九 ━━（淡）九二 ━━（淡）六三 ━ ━  高亮深墨艰难 ← 当前爻 九四 ━━（淡）六五 ━ ━（淡）上九 ━━（淡）

【信息模块】① 睽卦 · 六三（右上角）② 爻辞：「见舆曳其牛掣其人天且劓无初有终。」
③ 小象传：「位不当也。遇刚也。」④ 核心义理：睽之极难——位不当处境艰遍受阻碍，但遇刚则终吉。

【配色】深墨·艰难为主，泥泞用深赭暗褐，古车用深墨，牛影用暗褐，绳索用暗红，远处援手用暖金。16:9横幅。
```

## 九四 · 睽孤，遇元夫，交孚，厉无咎

```text
易经爻辞知识图鉴
睽卦 · 九四 · 睽孤，遇元夫，交孚，厉无咎

【画面主体意象】
交孚无咎，志行也——九四是睽卦最「交孚」的一爻。

「睽孤」——睽异而孤独，是睽之极致。九四阳爻居阴位上下皆不相应，是睽卦中最孤独的处境。
「遇元夫」——遇见了大丈夫。元夫是初九是刚健之人。在睽孤之中遇见了真正的同道是睽中最大的转机。
「交孚」——相互信任以诚相交。在睽异之时唯有诚信才能跨越分歧。
「厉无咎」——虽有危厉但无过失。「志行也」——志向得以实现是交孚的结果。

画面主体：旷野之中两个人影相向而立，中间有一段距离——那段距离是睽是分歧是孤独。
但两人都伸出了手手心向上向对方延伸——那伸出的手是交孚是诚信是跨越睽的桥梁。
两人之间那段空间有一道光从上方垂直射下，照亮了两双伸出的手。
旷野苍茫四周皆是睽异的黑暗，唯有两人之间那一道光是整幅画面唯一的暖色。
画面右侧竖排文字：「睽孤遇元夫交孚」「志行也」字色用深蓝。

【几何爻位结构】初九 ━━（淡）九二 ━━（淡）六三 ━ ━（淡）九四 ━━  高亮深蓝交孚 ← 当前爻 六五 ━ ━（淡）上九 ━━（淡）

【信息模块】① 睽卦 · 九四（右上角）② 爻辞：「睽孤遇元夫交孚厉无咎。」③ 小象传：「交孚无咎志行也。」
④ 核心义理：睽中交孚——孤立之中遇同道以诚相交志向得行，是睽中最深的化解之道。

【配色】深蓝·交孚为主，旷野用深蓝墨色，两人影用深墨，伸出的手用暖肤金色，中间光柱用纯金。16:9横幅。
```

## 六五 · 悔亡，厥宗噬肤，往何咎

```text
易经爻辞知识图鉴
睽卦 · 六五 · 悔亡，厥宗噬肤，往何咎

【画面主体意象】
厥宗噬肤，往有庆也——六五是睽卦最「往庆」的一爻。

「悔亡」——悔恨消失，是六五处置得当的结果。
「厥宗噬肤」——其宗族咬破皮肤。厥宗是与自己最亲近的人，噬肤是咬破皮肤。
皮肤是最薄的阻隔，宗族之间虽有睽异但阻隔最薄最易相通。噬肤是突破那最薄的隔阂重新相连。
「往何咎」——前往有什么过失呢？「往有庆也」——是有喜庆的。六五柔居尊位与九二相应，
宗族之间突破薄薄的隔阂重新相聚，是睽中最温暖的一爻。

画面主体：一个家族的庭院，院门半开。门内有人正在等待，门外有人正在归来——归来者步履轻快面带喜色。
门内等待者探身向外双手微张是迎接的姿态。两人之间那扇半开的门是「噬肤」——最薄的隔阂正在被推开。
庭院内灯火通明暖意融融，门外是睽异的旷野已在身后。
画面右侧竖排文字：「厥宗噬肤往有庆也」「悔亡往何咎」字色用暖金。

【几何爻位结构】初九 ━━（淡）九二 ━━（淡）六三 ━ ━（淡）九四 ━━（淡）六五 ━ ━  高亮暖金往庆 ← 当前爻 上九 ━━（淡）

【信息模块】① 睽卦 · 六五（右上角）② 爻辞：「悔亡，厥宗噬肤，往何咎。」③ 小象传：「厥宗噬肤，往有庆也。」
④ 核心义理：睽中归庆——最亲近的人之间隔阂最薄，突破即相聚往有庆，是睽中最温暖的转化。

【配色】暖金·往庆为主，庭院用暖赭暖木色，灯火用暖黄烟金，门外旷野用深蓝冷色。16:9横幅。
```

## 上九 · 睽孤，见豕负涂，遇雨则吉，群疑亡

```text
易经爻辞知识图鉴
睽卦 · 上九 · 遇雨则吉，群疑亡也

【画面主体意象】
遇雨之吉，群疑亡也——上九是睽卦最「群疑消散」的一爻，也是睽卦的终局。

「睽孤」——睽之极孤之极，上九居睽卦之终，疑心最重孤立最深。
「见豕负涂载鬼一车」——看见猪身上满是泥，看见一车的鬼。这是疑心极重时眼中所见的幻象：本是普通之物却因睽疑而看成了怪异。
「先张之弧后说之弧」——先拉开弓后又放下弓。先以为是敌张弓欲射，后来看清放下弓。
「匪寇婚媾」——不是强盗是来求婚的。那些看似可怕的其实是来亲近的。睽疑散去真相大白。
「往遇雨则吉」——前往遇到雨则吉。雨是阴阳相合的象征，是睽的消解是分歧的化解。「群疑亡也」——所有的疑虑都消散了。

画面主体分为左右两部分以一场雨为界：
左半部（雨前·睽疑）：一个人张弓面对远处的模糊人影——那人影在他眼中是鬼怪是威胁是一车的恐惧。画面用暗色扭曲是疑心的视觉化。
右半部（雨后·疑散）：同一个人弓已放下面前是一对来求婚的人，手持礼物笑容可辨。大雨倾盆洗去了一切疑云，画面用暖色清明是真相的视觉化。
画面正中一道大雨垂直而下，是左右两个世界的分界——雨是睽的终结是合的开始。
画面上方竖排大字：「遇雨之吉群疑亡也」字色用纯金。下方横排小字：「睽之时用大矣哉」

【几何爻位结构】初九 ━━（淡）九二 ━━（淡）六三 ━ ━（淡）九四 ━━（淡）六五 ━ ━（淡）上九 ━━  高亮纯金雨合 ← 当前爻

【信息模块】① 睽卦 · 上九（右上角）② 爻辞：「睽孤见豕负涂载鬼一车，先张之弧后说之弧，匪寇婚媾，往遇雨则吉。」
③ 小象传：「遇雨之吉群疑亡也。」
④ 核心义理：睽之终局——疑心极重时所見皆鬼；一场雨洗去所有疑云，异中终有同睽中终有合。
⑤ 角落批注：「睽之时用大矣哉」

【配色】纯金·雨合为主，左半疑域用深墨冷蓝扭曲人影用暗色，右半真域用暖金暖赭婚媾人影用暖色，中间大雨用冷银蓝白，背景渐变从深墨到暖金。左暗右明以雨为界。16:9横幅。
```
