# 睽卦（䷥）· 六爻提示词

> 卦序：第 38 卦 | 卦象：离上兑下 | 火泽睽 | 卦德：异

睽卦是易经最「奇异」的一卦——火向上燃，泽向下流，两女同居，志不同行。「天地睽而其事同，男女睽而其志通，万物睽而其事类」——差异不是破坏，是天地运行的本质。六爻：**丧马勿逐→遇主于巷→舆曳牛掣→遇元夫交孚→厥宗噬肤→遇雨则吉**。

## 睽卦「睽之时用」的深层结构

> 顺势自复（初九）→曲径相遇（九二）→艰难有终（六三）→孤立交孚（九四）→亲密相应（六五）→疑极悟通群疑亡（上九◯）
> 上九最奇：载鬼一车，先张后说，匪寇婚媾——是整部易经中最戏剧性的爻辞。

## 家人与睽的深层对照

| | 家人卦 | 睽卦 |
|--|--------|------|
| **核心** | 正家而天下定 | 天地睽而其事同 |
| **最深一句** | 威如之吉，反身之谓也 | 睽之时用大矣哉 |

家人是聚合温暖，睽是在背离中相遇。上火下泽，背离而行——在差异中寻找相遇，是睽卦最深的智慧。
