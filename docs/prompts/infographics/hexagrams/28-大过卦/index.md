# 大过卦（䷛）· 六爻提示词

> 卦序：第 28 卦 &nbsp;|&nbsp; 卦象：兑上巽下 &nbsp;|&nbsp; 泽风大过 &nbsp;|&nbsp; 卦德：过

大过——大者过也。栋桡，本末弱，中刚强。泽灭木，大过之象。「君子以独立不惧，遯世无闷」——这是易经中最「超越时代」的一卦。

## 大过卦「独立不惧遯世无闷」的深层结构

> 藉白茅柔下（初六）→枯杨生稊生机（九二）→栋桡独刚不辅（九三）→栋隆不桡吉（九四）→枯杨生华虚华（九五）→过涉灭顶壮烈（上六◯）
> 九二枯杨生稊与九五枯杨生华——生机与虚华，一字之差天壤之别；上六过涉灭顶是易经中最悲壮的一爻。

## 颐与大过的深层对照

| | 颐卦 | 大过卦 |
|--|------|------|
| **核心** | 养正则吉，天地养万物圣人养贤 | 独立不惧遯世无闷，大过之时义大矣哉 |
| **最深一句** | 由颐厉吉，大有庆也 | 过涉灭顶，凶，无咎 |

颐是养正自养的智慧，大过是超越常规的独立。颐养而后能大过——养正而后能独立不惧，二卦是养生与超越的两极。

## 初六 · 藉用白茅，无咎

```text
易经爻辞知识图鉴
大过卦 · 初六 · 藉用白茅，无咎

【画面主体意象】藉用白茅，柔在下也——初六是「慎重」的一爻，「藉用白茅」——用白茅垫衬，「无咎」；「柔在下也」。
大过之时柔在下慎重处事无咎。慎重是大过最初的处世智慧。

【几何爻位结构】大过卦六爻从下到上（兑上巽下）：初六 ━ ━  高亮纯白慎重 ← 当前爻 九二 ━━ 九三 ━━ 九四 ━━ 九五 ━━ 上六 ━ ━
标注：「泽灭木，大过；君子以独立不惧，遯世无闷」

【嵌入式大字】「初六」嵌入藉用白茅几何结构，字色用纯白，被柔下慎重的洁白部分遮挡——
「藉用白茅，无咎」，字，白茅，柔下，有大过卦最初的慎重柔下无咎。

【信息模块】① 大过卦 · 初六（右上角）② 爻辞：「藉用白茅，无咎。」③ 小象传：「藉用白茅，柔在下也。」
④ 彖传节录：「大过之时义大矣哉！」⑤ 核心义理：大过起点——柔在下慎重垫衬无咎。
⑨ 收尾精神之语：「用白茅垫衬——无咎；柔在下也——大过之时慎重是处世智慧。」

【配色】纯白·慎重为主，藉用白茅用纯白，柔在下用暗灰，无咎用暖金，背景用深蓝泽风。
16:9横幅，宋代慎重柔下意境，现代平面设计结构。
```

【嵌入式大字】「九二」嵌入枯杨生稊几何结构，字色用深绿，被生机无不利的绿意部分遮挡——
「枯杨生稊，老夫得其女妻，无不利」，字，生稊，相与，有大过卦最深的枯杨生机。

【信息模块】① 大过卦 · 九二（右上角）② 爻辞：「枯杨生稊，老夫得其女妻，无不利。」③ 小象传：「老夫女妻，过以相与也。」
④ 核心义理：大过生机之喜——枯杨生新芽是大过最深生机。与九五枯杨生华对比：生机与虚华一字天壤之别。
⑥ 收尾精神之语：「枯杨生新芽老夫得女妻——无不利；过以相与也——超过常规的结合最深生机。」

【配色】深绿·生机为主，枯杨生稊用深绿，相与用暗绿，无不利用纯金，背景用深蓝泽风。
