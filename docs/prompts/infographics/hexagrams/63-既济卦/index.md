# 既济卦（䷾）· 六爻提示词

> 卦序：第 63 卦 &nbsp;|&nbsp; 卦象：坎上离下 &nbsp;|&nbsp; 水火既济 &nbsp;|&nbsp; 卦德：成

既济卦，是易经最「既成」的一卦——水在火上，水火相交，是既济最深的卦象。「既济，亨，小者亨也」——既济，是小者亨。「初吉终乱，其道穷也」——既济最深的警示：初吉，但终乱，道穷。「刚柔正而位当」——刚柔各正其位，是既济最深的根基。「君子以思患而预防之」——既济，是思患预防最深的精神。六爻是既济的六种面貌：**曳轮濡尾→妇丧茀七日得→高宗伐鬼方→繻有衣袽→东邻西邻→濡其首厉**，是一部从义无咎到何可久的完整既济图鉴。

## 既济卦「初吉终乱思患而预防之」的深层结构

> 曳轮濡尾义无咎（初九）→中道七日得（六二）→三年惫小人勿用（九三）→有疑终日戒（六四）→西邻得时吉大来（九五）→濡首厉何可久（上六◯）
> 初九濡尾慎始，上六濡首厉终——首尾「濡」字呼应，是既济卦最深的「初吉终乱」象征。

## 小过与既济的深层对照

| | 小过卦 | 既济卦 |
|--|------|------|
| **核心** | 宜下不宜上，上逆而下顺 | 初吉终乱，思患而预防之 |
| **最深一句** | 飞鸟遗之音，不宜上宜下 | 东邻杀牛，不如西邻之时也 |

小过是宜下，既济是思患。小过知分寸而后能既济——过恭过哀的节制与思患预防的警戒，是两卦共同的慎重精神。

## 初九 · 曳其轮，濡其尾，无咎

```text
易经爻辞知识图鉴
既济卦 · 初九 · 曳其轮，濡其尾，无咎

【画面主体意象】义无咎，慎始——画面以「水在火上」为整体背景，水火相交是既济卦最根本的气息。

初九，「曳其轮，濡其尾，无咎」——曳其轮是拖曳车轮慎重前进；濡其尾是沾湿尾巴涉水之初；「义无咎也」——义理上无咎。

画面主体：曳轮慎进濡尾无咎——「曳其轮」慎重是既济之初最深的智慧；「濡其尾」涉水尾先湿；
「义无咎也」——义理正确则无咎。慎始是既济最初的智慧。

【几何爻位结构】既济卦六爻从下到上（坎上离下）：初九 ━━  高亮深赭慎始 ← 当前爻 六二 ━ ━ 九三 ━━ 六四 ━ ━ 九五 ━━ 上六 ━ ━
标注：「坎上离下，水在火上，既济」

【嵌入式大字】「初九」嵌入曳轮濡尾几何结构，字色用深赭，被水火相交的既济部分遮挡——
「曳其轮，濡其尾，无咎」，字，曳轮，义无咎，有既济卦最初的慎始无咎。

【信息模块】① 既济卦 · 初九（右上角）② 爻辞：「曳其轮，濡其尾，无咎。」③ 小象传：「曳其轮，义无咎也。」
④ 彖传节录：「初吉，柔得中也。终止则乱，其道穷也。」
⑤ 核心义理：既济的起点是「义无咎」——慎重前进义理正确；「义无咎也」——慎重义正则无咎。
⑧ 收尾精神之语：「拖曳车轮沾湿尾巴——无咎；曳其轮义无咎也——慎重义理正确则无咎。」

【配色】深赭·慎始为主，曳轮慎重用深赭，濡尾初涉用暗金，义无咎用暖金，背景用深蓝红。
16:9横幅，宋代水火既济慎始意境，现代平面设计结构。
```

【几何爻位结构】六爻：初九 ━━（淡）六二 ━ ━（淡）九三 ━━（淡）六四 ━ ━（淡）九五 ━━  高亮纯金时祭 ← 当前爻 上六 ━ ━（淡）

【信息模块】① 既济卦 · 九五（右上角）② 爻辞：「东邻杀牛，不如西邻之禴祭，实受其福。」

## 既济卦六爻 · 既济图鉴总结

```
初九 · 曳轮濡尾    ——  义无咎，慎始
六二 · 妇丧其茀    ——  勿逐，中道，七日得
九三 · 高宗伐鬼方  ——  三年克之，惫，小人勿用
六四 · 繻有衣袽    ——  终日戒，有所疑
九五 · 东邻西邻    ——  不如西邻之时，实受其福，吉大来
上六 · 濡其首      ——  厉，何可久
```

> **「初吉终乱，其道穷也；思患而预防之」**
> ——既济卦的终极义理，不是既成就安，而是「思患预防」——
> 初九濡尾慎始，上六濡首厉终——首尾「濡」字呼应是「初吉终乱」最深的象征；
> 西邻得时实受其福——不在盛大在得时；思患预防是既济卦最圆满的精神。
