# 噬嗑卦（䷔）· 六爻提示词

> 卦序：第 21 卦 &nbsp;|&nbsp; 卦象：离上震下 &nbsp;|&nbsp; 火雷噬嗑 &nbsp;|&nbsp; 卦德：噬

## 噬嗑卦「明罚敕法」的深层结构

> 履校灭趾不行（初九）→噬肤灭鼻乘刚（六二）→噬腊遇毒位不当（六三）→得金矢艰贞未光（九四）→得黄金贞厉得当（六五）→何校灭耳聪不明凶（上九◯）
> 从初九履校灭趾到上九何校灭耳——一脚一颈一小惩一极刑，是最完整的刑法因果链。

## 观与噬嗑的深层对照

| | 观卦 | 噬嗑卦 |
|--|------|------|
| **核心** | 神道设教，中正以观天下 | 明罚敕法，雷电合而章 |
| **最深一句** | 圣人以神道设教，而天下服矣 | 履校灭趾，不行也 |

观是神道设教，噬嗑是明罚敕法。观民设教与明罚敕法——教化与刑罚，是治理的两个侧面。

## 初九 · 履校灭趾，无咎

```text
易经爻辞知识图鉴
噬嗑卦 · 初九 · 履校灭趾，无咎

【画面主体意象】履校灭趾，不行也——初九是噬嗑卦最「初惩」的一爻，「履校灭趾」——脚戴刑具没过脚趾，
「无咎」；「履校灭趾，不行也」——不能行走则不能再犯。小惩大诫是噬嗑最初的执法智慧。

【几何爻位结构】噬嗑卦六爻从下到上（离上震下）：初九 ━━  高亮深赭初惩 ← 当前爻 六二 ━ ━ 六三 ━ ━ 九四 ━━ 六五 ━ ━ 上九 ━━
标注：「颐中有物，曰噬嗑；雷电合而章」

【嵌入式大字】「初九」嵌入履校灭趾几何结构，字色用深赭，被初惩不行的厚重部分遮挡——
「履校灭趾，无咎」，字，履校，不行，有噬嗑卦最初的小惩大诫。

【信息模块】① 噬嗑卦 · 初九（右上角）② 爻辞：「履校灭趾，无咎。」③ 小象传：「履校灭趾，不行也。」
④ 彖传节录：「颐中有物曰噬嗑，噬嗑而亨。刚柔分动而明，雷电合而章。」
⑤ 核心义理：噬嗑执法的起点——初犯最轻惩罚；小惩大诫不行无咎。⑨ 收尾精神之语：「脚戴刑具没过脚趾——无咎；不行也——小惩大诫不能再犯。」

【配色】深赭·初惩为主，履校灭趾用深赭，不行用暗灰，无咎用暖金，背景用深蓝火雷。
```
