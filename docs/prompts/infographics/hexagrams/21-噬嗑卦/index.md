# 噬嗑卦（䷔）· 六爻提示词

> 卦序：第 21 卦 | 卦象：离上震下 | 火雷噬嗑 | 卦德：决

噬嗑是易经最「刚决」的一卦——雷电交加，颐中有物，必须咬断才能亨通。「雷电合而章」——震是威，离是明，威而明是执法的最高境界。六爻：**履校→噬肤→噬腊→噬乾胏→噬乾肉→何校**。

## 噬嗑「明罚敕法」的深层结构

> 初九（脚械·不行）→六二（肤·深入）→六三（腊·遇毒）→九四（骨·金矢）→六五（肉·黄金）→上九（颈·极凶⚠️）

## 观与噬嗑的深层对照

| | 观卦 | 噬嗑卦 |
|--|------|--------|
| **君子之道** | 观民设教 | 明罚敕法 |
| **核心** | 中正以观天下 | 雷电合而章 |
| **最深一句** | 圣人以神道设教 | 先王以明罚敕法 |

先观后罚，先教后法。观而不化则噬嗑，噬嗑而明则天下治。

> 「明罚敕法」——明是雷电的光，罚是雷电的威。明而后罚，罚而有明。不是为了罚，是为了明——让天下在初九时就听到雷声、看到电光、停下来。
