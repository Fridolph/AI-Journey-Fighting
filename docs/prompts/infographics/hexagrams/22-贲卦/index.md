# 贲卦（䷕）· 六爻提示词

> 卦序：第 22 卦 &nbsp;|&nbsp; 卦象：艮上离下 &nbsp;|&nbsp; 山火贲 &nbsp;|&nbsp; 卦德：文

贲——文饰之道。山下有火，火光照山，文采焕发。「观乎天文，以察时变；观乎人文，以化成天下」——这是易经中最「美学」的一卦。

## 贲卦「观乎人文以化成天下」的深层结构

> 贲趾舍车义弗乘（初九）→贲须与上兴（六二）→濡如永贞终莫陵（九三）→白马翰如婚媾（六四）→丘园束帛有喜（六五）→白贲归真上得志（上九◯）
> 从文采自守到素白归真——文采以义为本，以素白为终；白贲是文采最高境界。

## 噬嗑与贲的深层对照

| | 噬嗑卦 | 贲卦 |
|--|------|------|
| **核心** | 明罚敕法，雷电合而章 | 文明以止，观乎人文以化成天下 |
| **最深一句** | 履校灭趾，不行也 | 白贲无咎，上得志也 |

噬嗑是刑法，贲是文采。明罚与文饰——法则与美学，是秩序的阴阳两面。

## 初九 · 贲其趾，舍车而徒

```text
易经爻辞知识图鉴
贲卦 · 初九 · 贲其趾，舍车而徒

【画面主体意象】舍车而徒，义弗乘也——初九是贲卦最「自守」的一爻，「贲其趾」——文饰其足趾，
「舍车而徒」——舍弃车乘徒步而行；「义弗乘也」——道义不允许乘车。文饰不失道义，贲是有义的文采。

【几何爻位结构】贲卦六爻从下到上（艮上离下）：初九 ━━  高亮深赭自守 ← 当前爻 六二 ━ ━ 九三 ━━ 六四 ━ ━ 六五 ━ ━ 上九 ━━
标注：「山下有火，贲；君子以明庶政，无敢折狱」

【嵌入式大字】「初九」嵌入贲趾舍车几何结构，字色用深赭，被义弗乘的自守厚重部分遮挡——
「贲其趾，舍车而徒」，字，舍车，义守，有贲卦最初的文采自守。

【信息模块】① 贲卦 · 初九（右上角）② 爻辞：「贲其趾，舍车而徒。」③ 小象传：「舍车而徒，义弗乘也。」
④ 彖传节录：「观乎人文，以化成天下。」⑤ 核心义理：贲道的起点——文采以义为本；义弗乘则舍车而徒。
⑨ 收尾精神之语：「文饰足趾——舍车徒步而行；义弗乘也——文饰不失道义。」

【配色】深赭·自守为主，贲其趾用深赭，舍车而徒用暗灰，义弗乘用暖金，背景用深蓝山火。
16:9横幅，宋代文采自守意境，现代平面设计结构。
```

## 上九 · 白贲，无咎

```text
易经爻辞知识图鉴
贲卦 · 上九 · 白贲，无咎

【画面主体意象】白贲无咎，上得志也——上九是贲卦最「归真」的一爻，「白贲」——素白的文采，文采至极归于素白；
「无咎」；「上得志也」——居上位而得志。

画面主体：白贲归真上得志——「白贲」素白的文采，文采至极归于无色，无色是色之极；
「上得志也」文采归于本真居上位而得志。从初九贲其趾义弗乘到上九白贲归真——贲卦六爻是文采最完整的圆满历程。

与老子「大音希声大象无形」同一精神——文采至极归于素白，是易经美学最高境界。

【几何爻位结构】六爻：初九 ━━（淡）六二 ━ ━（淡）九三 ━━（淡）六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━  高亮纯白归真 ← 当前爻
标注：「白贲无咎，上得志也」

【嵌入式大字】「上九」嵌入白贲归真几何结构，字色用纯白，被归真得志的素白光芒四方普照——
「白贲，无咎」，字，白贲，归真，有贲卦最终的素白归真；文采至极归于素白。

【信息模块】① 贲卦 · 上九（右上角）② 爻辞：「白贲，无咎。」③ 小象传：「白贲无咎，上得志也。」
④ 核心义理：贲道的终极境界——文采至极归于素白；白是色之极无色是文之极；观乎人文以化成天下。
⑨ 收尾精神之语：「白贲——文采至极归于素白；上得志无咎；从贲趾到白贲是文采最完整的圆满历程。」

【配色】纯白·归真为主，白贲用纯白，上得志用明白，无咎用暖金，背景用深蓝山火。
```
