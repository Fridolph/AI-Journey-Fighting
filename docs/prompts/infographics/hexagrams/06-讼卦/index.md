# 讼卦（䷅）· 六爻提示词

> 卦序：第 6 卦 &nbsp;|&nbsp; 卦象：乾上坎下 &nbsp;|&nbsp; 天水讼 &nbsp;|&nbsp; 卦德：争

讼卦的气质和需卦形成鲜明对比——需是等，讼是争。天往上走，水往下流，两者背道而驰，争讼由此而生。六爻是争讼的六种面貌：**息讼→逃讼→守旧德→止讼→断讼→以讼受服**，是一部关于「争」与「止」的人性图鉴。

## 初六 · 不永所事，小有言，终吉

```
易经爻辞知识图鉴
讼卦 · 初六 · 不永所事，小有言，终吉

【画面主体意象】
及早息讼，辩明而止——
画面以两条背道而驰的
水流与天光为背景，
暗示讼卦「天与水违行」
的根本结构。

画面中心：
一场争执的现场，
但重心不在争，
在「止」——
两人对峙的痕迹，
脚印相向，
但其中一方的脚印，
已经开始转向，
准备离开。

「不永所事」的意象：
一根燃了一半的蜡烛，
被主动吹灭——
不是烧完，
是主动停下来，
「讼不可长也」。

地面：
有争执留下的痕迹，
翻倒的器物，
散落的文书，
但已经开始被收拾——
「小有言」，
有过争执，
但没有走远。

画面一侧：
一道光，
从争执的阴云中透出，
「终吉」——
及早止讼，
光，就回来了。

【几何爻位结构】
讼卦六爻从下到上
（乾上坎下）：
初六 ━ ━  高亮淡墨 ← 当前爻
九二 ━━
六三 ━ ━
九四 ━━
九五 ━━
上九 ━━
标注：「乾上坎下，天与水违行」

【嵌入式大字】
「初六」嵌入争执现场
的几何冷灰块，
字色用淡墨，
被吹灭的蜡烛烟雾
部分遮挡，
有一种「止」的克制感——
字，不张扬。

【笔刷动线】
一条从中央向两侧
分开的笔刷，
模拟「违行」之势，
但在中心处
有一个明显的
收束停顿——
「不永」，
就是这个停顿。

【信息模块】
① 讼卦 · 初六（右上角）
② 卦象标注：天水讼 · 乾上坎下
③ 爻辞：「不永所事，
   小有言，终吉。」
④ 小象传：「不永所事，
   讼不可长也。
   虽有小言，其辩明也。」
⑤ 象传节录：「天与水违行，讼；
   君子以作事谋始。」
⑥ 爻位说明：阴爻居初，
   讼之始，
   及早息争，
   辩虽有，明则吉，
   讼不可长。
⑦ 核心义理：争，
   最好的结局，
   是在最开始就停下来。
   不永，是智慧，
   不是软弱。
⑧ 角落批注：
   「作事谋始——
    争之前，
    先想清楚终点。」
⑨ 收尾精神之语：
   「蜡烛还没烧完，
    就吹灭它，
    不是输，
    是比谁都清醒。」

【配色】
淡墨·冷灰为主，
争执痕迹用深赭，
吹灭的烛烟用冷灰白，
透出的光用极淡暖白，
整体克制、冷静，
是讼卦六爻中
最「止」的一张，
争，刚开始，
就停了。

16:9横幅，宋代人物叙事意境，
现代平面设计结构。
```

## 九二 · 不克讼，归而逋，其邑人三百户，无眚

```
易经爻辞知识图鉴
讼卦 · 九二 · 不克讼，归而逋，无眚

【画面主体意象】
知败而退，避祸全身——
九二是讼卦最具生存智慧的一爻，
「不克讼」——打不赢，
「归而逋」——回去，逃走，
不是懦弱，
是「自下讼上，患至掇也」——
以下讼上，
是自取其祸，
退，是最聪明的选择。

画面主体：
一条退路，
从险境通向远方的村落——
「其邑人三百户」，
那里有家，有人，
有「无眚」的安全。

画面前景：
一个背影，
正在快步离开，
不回头——
「归而逋」，
走，就走得干净，
不拖泥带水。

背景：
远处是高大的「上」的意象——
乾卦的天，
高不可攀，
「自下讼上」，
那个方向，
是深渊，
「入于渊也」。

退路上：
有轻松的感觉——
不是失败的沉重，
是「无眚」的轻——
没有灾祸，
是这一爻最大的吉。

村落轮廓：
极淡，极远，
但方向清晰，
「三百户」——
不是孤身，
是有归处。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
九二 ━━   高亮深墨 ← 当前爻
六三 ━ ━  （淡）
九四 ━━   （淡）
九五 ━━   （淡）
上九 ━━   （淡）
标注：「自下讼上，患至掇也」

【嵌入式大字】
「九二」嵌入退路几何层，
字色用深墨，
被背影和退路
部分遮挡，
字的方向，
微微向退路倾斜——
「逋」，
连字，都在走。

【笔刷动线】
一条从险境方向
急速转向退路方向的笔刷，
转折果断，
不犹豫，
在背影处最浓，
向村落方向渐淡——
「归而逋」，
走，就走得彻底。

【信息模块】
① 讼卦 · 九二（右上角）
② 爻辞：「不克讼，归而逋，
   其邑人三百户，无眚。」
③ 小象传：「不克讼，归而逋也。
   自下讼上，患至掇也。」
④ 爻位说明：阳爻居二，
   坎卦之中，
   以下讼上必败，
   知败而退，
   保全邑人，
   无眚之智。
⑤ 核心义理：打不赢的讼，
   最好的结果，
   是在打之前就走。
   「无眚」，
   是讼卦里最实在的吉。
⑥ 角落批注：
   「患至掇——
    掇，是自取，
    是伸手去捡祸患。
    不伸手，就无眚。」
⑦ 收尾精神之语：
   「退，
    不是因为没有理，
    是因为知道，
    这场争，
    赢了也是输。」

【配色】
深墨·逃色为主，
退路用深赭褐渐淡，
险境方向用深冷灰，
村落用极淡暖赭，
背影用最深墨，
整体有一种
「走得干净」的利落感，
不沉重，
是「无眚」的轻。

16:9横幅，宋代行旅意境，
现代平面设计结构。
```

## 六三 · 食旧德，贞厉，终吉，或从王事，无成

```
易经爻辞知识图鉴
讼卦 · 六三 · 食旧德，贞厉，终吉，或从王事，无成

【画面主体意象】
守旧德，不争功——
六三是讼卦最「安分」的一爻，
「食旧德」——
靠着祖先或前人积累的德业生活，
不去争新功，
「贞厉，终吉」——
守正虽有危，
但终究吉祥。

画面主体：
一座祖宅或旧屋，
不华丽，
但根基深厚，
墙上有岁月的痕迹——
「旧德」，
是时间积累的东西，
不是一朝一夕。

屋内：
一张旧案，
上面摆着祖先留下的器物，
或一卷旧典，
「食旧德」——
不是守旧，
是知道自己的根在哪里。

屋外：
有争讼的喧嚣，
极淡，极远，
「或从王事，无成」——
参与了，
但不居功，
不争先，
「从上吉也」。

画面光线：
从旧屋的窗格透入，
是斑驳的光，
有岁月感，
「终吉」——
守着这道光，
就够了。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
九二 ━━   （淡）
六三 ━ ━  高亮赭褐 ← 当前爻
九四 ━━   （淡）
九五 ━━   （淡）
上九 ━━   （淡）
标注：「食旧德，从上吉也」

【嵌入式大字】
「六三」嵌入旧屋几何结构，
字体有碑刻感、岁月感，
被旧屋纹理和斑驳光影
部分遮挡，
字色用旧金赭，
有一种「旧」的温度——
不新，但有根。

【笔刷动线】
一条沉稳、缓慢的笔刷，
沿旧屋轮廓流动，
笔触厚重，
有岁月积淀的质感，
不急，不争，
「食旧德」，
就是这条线
走得最慢、最稳的样子。

【信息模块】
① 讼卦 · 六三（右上角）
② 爻辞：「食旧德，贞厉，终吉，
   或从王事，无成。」
③ 小象传：「食旧德，从上吉也。」
④ 爻位说明：阴爻居三，
   不当位，
   但知守旧德，
   不争新功，
   从上而吉，
   终吉之象。
⑤ 核心义理：不是所有的时候
   都需要争。
   知道自己的根，
   守着旧德，
   有时候，
   比争功更有力量。
⑥ 角落批注：
   「无成——
    不是没有贡献，
    是把功，
    归还给它该去的地方。」
⑦ 收尾精神之语：
   「旧屋不华丽，
    但根基在。
    食旧德，
    是知道自己从哪里来。」

【配色】
赭褐·旧金为主，
旧屋用深赭褐，
器物用暗旧金，
斑驳光影用暖白，
屋外喧嚣用极淡冷灰，
整体沉稳、有岁月感，
是讼卦六爻中
最「旧」的一张，
旧，是根，
是力量。

16:9横幅，宋代旧宅意境，
现代平面设计结构。
```

## 九四 · 不克讼，复即命，渝安贞，吉

```
易经爻辞知识图鉴
讼卦 · 九四 · 不克讼，复即命，渝安贞，吉

【画面主体意象】
回头认命，转念归正——
九四是讼卦最具转折感的一爻，
「不克讼」——打不赢，
「复即命」——回头，
回到本分，
「渝安贞」——
改变心态，
转为安守正道，
吉。

画面主体：
一个转身的瞬间——
画面中心是一个人，
正在转身，
从争讼的方向
转向归正的方向，
这个转身的弧度，
是「渝」的形状——
改变，
发生在这一刻。

转身前的方向：
争讼的痕迹，
文书，印章，
对峙的阴影，
但已经在背后了。

转身后的方向：
一条平静的路，
有光，
不宽阔，
但方向清晰——
「安贞」，
安，是放下争的那一刻；
贞，是重新找到正道。

地面：
转身处，
有一个清晰的脚印转向——
「复即命」，
回到本分，
脚印，是最诚实的证明。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
九二 ━━   （淡）
六三 ━ ━  （淡）
九四 ━━   高亮深灰转暖 ← 当前爻
九五 ━━   （淡）
上九 ━━   （淡）
标注：「复即命，不失也」

【嵌入式大字】
「九四」嵌入转身几何弧，
字的方向，
微微呈现转折感——
左半字在争讼的冷色里，
右半字在归正的暖色里，
「渝」，

【笔刷动线】
一条明显转折的笔刷，
从左侧争讼方向
急转向右侧归正方向，
转折处最浓，
两端渐淡——
「复即命」，

【信息模块】

   不失也。」
④ 爻位说明：阳爻居四，
   近君位，
   知不克而复命，
   渝而安贞，
   不失其正，
   吉。
⑤ 核心义理：「渝」，
   是讼卦最珍贵的字——
   改变，
   不是失败，
   是找回自己。
   复即命，
   是最有勇气的转身。
⑥ 角落批注：
   「不失也——
    转身，
    不是丢失，
    是找回。」
⑦ 收尾精神之语：
   「打不赢的讼，
    最好的结局，
    是在转身的那一刻，
    重新找到自己。」

【配色】
深灰转暖为主，
争讼方向用冷深灰，
归正方向用暖赭，
转折处用深墨，
归正之路用淡暖光，
整体有明显的
冷暖转变——
从冷到暖，
是「渝安贞」的颜色。
```

## 九五 · 讼，元吉

```
易经爻辞知识图鉴
讼卦 · 九五 · 讼，元吉

【画面主体意象】
中正裁断，元吉——
九五是讼卦唯一真正「元吉」的爻，
不是因为争赢了，
是因为「以中正也」——
中正的裁断者，
是讼卦最需要的存在。

画面主体：
一座高台，
或一个裁断的场所，
不是法庭，
是天地之间的
一个中正的位置——
「位乎天位，以正中也」。

高台上：
不画具体的人物，
用一道天光，
从云层正中透下，
垂直，笔直，
照在高台中央——
这是「中正」的光，
不偏不倚。

高台下：
两侧是争讼的双方，
极淡，极小，
在这道光面前，
争，显得很小——
「尚中正也」，
中正，比争，
大得多。

天空：
乾卦的天，
高远，
云层在两侧，
中间是那道光——
「刚来而得中」，
刚健而中正，
是九五的本质。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
九二 ━━   （淡）
六三 ━ ━  （淡）
九四 ━━   （淡）
九五 ━━   高亮天青金 ← 当前爻
上九 ━━   （淡）
标注：「以中正也，尚中正也」

【嵌入式大字】
「九五」嵌入天光几何柱，
字色用冷金，
被那道垂直天光
正中穿过——
中正，
连字，都是正的。

【笔刷动线】
一条完全垂直的笔刷，
从天顶直落高台，
笔触均匀，
不偏不倚，
「中正」，
就是这条线
最准确的形状。

【信息模块】
① 讼卦 · 九五（右上角）
② 爻辞：「讼，元吉。」
③ 小象传：「讼元吉，以中正也。」
④ 彖传节录：「利见大人，尚中正也。」
⑤ 爻位说明：阳爻居五，
   刚健中正，
   天位之尊，
   裁断争讼，
   以中正为本，
   元吉。
⑥ 核心义理：讼卦唯一的元吉，
   不在争的任何一方，
   在中正的裁断者。
   大人，是这个卦
   最需要的人。
⑦ 角落批注：
   「利见大人——
    争，需要有人
    站在中间，
    看清楚。」
⑧ 收尾精神之语：
   「元吉，
    不属于争赢的那个，
    属于站在中间，
    把光照得最正的那个。」

【配色】
天青·冷金为主，
天光用最纯的冷白金，
高台用深青灰，
云层用深墨蓝，
争讼双方用极淡冷灰，
整体高远、冷静、权威，
是讼卦六爻中
最「正」的一张，
正，是这一爻
唯一的颜色。
```

## 上九 · 或锡之鞶带，终朝三褫之

```
易经爻辞知识图鉴
讼卦 · 上九 · 或锡之鞶带，终朝三褫之

【画面主体意象】
以讼受服，旋赐旋夺——
上九是讼卦最具警示性的结尾，
「或锡之鞶带」——
也许得到了赏赐的大带，
「终朝三褫之」——
但一个早晨，
就被剥夺了三次，
「以讼受服，亦不足敬也」。

画面主体：
一条华丽的鞶带（礼服大带），
摆在画面中心，
但它的状态是——
被解开的，
或半落在地上，
或被手拿着，
但那只手，
是在剥夺，
不是在赐予。

鞶带的质地：
华丽，有金线，
但此刻，
华丽，是讽刺——
「以讼受服」，
靠争讼得来的荣耀，
不值得尊敬。

画面背景：
```

## 讼卦六爻的「争与止」光谱

| 爻位 | 对讼的态度 | 结果 |
|------|----------|------|
| **初六** | 及早息讼 | 终吉 |
| **九二** | 知败而退 | 无眚 |
| **六三** | 守旧德不争 | 终吉 |
| **九四** | 转念归正 | 吉 |
| **九五** | 中正裁断 | 元吉 |
| **上九** | 争到底得服 | 三褫，不足敬 |

讼卦最深的哲学：六爻里，吉的，都是**止**——初六止，九二退，六三守，九四转。唯一元吉的九五，不是争的一方，是站在中间的那个。上九争到最后，得了鞶带，一个早晨，三次没了。「作事谋始」——象传这四个字，是讼卦送给所有人的礼物：争之前，先想清楚，这场争，值不值得开始。
