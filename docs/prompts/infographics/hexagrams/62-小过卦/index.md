# 小过卦（䷽）· 六爻提示词

> 卦序：第 62 卦 &nbsp;|&nbsp; 卦象：震上艮下 &nbsp;|&nbsp; 雷山小过 &nbsp;|&nbsp; 卦德：过

小过卦，是易经最「小过」的一卦——山上有雷，是雷在山上震动、飞鸟遗音的卦象。「小过，小者过而亨也」——小过，是小事过而亨，不可大事。「飞鸟遗之音，不宜上宜下，大吉」——飞鸟遗音，宜下不宜上，是小过最深的天道。「上逆而下顺也」——上逆下顺，是小过最深的义理。「君子以行过乎恭，丧过乎哀，用过乎俭」——小过，是过恭过哀过俭最深的精神。六爻是小过的六种面貌：**飞鸟以凶→过祖遇妣→弗过防之→弗过遇之→密云不雨→弗遇过之**，是一部从不可如何到已亢灾眚的完整小过图鉴。

## 小过卦「飞鸟遗之音不宜上宜下」的深层结构

> 飞鸟以凶（初六）→过祖遇妣（六二）→弗防戕凶（九三）→位不当终不长（九四）→密云不雨已上（六五）→已亢灾眚（上六◯）
> 飞鸟意象贯穿首尾——初六飞鸟以凶，上六飞鸟离之；宜下不宜上上逆而下顺；行过乎恭丧过乎哀用过乎俭。

## 中孚与小过的深层对照

| | 中孚卦 | 小过卦 |
|--|------|------|
| **核心** | 信及豚鱼，孚乃化邦 | 宜下不宜上，上逆而下顺 |
| **最深一句** | 鸣鹤在阴，其子和之 | 飞鸟遗之音，不宜上宜下 |

中孚是诚信感召，小过是宜下不宜上。中孚而不可小过——诚信为上而小过宜下，是中孚与小过的天道对照。
