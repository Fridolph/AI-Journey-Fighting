# 明夷卦（䷣）· 六爻提示词

> 卦序：第 36 卦 &nbsp;|&nbsp; 卦象：坤上离下 &nbsp;|&nbsp; 地火明夷 &nbsp;|&nbsp; 卦德：晦

明夷——明入地中。内文明而外柔顺，以蒙大难。「用晦而明」——这是易经中最「晦暗中守正」的一卦。

## 明夷卦「内文明而外柔顺以蒙大难」的深层结构

> 垂翼义行不食（初九）→拯马顺以则吉（六二）→南狩大首志大得（九三）→入腹获心出门庭（六四）→箕子利贞明不息（六五）→登天入地失则（上六◯）
> 六五箕子「明不可息」与上六「失则入地」形成最深刻对比——守正则明不可息，失则则登天入地。用晦而明。

## 晋与明夷的深层对照

| | 晋卦 | 明夷卦 |
|--|------|------|
| **核心** | 明出地上顺而丽乎大明，自昭明德 | 明入地中，用晦而明，内难而能正其志 |
| **最深一句** | 失得勿恤，往有庆也 | 箕子之贞，明不可息也 |

晋是光明上进，明夷是晦暗中守正。明出地上为晋，明入地中为明夷——二卦是光明与晦暗的终极对照。

## 初九 · 明夷于飞，垂其翼，三日不食

```text
易经爻辞知识图鉴
明夷卦 · 初九 · 明夷于飞，垂其翼；君子于行，三日不食

【画面主体意象】君子于行，义不食也——初九是「义行」的一爻，明夷之鸟垂翼而飞，君子义行三日不进食。
「义不食也」——义不进食是明夷最初的精神。垂翼义行是明夷之始。

【几何爻位结构】明夷卦从下到上（坤上离下）：初九 ━━  高亮深赭义行 ← 当前爻 六二 ━ ━ 九三 ━━ 六四 ━ ━ 六五 ━ ━ 上六 ━ ━
标注：「明入地中，明夷；君子以莅众，用晦而明」

【嵌入式大字】「初九」嵌入垂翼义行几何结构，字色用深赭，被垂翼义行不食的沉重部分遮挡——
「明夷于飞，垂其翼；君子于行，三日不食」，字，垂翼，义行，有明夷卦最初垂翼义行。

【信息模块】① 明夷卦 · 初九 (右上角）② 爻辞：「明夷于飞垂其翼；君子于行三日不食。」③ 小象传：「君子于行，义不食也。」
④ 彖传节录：「晦其明也，内难而能正其志，箕子以之。」⑤ 核心义理：明夷起点——义行垂翼是明夷最初精神。
⑨ 收尾精神之语：「明夷之鸟垂翼而飞——义行三日不食；义不食也——义是明夷最初的精神。」

【配色】深赭·义行为主，垂翼用深赭，三日不食用深灰，义不食用暖金，背景用深蓝地火。
16:9横幅，宋代垂翼义行意境，现代平面设计结构。
```

## 六五 · 箕子之明夷，利贞

```text
易经爻辞知识图鉴
明夷卦 · 六五 · 箕子之明夷，利贞

【画面主体意象】箕子之贞，明不可息也——六五是「不息」的一爻，「箕子之明夷利贞」——箕子守正明不可息；
「明不可息也」——光明不可熄灭。箕子是明夷之时最深守正典范。与上六「失则入地」形成最深刻对比。

【几何爻位结构】六爻：初九 ━━（淡）六二 ━ ━（淡）九三 ━━（淡）六四 ━ ━（淡）六五 ━ ━  高亮纯金不息 ← 当前爻 上六 ━ ━（淡）
标注：「箕子之贞，明不可息也」

【嵌入式大字】「六五」嵌入箕子不息几何结构，字色用纯金，被明不可息的光芒四方普照——
「箕子之明夷，利贞」，字，箕子，不息，有明夷卦最深箕子明夷利贞明不可息。

【信息模块】① 明夷卦 · 六五（右上角）② 爻辞：「箕子之明夷，利贞。」③ 小象传：「箕子之贞，明不可息也。」
④ 核心义理：明夷不息之光——箕子守正明不可息。守正则明不息失则则入地——与上六最深刻对比。
⑧ 收尾精神之语：「箕子之贞——明不可息也；内难而能正其志箕子以之。」

【配色】纯金·不息为主，箕子用纯金，利贞用明金，不以息用暖金，背景用深蓝地火透金光。
16:9横幅，宋代箕子明不息意境，现代平面设计结构。
```

## 明夷卦六爻 · 完整晦明图鉴

```
初九 · 垂翼义行    ——  三日不食，义不食
六二 · 拯马顺则    ——  左股，吉
九三 · 南狩大得    ——  大首，志乃大得
六四 · 入腹获心    ——  出门庭，获心意
六五 · 箕子不息    ——  利贞，明不可息
上六 · 登天入地    ——  照四国，失则
```

> **「内文明而外柔顺以蒙大难，文王以之；晦其明也内难而能正其志，箕子以之」**
> ——从初九「垂翼义行」到上六「登天入地失则」，明夷六爻是完整晦明图鉴——
> 箕子之贞明不可息是守正之光，上六失则入地是失道之暗；
> 守正则明不息失则则入地；明入地中君子用晦而明。
