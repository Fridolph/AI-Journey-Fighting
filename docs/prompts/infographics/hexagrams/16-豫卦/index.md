# 豫卦（䷏）· 六爻提示词

> 卦序：第 16 卦 | 卦象：震上坤下 | 雷地豫 | 卦德：乐

豫卦是易经最「动」的一卦——雷出地奋，春雷一声，万物震动，是顺动的喜悦。六爻是豫的六种面貌：**鸣豫→介石→盱豫→由豫→贞疾→冥豫**。

## 豫卦「顺以动」的四层结构

> 天地顺以动 → 日月不过，四时不忒
> 圣人顺以动 → 刑罚清而民服
> 建侯顺以动 → 利建侯行师
> 君子顺以动 → 豫之时义大矣哉！

## 谦与豫的深层对照

| | 谦卦 | 豫卦 |
|--|------|------|
| **卦象** | 地山谦（山藏于地） | 雷地豫（雷出地奋） |
| **君子之道** | 裒多益寡，称物平施 | 作乐崇德，殷荐上帝 |
| **核心** | 卑以自牧 | 顺以动 |
| **最深一句** | 谦尊而光，卑而不可踰 | 豫之时义大矣哉 |

谦是收，豫是发。谦而后能豫，豫而不忘谦，才是「顺以动」最完整的循环。

> 豫卦最深的一句：「介于石，不终日」——在万物欢腾的豫运中，坐在石上，见几而作，不等终日。喜悦，需要清醒；顺动，需要见几。
