# 蹇卦（䷦）· 六爻提示词

> 卦序：第 39 卦 | 卦象：坎上艮下 | 水山蹇 | 卦德：难

蹇卦是易经最「艰难」的一卦——山上有水，险在前，止于后。「见险而能止，知矣哉」——知止是蹇卦最深的智慧。六爻：**往蹇来誉→王臣蹇蹇→往蹇来反→往蹇来连→大蹇朋来→往蹇来硕**。

## 蹇卦「反身修德」的深层结构

> 知止待时（初六）→忠义赴难（六二）→刚健知返（九三）→柔顺连众（六四）→大蹇朋来（九五）→蹇极来硕（上六◯）
> 六爻「来」字精妙：来誉、来反、来连、朋来、来硕——每一爻的「来」，都是蹇难中向内回归的不同面貌。

## 睽与蹇的深层对照

| | 睽卦 | 蹇卦 |
|--|------|------|
| **核心** | 天地睽而其事同 | 见险而能止，知矣哉 |
| **最深一句** | 睽之时用大矣哉 | 蹇之时用大矣哉 |

睽是在背离中相遇，蹇是在险难中知止。蹇难，是反身修德最深的契机。
