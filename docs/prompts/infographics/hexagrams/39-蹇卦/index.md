# 蹇卦（䷦）· 六爻提示词

> 卦序：第 39 卦 &nbsp;|&nbsp; 卦象：坎上艮下 &nbsp;|&nbsp; 水山蹇 &nbsp;|&nbsp; 卦德：难

蹇——难也，险在前也。山上有水，行路艰难，是为蹇。然「蹇之时用大矣哉」——见险而止，反身修德，是蹇卦最深的智慧。

## 卦辞总图 · 见险而能止

```text
易经爻辞知识图鉴
蹇卦 · 卦辞总图 · 见险而能止，知矣哉

【画面主体意象】
山上有水，蹇；君子以反身修德——

画面主体：一座险峻的山居画面中央，山势陡峭岩壁如削。
山腰之上，一道湍急的水流横截山路，水势汹涌，是坎险之象。
山路之上一个旅人正走向那道险水，却在水前停住了脚步——不是退缩，是「见险而能止」。
旅人停步处他转过身面向来路低头沉思——这是「反身修德」的姿态：不强行向前，而是向内向自身寻求答案。
画面西南方向天色渐明云开见日是「利西南」之象；东北方向山路险绝云雾封锁是「不利东北」之象。

【彖传节录】「蹇，难也，险在前也。见险而能止，知矣哉！」「蹇之时用大矣哉」

【构图】横版 16:9，险山居中水险横截，旅人停步转身居前景，西南明东北暗形成强烈方位对比。
【色调】山体用深墨青灰，险水用冷蓝银白，旅人用深赭暖色，西南天色用暖金，东北云雾用冷灰深蓝，背景用深墨。
```

## 初六 · 往蹇，来誉

```text
易经爻辞知识图鉴
蹇卦 · 初六 · 往蹇，来誉

【画面主体意象】往蹇来誉，宜待也——初六是蹇卦最「宜待」的一爻。

「往蹇」——前往则遇蹇难。初六力量最弱此时前往必遇险阻。
「来誉」——退来则得誉。不强行前进退而待时反而赢得赞誉。
「宜待也」——应当等待。蹇之初时机未至宜待不宜往。

画面主体：一条山路前方云雾深锁险阻重重路断于雾中。路上一个旅人停住脚步转身退回原处。
退回之处有人等候向退回的旅人投以赞许的目光是「来誉」。画面左侧一棵枝叶茂盛根深蒂固的大树是「待」的象征。

【几何爻位结构】蹇卦从下到上（坎上艮下）：初六 ━ ━  高亮冷青待时 ← 当前爻 六二 ━ ━ 九三 ━━ 六四 ━ ━ 九五 ━━ 上六 ━ ━
标注：「山上有水，蹇；君子以反身修德」

【嵌入式大字】「初六」嵌入往蹇来誉几何结构，字色用冷青，被宜待知时的沉稳部分遮挡——
「往蹇，来誉」，字，来誉，宜待，有蹇卦最初知时待机智慧。

【信息模块】① 蹇卦 · 初六（右上角）② 爻辞：「往蹇来誉。」③ 小象传：「往蹇来誉宜待也。」
④ 核心义理：蹇之初——知时而待不强于险退守得誉。⑨ 收尾精神之语：「往蹇来誉——宜待也；知时是蹇之初智。」

【配色】冷青·待时为主，山路用深青灰，云雾用冷蓝白，旅人用深赭，等候之人用暖金，大树用深墨绿。16:9横幅。
```

## 六二 · 王臣蹇蹇，匪躬之故

```text
易经爻辞知识图鉴
蹇卦 · 六二 · 王臣蹇蹇，匪躬之故

【画面主体意象】王臣蹇蹇，终无尤也——六二是蹇卦最「忠蹇」的一爻。

「王臣蹇蹇」——王的臣子蹇而又蹇。蹇蹇是重重艰难，不止一次的险阻。王臣是为王效力的臣子，在蹇难之中仍然前行是忠义的体现。
「匪躬之故」——不是为了自身的缘故。匪是非非，躬是自身。不是为了个人利益安危而是为了君王大义甘受蹇难。
「终无尤也」——终究没有过失。虽历重重蹇难但出于公义终无过失。为公义而蹇蹇而无尤。

画面主体：一条崎岖山路险阻重重。一个臣子身着官服手持笏板在重重险阻中艰难前行——不是为自己是为了远处隐约可见的王城。
他的身姿沉重每一步都要用力，但他没有停没有回头。

【几何爻位结构】初六 ━ ━（淡）六二 ━ ━  高亮深赭忠蹇 ← 当前爻 九三 ━━（淡）六四 ━ ━（淡）九五 ━━（淡）上六 ━ ━（淡）

【信息模块】① 蹇卦 · 六二（右上角）② 爻辞：「王臣蹇蹇，匪躬之故。」③ 小象传：「终无尤也。」
④ 核心义理：蹇中忠义——为公义而历重蹇非为一己之私终无过失。
⑥ 收尾精神之语：「王臣蹇蹇——匪躬之故；终无尤也——忠义是蹇中最深精神。」

【配色】深赭·忠蹇为主，山路用深赭暗褐，险阻用深墨，臣子用深赭官服，王城用烟金。16:9横幅。
```

## 九三 · 往蹇来反

```text
易经爻辞知识图鉴
蹇卦 · 九三 · 往蹇来反

【画面主体意象】往蹇来反，内喜之也——九三是蹇卦最「知止内喜」的一爻。

「往蹇」——前往则遇蹇难。九三刚健之质本欲前进，但前方险阻往则必蹇。
「来反」——退回来回到内卦。反是返回是回归。九三知险而返不是失败是智慧。
「内喜之也」——内卦之人为之喜悦。九三是内卦主爻是大家的依靠；他的返回是对内卦的守护。

画面主体：一个刚健旅人已走到险前察看判断然后毅然转身大步返回——那个转身的瞬间是整幅画面最有力量的时刻。
返回的路上内卦的人们迎上来面带喜色是「内喜之也」。

【几何爻位结构】初六 ━ ━（淡）六二 ━ ━（淡）九三 ━━  高亮暗金知止 ← 当前爻 六四 ━ ━（淡）九五 ━━（淡）上六 ━ ━（淡）

【信息模块】① 蹇卦 · 九三（右上角）② 爻辞：「往蹇来反。」③ 小象传：「往蹇来反内喜之也。」
④ 核心义理：蹇中知止——刚健之人见险知返守护内部内喜之。
⑥ 收尾精神之语：「往蹇来反——内喜之也；知止是刚者之智。」

【配色】暗金·知止为主，险路用深青灰，坎险用冷蓝，旅人用暗金深赭，转身用暖金高亮，迎接之人用暖色。16:9横幅。
```

## 六四 · 往蹇来连

```text
易经爻辞知识图鉴
蹇卦 · 六四 · 往蹇来连

【画面主体意象】往蹇来连，当位实也——六四是蹇卦最「连合」的一爻。

「往蹇」——前往则遇蹇难。六四阴爻居上卦前方险阻仍在。
「来连」——退来则得连合。连是连结是联合。六四退而不孤与众爻相连合力共济。
「当位实也」——居位恰当内心充实。六四不孤军奋进而是连结众力以实力应蹇。

画面主体：六四的人影没有独自前进而是转身向后方的众人伸出手是召唤是连结。后方众人一一上前手手相连形成一条人链——
那条人链是「来连」的视觉化。人链的力量汇聚于六四共同面对前方险阻。

【几何爻位结构】初六 ━ ━（淡）六二 ━ ━（淡）九三 ━━（淡）六四 ━ ━  高亮深蓝连合 ← 当前爻 九五 ━━（淡）上六 ━ ━（淡）

【信息模块】① 蹇卦 · 六四（右上角）② 爻辞：「往蹇来连。」③ 小象传：「往蹇来连当位实也。」
④ 核心义理：蹇中连合——独力不足连结众力当位而实是蹇中最深合力之道。
⑥ 收尾精神之语：「往蹇来连——当位实也；连合众力以应大蹇。」

【配色】深蓝·连合为主，险路用深青灰，人链用深蓝渐暖，六四用深蓝高亮，连结的手用纯金。16:9横幅。
```

## 九五 · 大蹇朋来

```text
易经爻辞知识图鉴
蹇卦 · 九五 · 大蹇朋来

【画面主体意象】大蹇朋来，以中节也——九五是蹇卦最「朋来」的一爻，也是蹇卦的核心。

「大蹇」——最大的蹇难。九五居君位所面对的不是个人的蹇是天下的蹇是大蹇。
「朋来」——朋友同道来了。在大蹇之中九五以中正之德感召同道四方朋友纷纷来援。
「以中节也」——因为守中持节。九五之所以能在大蹇中感召朋来是因为他守中持节以德感人。

画面主体：高山之巅险水环绕。九五的身影居中而立身姿中正面对四方险阻不动如山。
四面八方有人影从远处向山巅汇聚——都在向那个中正的身影靠拢。那些汇聚而来的人影是「朋来」——是中正之德感召天下的力量。
九五身影周围有一圈光晕是中正之德的象征。

【几何爻位结构】初六 ━ ━（淡）六二 ━ ━（淡）九三 ━━（淡）六四 ━ ━（淡）九五 ━━  高亮烟金朋来 ← 当前爻 上六 ━ ━（淡）

【信息模块】① 蹇卦 · 九五（右上角）② 爻辞：「大蹇朋来。」③ 小象传：「大蹇朋来以中节也。」
④ 核心义理：蹇中朋来——大蹇之时中正守节感召四方同道朋来共济。
⑧ 收尾精神之语：「大蹇朋来——以中节也；中正之德感召天下。」

【配色】烟金·朋来为主，山巅用深青灰，险水用冷蓝，九五用烟金高亮，光晕用纯金，朋来用暖色。16:9横幅。
```

## 上六 · 往蹇来硕，吉；利见大人

```text
易经爻辞知识图鉴
蹇卦 · 上六 · 往蹇来硕，吉；利见大人

【画面主体意象】往蹇来硕志在内也，利见大人以从贵也——上六是蹇卦最「硕果归来」的一爻。

「往蹇来硕」——前往则蹇退来则硕。硕是丰硕是成果是收获。历经全卦蹇难不强行向前而是向内向自身向修德，最终来而得硕——丰硕的果突是反身修德的结果。
「志在内也」——志向在于内部。上六的志向不在于外部征服而在于内部充实——反身修德是蹇卦象传最深的「君子以反身修德」的最终体现。
「利见大人以从贵也」——历尽蹇难归来丰硕见大人从贵道吉。

画面主体：一个旅人历经重重蹇难终于归来。他的身后是走过的险山渡过的险水历经的重重阻碍——那些阻碍已成为他身后的风景。
他的手中捧着丰硕的果实——那是「来硕」——是内部修炼的成果。画面前方一位大人立于云端向他伸出手是「利见大人」。
旅人仰望大人神情是从容——是内心已充实丰硕之人与大人相遇时应有的从容。

【几何爻位结构】初六 ━ ━（淡）六二 ━ ━（淡）九三 ━━（淡）六四 ━ ━（淡）九五 ━━（淡）上六 ━ ━  高亮纯金硕归 ← 当前爻

【信息模块】① 蹇卦 · 上六（右上角）② 爻辞：「往蹇来硕，吉；利见大人。」③ 小象传：「志在内也。以从贵也。」
④ 核心义理：蹇之终局——历尽蹇难志在内修来而得硕见大人从贵道是反身修德最终圆满。
⑨ 收尾精神之语：「往蹇来硕——志在内也；反身修德硕果归来；蹇之时用大矣哉。」

【配色】纯金·硕归为主，身后险山用深青灰渐远，旅人用暖赭，硕果用纯金橙红，大人用烟金光晕，天色用暖金渐明。16:9横幅。
```
