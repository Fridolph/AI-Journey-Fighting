## 初九 · 潜龙勿用

```
易经爻辞知识图鉴
乾卦 · 初九 · 潜龙，勿用

【画面主体意象】
深渊之底——
画面以深邃的水底空间为主体，
水压感强，光线极微，
从画面最上方透下一缕极细的冷白光，
光到达不了底部，
只是暗示"上面有天"。

水底深处，一条龙盘卧——
不是张牙舞爪的龙，
是收束的、静止的、蓄势的龙，
龙身蜷曲如太极，
鳞片隐约可见，
眼睛微睁，
不是睡着，是在等。

龙的周围：
深水的压力用层层墨晕表现，
越深越黑，越黑越沉，
但龙本身有一种内在的光——
不外露，只是隐隐透出，
像炭火在灰烬下。

【几何爻位结构】
画面右侧竖排六爻线条：
━━  （九二，淡）
━━  （九三，淡）
━━  （九四，淡）
━━  （九五，淡）
━━  （上九，淡）
━━  （初九，高亮金色）← 当前爻

【嵌入式大字】
「初九」两字拆解为超大背景字，
嵌入深水几何块面之中，
被水压与龙身部分遮挡，
字色用极淡冷金，
若隐若现于深墨之中。

【笔刷动线】
一条深墨笔刷从左下角
向右上方流动，
模拟水流压力与潜藏之势，
飞白处透出极淡的冷光，
在龙身处轻轻托起，
暗示蓄势待发。

【信息模块】
① 乾卦 · 初九（右上角清晰标注）
② 爻辞：「潜龙，勿用」（最突出引用块）
③ 小象传：「潜龙勿用，阳在下也」
④ 文言：「龙德而隐者也。
   不易乎世，不成乎名；
   确乎其不可拔，乾龙也。」
⑤ 爻位说明：阳爻，居最下位，
   阳气潜藏，时机未至
⑥ 核心义理：隐而未见，非无能，
   是知时。潜，是最深的蓄积。
⑦ 角落批注：「遁世而无闷，
   不见是而无闷」
⑧ 收尾精神之语：
   「潜，不是没有。
    是还没到时候。」

【配色】
深墨黑为主，冷蓝为辅，
龙身用极深的墨绿金，
唯一的光源是画面顶部
那一缕极细的冷白——
整体压抑、深沉、蓄势，
是六爻中最暗的一张。

现代平面设计结构。
```

## 九二 · 见龙在田，利见大人

```
易经爻辞知识图鉴
乾卦 · 九二 · 见龙在田，利见大人

【画面主体意象】
清晨田野——
画面以广阔的田野为主体，
晨雾未散，光线刚刚铺开，
是一天中最清醒、最干净的时刻。

田野中央，一条龙初现——
不是腾空的龙，
是刚刚从地平线升起、
半身显现于晨雾之中的龙，
龙头抬起，目光平和，
有一种「德施普也」的从容——
不急着飞，不急着证明，
只是在这里，清清楚楚地在这里。

龙身下方：
田垄纹理清晰，
暗示脚踏实地，
暗示「庸言之信，庸行之谨」。

晨光从右侧斜入，
照在龙的侧面，
是「天下文明」的光，
不刺眼，是普照的光。

【几何爻位结构】
画面右侧竖排六爻：
━━  （初九，淡）
━━  （九三，淡）
━━  （九四，淡）
━━  （九五，淡）
━━  （上九，淡）
━━  （九二，高亮暖金）← 当前爻

【嵌入式大字】
「九二」嵌入晨雾几何层之中，
字色用暖赭金，
被晨雾与龙身部分遮挡，
整体可读。

【笔刷动线】
一条从左至右的横向笔刷，
模拟晨光铺展之势，
从左侧浓墨渐变为右侧淡金，
扫过田垄，托起龙身，
在画面右侧自然散开，
如德之普施。

【信息模块】
① 乾卦 · 九二（右上角）
② 爻辞：「见龙在田，利见大人」
③ 小象传：「见龙在田，德施普也」
④ 文言：「庸言之信，庸行之谨，
   闲邪存其诚，善世而不伐，
   德博而化。」
⑤ 爻位说明：阳爻居二，
   得中不得正，
   德已显，时已至，
   天下文明始于此。
⑥ 核心义理：见，不是炫耀，
   是德自然显现。
   大人，是德与位相符之人。
⑦ 角落批注：「与时偕行」
⑧ 收尾精神之语：
   「不是你去找大人，
    是你的德，
    让大人看见了你。」

【配色】
深墨为底，晨光暖赭为主调，
龙身用深墨金，
田垄用温润赭褐，
晨雾用淡灰白，
整体比初九明亮，
有一种清醒、从容、
刚刚开始的质感。
```

## 九三 · 君子终日乾乾，夕惕若，厉无咎

```
易经爻辞知识图鉴
乾卦 · 九三 · 君子终日乾乾，夕惕若，厉无咎

【画面主体意象】
昼与夜的交界——
画面一分为二：
左半部是烈日当空的白昼，
右半部是星月初现的黄昏，
两者在画面正中交汇，
没有明确的分界线，
是渐变的，是「终日」的感觉。

画面中心：
一张书案，无人，
但有人在过的痕迹——
摊开的竹简，
未干的墨迹，
一盏灯，白昼点着，
夜晚也点着，
从未熄灭。

案上竹简隐约可见：
「进德修业」四字，
是这一爻的精神内核。

天空中：
日与月同时出现在画面两侧，
不是奇景，是「终日乾乾」的视觉化——
从日出到日落，
从日落到夜深，
都在这一张图里。

【几何爻位结构】
画面右侧竖排六爻：
━━  （初九，淡）
━━  （九二，淡）
━━  （九四，淡）
━━  （九五，淡）
━━  （上九，淡）
━━  （九三，高亮）← 当前爻
标注：「重刚不中，上不在天，下不在田」

【嵌入式大字】
「九三」嵌入昼夜交界的几何块面，
左字在日光中，右字在暮色里，
同一个名字，两种光线，
暗示「因其时而惕」。

【笔刷动线】
一条从左上（日）到右下（夜）
的斜向笔刷，
模拟时间流逝之势，
浓淡变化明显，
在书案处短暂停顿，
形成画面的重心。

【信息模块】
① 乾卦 · 九三（右上角）
② 爻辞：「君子终日乾乾，
   夕惕若，厉无咎」
③ 小象传：「终日乾乾，反复道也」
④ 文言：「君子进德修业，
   知至至之，知终终之，
   居上位而不骄，在下位而不忧。」
⑤ 爻位说明：重刚不中，
   处境最险，
   上不在天，下不在田，
   唯有乾乾方能无咎。
⑥ 核心义理：危，不是因为弱，
   是因为位置本身就难。
   知道难，还是做，这是九三。
⑦ 角落批注：「虽危无咎矣」
⑧ 收尾精神之语：
   「白天做，晚上警惕，
    不是焦虑，是自知。」

【配色】
左侧烈日金·炭黑，
右侧暮色深蓝·冷灰，
书案用暖赭，灯火用点金，
整体张力最强，
是六爻中情绪最紧绷的一张，
但灯火不灭，暗示无咎。

16:9横幅，宋代文人书斋意境，
现代平面设计结构。
```

## 九四 · 或跃在渊，无咎

```
易经爻辞知识图鉴
乾卦 · 九四 · 或跃在渊，无咎

【画面主体意象】
跃与不跃之间——
这是六爻中张力最特殊的一张：
不是已经跃起，不是还在潜藏，
是那个「或」字——
悬而未决的一刻。

画面构图：
上方是天，云层厚重，
有一道裂缝，透出光，
暗示「天」已在上方等待。

下方是深渊，墨黑深沉，
是初九的那个深渊，
但此刻龙已不在渊底，
而是在渊口——

画面正中：
一条龙，半身出渊，
半身仍在水中，
龙头向上，
眼睛看着那道云裂，
身体保持着
「将跃未跃」的弧度——
这是「或」的形状。

没有人告诉它跃还是不跃，
它在自己决定。

【几何爻位结构】
画面右侧竖排六爻：
━━  （初九，淡）
━━  （九二，淡）
━━  （九三，淡）
━━  （九五，淡）
━━  （上九，淡）
━━  （九四，高亮）← 当前爻
标注：「上下无常，进退无恒，欲及时也」

【嵌入式大字】
「九四」嵌入渊口的水气几何层，
字体一半在水中（深墨），
一半在天光中（冷金），
与龙身的处境完全呼应。

【笔刷动线】
一条垂直向上的笔刷，
从渊底向天空冲去，
在龙身处最浓，
向上渐淡，
在云裂处化为飞白——
是「或跃」的动势，
但没有完成，
在飞白处悬停。

【信息模块】
① 乾卦 · 九四（右上角）
② 爻辞：「或跃在渊，无咎」
③ 小象传：「或跃在渊，进无咎也」
④ 文言：「上下无常，非为邪也。
   进退无恒，非离群也。
   君子进德修业，欲及时也。」
⑤ 爻位说明：阳爻居四，
   近君位而未至，
   自试之爻，
   乾道乃革之时。
⑥ 核心义理：「或」是智慧，
   不是犹豫。
   知道可以跃，也知道
   跃的时机由自己判断。
⑦ 角落批注：「疑之者，慎之也」
⑧ 收尾精神之语：
   「或，不是不确定，
    是把选择权，
    留给自己。」

【配色】
上半天空：深蓝冷灰，云裂冷白金；
下半深渊：深墨黑，极深；
龙身：跨越两色，
上半身用冷金，
下半身用深墨，
是六爻中对比最强烈的一张，
张力感最强。

16:9横幅，宋代山水渊天意境，
现代平面设计结构。
```

## 九五 · 飞龙在天，利见大人

```
易经爻辞知识图鉴
乾卦 · 九五 · 飞龙在天，利见大人

【画面主体意象】
苍穹正中——
这是六爻中最磅礴的一张，
也是最需要克制的一张：
磅礴而不炫耀，
大气而不喧嚣。

画面以高天为主体，
云层在下方，
龙在云层之上——
不是穿云而过，
是居于天，
俯视万物，
「乃位乎天德」。

龙的姿态：
舒展，不是张牙舞爪，
是「与天地合其德」的从容，
龙身形成一条
优雅的S形弧线，
横贯画面，
如「大明终始」。

天光：
从龙身四周自然散出，
不是龙在发光，
是龙所在之处，
天本来就是这样亮的。

远处云下：
极小的山河轮廓，
暗示「万国咸宁」，
暗示「云从龙，风从虎」，
暗示天下在这道光里。

【几何爻位结构】
画面右侧竖排六爻：
━━  （初九，淡）
━━  （九二，淡）
━━  （九三，淡）
━━  （九四，淡）
━━  （上九，淡）
━━  （九五，高亮金）← 当前爻
标注：「刚健中正，大人造也」

【嵌入式大字】
「九五」嵌入高天云层几何块，
字体极大，
被龙身S弧线穿过，
字色用深邃苍穹蓝，
与金色龙身形成
最高级的对比。

【笔刷动线】
一条横贯全图的宽阔笔刷，
从左至右，
模拟天风浩荡之势，
在龙身处最浓最金，
两端渐淡渐白，
如「云行雨施，天下平也」。

【信息模块】
① 乾卦 · 九五（右上角）
② 爻辞：「飞龙在天，利见大人」
③ 小象传：「飞龙在天，大人造也」
④ 文言：「同声相应，同气相求；
   水流湿，火就燥；云从龙，风从虎。
   圣人作，而万物覩。」
⑤ 爻位说明：阳爻居五，
   刚健中正，君位天德，
   六爻之最吉，
   乃位乎天德。
⑥ 核心义理：飞龙在天，
   不是权力的顶点，
   是德与位终于相符的时刻。
   大人见大人，各从其类。
⑦ 角落批注：
   「与天地合其德，
    与日月合其明」
⑧ 收尾精神之语：
   「不是飞得最高，
    是飞在最对的位置。」

【配色】
苍穹深蓝为主，
龙身用最纯的金，
天光用冷白，
云层用银灰，
远处山河用极淡墨，
是六爻中最明亮、
最开阔、最大气的一张，
但金不可过满——
因为上九就在旁边。
```

## 上九 · 亢龙有悔

```
易经爻辞知识图鉴
乾卦 · 上九 · 亢龙有悔

【画面主体意象】
极高处的孤独——
这是六爻中情感最复杂的一张：
不是失败，不是惩罚，
是「盈不可久也」的必然，
是「知进而不知退」的代价。

画面以极高的天空为主体，
比九五更高，
但这个高，是孤寂的高——
云层全在下方，
远得看不见，
山河全在下方，
远得看不见，
只有龙，
独自在这个高度。

龙的姿态：
仍然是龙，仍然有力，
但龙头微低，
不是俯视万物的从容，
是「贵而无位，高而无民」
的那种低——
不是软弱，是知道了。

龙的周围：
没有云，没有风，
只有极深的暗蓝天空，
和极远处一点点
将要熄灭的金光——
那是九五的光，
已经在下面了。

画面下方极远处：
隐约可见九五的飞龙，
极小，极远，
暗示「亢」的含义——
不是比九五更好，
是超过了那个「正」的位置。

【几何爻位结构】
画面右侧竖排六爻：
━━  （初九，淡）
━━  （九二，淡）
━━  （九三，淡）
━━  （九四，淡）
━━  （九五，淡金）
━━  （上九，高亮冷白）← 当前爻
标注：「穷之灾也，盈不可久也」

【嵌入式大字】
「上九」嵌入极高天空的几何暗块，
字色用冷白带灰，
被龙身遮挡一部分，
整体有一种
「在最高处，
反而最不被看见」的感觉。

【笔刷动线】
一条从画面中央
向右上角消散的笔刷，
模拟「亢」的走势——
力道还在，
但方向已经超出了
画面能承载的范围，
在右上角化为飞白，
消散于虚空，
是「动而有悔」的视觉化。

【信息模块】
① 乾卦 · 上九（右上角）
② 爻辞：「亢龙有悔」
③ 小象传：「亢龙有悔，盈不可久也」
④ 文言：「贵而无位，高而无民，
   贤人在下而无辅，
   是以动而有悔也。」
⑤ 爻位说明：阳爻居极上，
   超越君位，
   与时偕极，穷之灾也。
⑥ 核心义理：亢，不是错，
   是超过了那个「正」。
   悔，不是后悔，
   是知道了进退存亡之道。
⑦ 角落批注：
   「知进退存亡而不失其正，
    其唯圣人乎？」
⑧ 收尾精神之语：
   「飞到最高，
    才知道什么叫太高。
    悔，是龙开始懂得的那一刻。」

【配色】
极深的暗蓝黑为主，
龙身用暗金（不是亮金，
是将熄的金），
天空用最深的墨蓝，
唯一的光是画面远处
那一点九五的余光——
极小，极暖，极远。
是六爻中最孤寂、
最深邃的一张，
也是最有哲学重量的一张。

16:9横幅，宋代高远山水意境，
现代平面设计结构。
```

## 乾卦六爻连读的视觉弧线

```
初九    →    九二    →    九三    →    九四    →    九五    →    上九
深渊         田野         书斋         渊口         苍穹         极高天
最暗         清醒         最紧绷       最悬         最亮         最孤
潜藏         初现         乾乾         或跃         飞龙         亢悔
墨黑    →    暖金    →    双色    →    冷金    →    纯金    →    暗金
```
