# 观卦（䷓）· 六爻提示词

> 卦序：第 20 卦 | 卦象：巽上坤下 | 风地观 | 卦德：观

观卦是易经最「静深」的一卦——风行地上，无处不到。「大观在上，顺而巽，中正以观天下」——观，不是旁观，是以中正之道观照天下。六爻：**童观→窥观→观我生→观国之光→观我生→观其生**。

## 观卦「六观」的深层结构

> 童观（初六）浅 → 窥观（六二）偏 → 观我生（六三）内省 ↺ → 观国之光（六四）向光 ☀️ → 观我生（九五）以民为镜 ◎ → 观其生（上九）志未平 ∞

## 临与观的深层对照

| | 临卦 | 观卦 |
|--|------|------|
| **核心** | 刚浸而长 | 中正以观天下 |
| **君子之道** | 教思无穷，容保民无疆 | 省方，观民设教 |
| **最深一句** | 教思无穷 | 圣人以神道设教，天下服矣 |

临是向下亲近，观是向上观照。先临后观，先亲近后设教。

>「圣人以神道设教，而天下服矣」——不是以力服，是以观照天地之道设教化民。观，观到了最深处；教，教到了最真处。
