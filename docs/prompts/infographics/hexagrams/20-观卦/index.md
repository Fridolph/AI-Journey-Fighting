# 观卦（䷓）· 六爻提示词

> 卦序：第 20 卦 &nbsp;|&nbsp; 卦象：巽上坤下 &nbsp;|&nbsp; 风地观 &nbsp;|&nbsp; 卦德：观

观卦，是易经最「神道设教」的一卦——风行地上，无所不至，是观最深的意象。「大观在上，顺而巽，中正以观天下」——中正以观，是观最深的根基。「观天之神道，而四时不忒，圣人以神道设教，而天下服矣」——神道设教，是观最深的宇宙观。「先王以省方，观民设教」——省方观民，是观最深的君子之道。六爻是观的六种面貌：**童观→窥观→观我生进退→观国之光→观我生君子无咎→观其生志未平**，是一部从小人道到志未平的完整观卦图鉴。

## 观卦「神道设教」的深层结构

> 童观小人道（初六）→窥观亦可丑（六二）→观我生未失道（六三）→观国之光尚宾（六四）→观我生观民（九五）→观其生志未平（上九◯）
> 观卦六层「观」的递进——童观→窥观→观我生→观国之光→观我生观民→观其生——是易经最完整的「观」的层次升华。

## 临与观的深层对照

| | 临卦 | 观卦 |
|--|------|------|
| **核心** | 大亨以正，天之道也，教思无穷 | 神道设教，中正以观天下 |
| **最深一句** | 教思无穷，容保民无疆 | 圣人以神道设教，而天下服矣 |

临是居高而临的治道，观是中正以观的感化。临而教思，观而设教——临与观是治理与教化的两个面向。

## 初六 · 童观，小人无咎，君子吝

```text
易经爻辞知识图鉴
观卦 · 初六 · 童观，小人无咎，君子吝

【画面主体意象】小人道，童观——画面以「风行地上」为整体背景，风行地上无所不至是观卦最根本的气息。

初六，「童观，小人无咎，君子吝」——如童子之观浅陋；「初六童观，小人道也」。

画面主体：童观小人道君子吝——「童观」是小人道；小人童观无咎，君子童观则吝。
童观是观最初的警示——君子不当童观。

【几何爻位结构】观卦六爻从下到上（巽上坤下）：初六 ━ ━  高亮深墨小人道 ← 当前爻 六二 ━ ━ 六三 ━ ━ 六四 ━ ━ 九五 ━━ 上九 ━━
标注：「巽上坤下，风行地上，观」

【嵌入式大字】「初六」嵌入童观小人道几何结构，字色用深墨，被小人道的阴暗部分遮挡——
「童观，小人无咎，君子吝」，字，童观，小人道，有观卦最初的小人道童观警示。

【信息模块】① 观卦 · 初六（右上角）② 爻辞：「童观，小人无咎，君子吝。」③ 小象传：「初六童观，小人道也。」
④ 彖传节录：「圣人以神道设教，而天下服矣。」⑤ 核心义理：观的警示是「童观」——童观是小人道；君子不当童观。
⑨ 收尾精神之语：「如童子之观——小人无咎君子吝；小人道也——童观是小人道君子不当童观。」

【配色】深墨·小人道为主，童观用深墨，小人道用暗墨，君子吝用暗灰，背景用深蓝风地。
16:9横幅，宋代风地观童观意境，现代平面设计结构。
```

## 九五 · 观我生，君子无咎

```text
易经爻辞知识图鉴
观卦 · 九五 · 观我生，君子无咎

【画面主体意象】观民，观我生君子无咎——九五是观卦最「观民」的一爻，「观我生」——观察自己的行为，

画面主体：观我生观民君子无咎——「观我生观民」观我生是观民，观民是观我生最深的延伸；

【几何爻位结构】六爻：初六 ━ ━（淡）六二 ━ ━（淡）六三 ━ ━（淡）六四 ━ ━（淡）九五 ━━  高亮纯金观民 ← 当前爻 上九 ━━（淡）

「观我生，君子无咎」，字，观我，观民，有观卦中最深的观我生观民君子无咎。

【信息模块】① 观卦 · 九五（右上角）② 爻辞：「观我生，君子无咎。」③ 小象传：「观我生，观民也。」
④ 核心义理：观中最「观民」的君主——观我生是观民；观民则君子无咎；圣人之神道设教而天下服矣。
```
