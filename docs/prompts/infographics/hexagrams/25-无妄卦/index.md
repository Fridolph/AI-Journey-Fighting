# 无妄卦（䷘）· 六爻提示词

> 卦序：第 25 卦 | 卦象：乾上震下 | 天雷无妄 | 卦德：真

无妄是易经最「纯正」的一卦——天下雷行，万物皆得无妄之道。「动而健，刚中而应，大亨以正，天之命也」——无妄是天命的状态。六爻：**无妄往吉→不耕获→无妄之灾→可贞无咎→无妄之疾→无妄行眚**。

## 无妄卦的深层结构

> 初九（得志往吉☀️）→六二（顺时自然）→六三（横祸无辜※冤屈）→九四（守固无咎）→九五（勿药自愈）→上九（穷极行眚⚠️）
> ※ 六三最深警示：无妄不能保证不受横祸——这是天道最诚实的承认。

## 复与无妄的深层对照

| | 复卦 | 无妄卦 |
|--|------|--------|
| **核心** | 刚反，动而以顺行 | 动而健，刚中而应 |
| **最深一句** | 复其见天地之心乎 | 大亨以正，天之命也 |

复是「归」，无妄是「真」。回归之后才能真实，真实才能顺天而动。

>「大亨以正，天之命也」——不是人力强求的亨，是以正，顺天命，自然而亨。不耕获，不菑畲，勿药，可贞——真实，是天命最深的赋予。
