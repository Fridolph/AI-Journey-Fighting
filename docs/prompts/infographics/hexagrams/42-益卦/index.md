# 益卦（䷩）· 六爻提示词

> 卦序：第 42 卦 &nbsp;|&nbsp; 卦象：巽上震下 &nbsp;|&nbsp; 风雷益 &nbsp;|&nbsp; 卦德：益

益卦，是易经最「广大」的一卦——风雷交作，损上益下，民说无疆。「益」，不是自益，是「自上下下，其道大光」——从上施益于下，是益卦最深的精神。「凡益之道，与时偕行」——益，与时俱进，日进无疆。六爻是益的六种面貌：**大作元吉→十朋之龟→益用凶事→告公从→有孚惠心→莫益之凶**，是一部从下受益到益极生凶的完整益道图鉴。

## 益卦「与时偕行」的深层结构

> 大作元吉（初九）→天益自外（六二）→凶事中益（六三）→告公迁国（六四）→刚中惠心（九五）→益极生凶（上九◯）
> 益，与时偕行——风雷交作，损上益下；凡益之道，与时偕行。

## 损与益的深层对照

| | 损卦 | 益卦 |
|--|------|------|
| **核心** | 损益盈虚，与时偕行 | 损上益下，民说无疆 |
| **最深一句** | 损而有孚，元吉 | 自上下下，其道大光 |

损是减损蓄养，益是广大施予。二簋可用享——简约，是损卦最深的精神；凡益之道，与时偕行——广大，是益卦最深的精神。

## 初九 · 利用为大作，元吉，无咎

## 益卦六爻 · 益道图鉴总结

```
初九 · 利用为大作    ——  受益大作，元吉无咎
六二 · 十朋之龟      ——  天益自外，永贞享帝
六三 · 益之用凶事    ——  凶事中益，有孚中行
六四 · 中行告公从    ——  中行告公，迁国益志
九五 · 有孚惠心      ——  刚中惠心，大得志
上九 · 莫益之        ——  益极生凶，立心勿恒
```

> **「凡益之道，与时偕行」**
> ——益卦的终极义理，不是无限施益，而是与时偕行——
> 风雷交作，损上益下，民说无疆；君子以见善则迁，有过则改——
> 益，最深的修身，是见善迁，过则改；
> 天施地生，其益无方——益，是天地最广大的施予；
> 益极则凶，与时偕行，是益道最深的智慧。
