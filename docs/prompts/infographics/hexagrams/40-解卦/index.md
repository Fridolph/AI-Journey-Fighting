# 解卦（䷧）· 六爻提示词

> 卦序：第 40 卦 &nbsp;|&nbsp; 卦象：震上坎下 &nbsp;|&nbsp; 雷水解 &nbsp;|&nbsp; 卦德：散

解——险以动，动而免乎险，解也。雷雨作而百果草木皆甲坼，是为解。然「解之时义大矣哉」——解难在时，赦过宥罪，是解卦最深的智慧。

## 卦辞总图 · 雷雨作·百果甲坼

```text
易经爻辞知识图鉴
解卦 · 卦辞总图 · 解之时义大矣哉

【画面主体意象】雷雨作，解；君子以赦过宥罪——

画面主体：一场雷雨正在大地上降临。画面上方雷云翻涌电光划破长空震雷滚滚；画面中央大雨倾盆雨幕如帘从天而降；画面下方大地之上百果草木在雷雨之中破壳而出——种子的外壳正在甲坼裂开新芽探出。画面右侧一个君子身影立于雨中双手展开姿态如「赦过宥罪」——是宽宥是解放是让一切重新开始。

【彖传节录】「天地解而雷雨作，雷雨作而百果草木皆甲坼」

【构图】横版 16:9，雷云占上三分之一雨幕占中甲坼草木占下，君子身影居右以天地之解对照人心之解。
【色调】雷云用深墨蓝紫，电光用冷白银蓝，雨幕用冷蓝银白，甲坼新芽用嫩绿暖金，君子用纯白。由险转解，生机勃发。
```

## 初六 · 无咎

```text
易经爻辞知识图鉴
解卦 · 初六 · 无咎

【画面主体意象】刚柔之际，义无咎也——初六是解卦最「际会」的一爻。

「无咎」——没有过失。初六是解卦第一爻解难刚刚开始，爻辞只有二字：无咎。简洁却意味深长。
「刚柔之际义无咎也」——初六阴爻与九二阳爻相邻，刚柔相济是解之初最自然的状态。解难之初不过分作为顺应刚柔之际自然无咎。

画面主体：雷雨刚过天色初晴。一道初晴柔光从云层透出。光照之处地面上有一道细微裂缝——是甲坼之始是解的开端极细极微却是真实的开始。
裂缝两侧一侧是刚岩一侧是柔土。极简大留白以少胜多。

【几何爻位结构】解卦从下到上（震上坎下）：初六 ━ ━  高亮冷青际会 ← 当前爻 九二 ━━ 六三 ━ ━ 九四 ━━ 六五 ━ ━ 上六 ━━
标注：「雷雨作，解；君子以赦过宥罪」

【嵌入式大字】「初六」嵌入刚柔际会几何结构，字色用冷青——「无咎」，字，际会，有解卦最初刚柔相际义无咎。

【信息模块】① 解卦 · 初六（右上角）② 爻辞：「无咎。」③ 小象传：「刚柔之际，义无咎也。」
④ 核心义理：解之初——不过分作为刚柔自然相济顺理无咎。⑨ 收尾之语：「无咎——刚柔之际义无咎；解之初顺理自然。」

【配色】冷青·际会为主，天色用冷青渐明，岩石用深灰，土壤用深赭，裂缝用冷白，光用淡金。极简留白。16:9横幅。
```

## 九二 · 田获三狐，得黄矢，贞吉

```text
易经爻辞知识图鉴
解卦 · 九二 · 田获三狐，得黄矢，贞吉

【画面主体意象】九二贞吉，得中道也——九二是解卦最「得中道」的一爻。

「田获三狐」——田猎中捕获三只狐狸。三狐是隐藏于野的狡猾之物是小人象征是潜伏的阴险之患。获三狐是除去潜伏隐患。
「得黄矢」——黄是中央之色中正之色。矢是箭是直。黄矢是中正而直的象征。除去小人得到的是中正之道。
「贞吉」——守正则吉。得中道也。

画面主体：一片旷野雷雨初歇。猎人身持弓矢已获三狐伏于脚边。猎人手中一支黄色箭矢在光中闪着金色——那黄矢是画面光源是中正之道象征。
三狐姿态各异是三种潜伏之患皆已被识破解除。背景天色渐明是解难之后的清朗。

【几何爻位结构】初六 ━ ━（淡）九二 ━━  高亮暗金得中 ← 当前爻 六三 ━ ━（淡）九四 ━━（淡）六五 ━ ━（淡）上六 ━━（淡）

【信息模块】① 解卦 · 九二（右上角）② 爻辞：「田获三狐得黄矢贞吉。」③ 小象传：「得中道也。」
④ 核心义理：解中得中——除潜伏小人守中正之道贞吉。⑥ 收尾之语：「田获三狐——得黄矢得中道也；中道是解难最深智慧。」

【配色】暗金·得中为主，旷野用深赭暖土，天色用冷青渐明，猎人用深墨，黄矢用纯金高亮。16:9横幅。
```

## 六三 · 负且乘，致寇至，贞吝

```text
易经爻辞知识图鉴
解卦 · 六三 · 负且乘，致寇至，贞吝

【画面主体意象】负且乘，亦可丑也；自我致戎，又谁咎也——六三是解卦最「自咎」的一爻。

「负且乘」——背着重物却坐在车上。负是小人贫贱者之事，乘是君子贵人之事。负且乘是以小人之质居君子之位——名不符实位不当德。
「致寇至」——招来了强盗。不是强盗无故而来是「自我致戎」是自己招来的。因为位不当德所以招来觊觎。
「贞吝」——守正也有遗憾。「又谁咎也」——是自己招来的能怪谁呢。解难先要解自身之「负且乘」。

画面主体：一辆华贵马车停于道路。车上人衣着华贵姿态傲慢是「乘」，但背上还背着沉重包袱是「负」——负且乘是最荒诞也最深刻的意象。
道路两侧强盗正在靠近——「致寇至」。负且乘的荒诞感是整幅画面最有张力的矛盾。

【几何爻位结构】初六 ━ ━（淡）九二 ━━（淡）六三 ━ ━  高亮深墨自咎 ← 当前爻 九四 ━━（淡）六五 ━ ━（淡）上六 ━━（淡）

【信息模块】① 解卦 · 六三（右上角）② 爻辞：「负且乘致寇至贞吝。」③ 小象传：「自我致戎又谁咎也。」
④ 核心义理：解中自咎——位不当德名不符实祸患自招。解难先解自身之负且乘。
⑥ 收尾之语：「负且乘——致寇至自招祸；自我致戎又谁咎也。」

【配色】深墨·自咎为主，马车用深赭华色，负者包袱用暗褐，强盗剪影用深墨冷色，道路用深赭。16:9横幅。
```

## 九四 · 解而拇，朋至斯孚

```text
易经爻辞知识图鉴
解卦 · 九四 · 解而拇，朋至斯孚

【画面主体意象】解而拇，未当位也——九四是解卦最「朋孚」的一爻。

「解而拇」——解开脚拇指上不当连结。九四与初六的不当牵连如细绳缠拇。解而拇是解除不当是自我纠正。
「朋至斯孚」——解除不当牵连后真正同道才能到来才能建立真正信任。
「未当位也」——九四阳居阴位未当，解而拇是纠正不当回归正位。先解不当牵连才能迎来真正同道。

画面主体：分两个时间层次——左半解拇之前：人影脚踝有细绳连着远处，两人之间有不自然。右半解拇之后：细绳已断人影站得更正更稳，
远方真正朋友正向走来步伐轻快面带信任。相遇处有一道光——「斯孚」是真正信任建立的那一刻。

【几何爻位结构】初六 ━ ━（淡）九二 ━━（淡）六三 ━ ━（淡）九四 ━━  高亮深蓝朋孚 ← 当前爻 六五 ━ ━（淡）上六 ━━（淡）

【信息模块】① 解卦 · 九四（右上角）② 爻辞：「解而拇朋至斯孚。」③ 小象传：「解而拇未当位也。」
④ 核心义理：解中正位——解除不当牵连回归正位朋至斯孚是解卦最深正位之道。
⑥ 收尾之语：「解而拇——朋至斯孚；断不当迎真朋，正位之道。」

【配色】深蓝·朋孚为主，左半用冷蓝深色细绳用暗红，右半渐暖朋友用暖金相遇光用纯金。由冷转暖。16:9横幅。
```

## 六五 · 君子维有解，吉；小人退也

```text
易经爻辞知识图鉴
解卦 · 六五 · 君子维有解，吉；小人退也

【画面主体意象】君子有解，小人退也——六五是解卦最「君解小退」的一爻。

「君子维有解吉」——君子能够解除则吉。六五柔居尊位以德化解以位解难。
「有孚于小人」——对小人有诚信的影响。君子的解不是以力压制而是以诚感化。小人感受到君子的诚信与德行自然退去。
「小人退也」——不是被驱逐是被感化而退。君子以德化人以诚感人小人自退不战而解。

画面主体：广阔庭院或朝堂之上。君子居中而立姿态从容不怒不争只是静静散发出一种中正气场如光晕向四周扩散。
画面四周小人身影在光晕边缘一步一步向后退去——不是仓皇是自然地退。明暗之间是「有孚」的力量——诚信光照退了小人。

【几何爻位结构】初六 ━ ━（淡）九二 ━━（淡）六三 ━ ━（淡）九四 ━━（淡）六五 ━ ━  高亮烟金君解 ← 当前爻 上六 ━━（淡）

【信息模块】① 解卦 · 六五（右上角）② 爻辞：「君子维有解，吉；有孚于小人。」③ 小象传：「小人退也。」
④ 核心义理：解中以德——君子以中正之德以诚信感化小人自退。不战而解是解卦最深君子之道。
⑧ 收尾之语：「君子有解——小人退也；以德化人不战而解。」

【配色】烟金·君解为主，君子身影用烟金光晕，中央用暖金明色，小人剪影用深墨冷色，退去方向用深蓝渐暗。16:9横幅。
```

## 上六 · 公用射隼于高墉，获之无不利

```text
易经爻辞知识图鉴
解卦 · 上六 · 公用射隼，于高墉之上，获之，无不利

【画面主体意象】公用射隼以解悖也——上六是解卦最「解悖」的一爻，也是解卦终局。

「公用射隼于高墉之上」——公侯用箭射高墙上的隼。隼是猛禽是凶悍悖逆之象是盘踞高处的顽固之患。高墉是高墙是险要之处。
「获之无不利」——成功射获无所不利。获之是彻底解决没有任何遗憾。
「以解悖也」——是为了解除悖逆。解难之终必有最后一个顽固悖逆需要果断出手一击而解无不利。

画面主体：高大城墙高耸入画面上方。城墙上隼展翅目光凶悍盘踞高处。画面前景公侯张弓搭箭目光如炬——弓弦绷紧箭矢蓄势是将发未发的那一刻。
一击解悖之后天地清明无不利。「解之时义大矣哉」

【几何爻位结构】初六 ━ ━（淡）九二 ━━（淡）六三 ━ ━（淡）九四 ━━（淡）六五 ━ ━（淡）上六 ━━  高亮纯金解悖 ← 当前爻

【信息模块】① 解卦 · 上六（右上角）② 爻辞：「公用射隼于高墉之上获之无不利。」③ 小象传：「以解悖也。」
④ 核心义理：解之终局——最后顽固悖逆果断一击获之无不利。解难到底不留尾患。
⑨ 收尾之语：「公用射隼——以解悖也；获之无不利；解之时义大矣哉。」

【配色】纯金·解悖为主，高墉用深赭青砖，隼用深墨冷色目光用冷银，公侯身影用深赭，弓矢用纯金高亮。张力十足蓄势待发。16:9横幅。
```
