# 升卦（䷭）· 六爻提示词

> 卦序：第 46 卦 &nbsp;|&nbsp; 卦象：坤上巽下 &nbsp;|&nbsp; 地风升 &nbsp;|&nbsp; 卦德：升

升卦，是易经最「生长」的一卦——地中生木，是种子破土、向上生长的卦象。「升」，是上升，是「柔以时升，巽而顺，刚中而应」——柔顺而与时俱进，是升卦最深的精神。「积小以高大」——升，不是一蹴而就，是积小成大的渐进。六爻是上升的六种面貌：**允升大吉→孚乃利用禴→升虚邑→王用亨于岐山→贞吉升阶→冥升不息**，是一部从上合志到消不富的完整升进图鉴。

## 升卦「积小以高大」的深层结构

> 允升大吉（初六）→孚乃利用禴（九二）→升虚邑（九三）→王用亨于岐山（六四）→贞吉升阶（六五）→冥升不息（上六◯）
> 地中生木，积小以高大——升不是一蹴而就，是顺德积累的渐进；冥升不知止，消不富，是升卦最深的警示。

## 萃与升的深层对照

| | 萃卦 | 升卦 |
|--|------|------|
| **核心** | 顺以说，刚中而应，观其所聚 | 柔以时升，巽而顺，积小以高大 |
| **最深一句** | 天地万物之情可见矣 | 积小以高大 |

萃是聚合，升是上升。萃聚而后能升——二者是万物运行的两个紧密环节。

## 初六 · 允升，大吉

```text
易经爻辞知识图鉴
升卦 · 初六 · 允升，大吉

【画面主体意象】
上合志，允升大吉——
画面以「地中生木」为整体背景，大地厚重于下，木从地中破土而出——
柔以时升，巽而顺，是升卦最根本的气息。

初六，「允升，大吉」——允升是被允许上升，是得到认可的上升；「大吉」；「上合志也」——与上位之志相合。

画面主体：一棵幼木破土而出——「允升」是被允许上升，不强行，得到认可；
「大吉」是允升之后最深的吉祥；「上合志也」是与上位相合——升从上合志开始，是升卦最稳固的起点。

「允升」的意象：允是允许和认可——初六柔顺居初，不强行上升，而是得到上位的允许；
「允升」是柔顺上升最深的方式；「大吉」是允升之后最自然的结果。

【几何爻位结构】
升卦六爻从下到上（坤上巽下）：初六 ━ ━  高亮深赭允升 ← 当前爻 九二 ━━ 九三 ━━ 六四 ━ ━ 六五 ━ ━ 上六 ━ ━
标注：「坤上巽下，地中生木」

【嵌入式大字】
「初六」嵌入幼木破土几何结构，字色用深赭，被破土而出的动势部分遮挡——
「允升，大吉」，字，允升，大吉，有升卦最初的生长喜悦。

【信息模块】

③ 小象传：「允升大吉，上合志也。」
④ 象传节录：「地中生木，升；君子以顺德，积小以高大。」
⑤ 核心义理：升的起点是「允」——不强行，被允许；「上合志」是允升最深的根基；

⑦ 收尾精神之语：「幼木破土而出——被允许上升；大吉是允升之后最圆满的吉祥；上合志，是升卦最稳固的开端。」

【配色】深赭·允升为主，幼木破土用深赭，允升大吉用暖金，上合志用深金，背景用深暖绿。
```

## 升卦六爻 · 升进图鉴总结

```

```

> **「积小以高大」**
> ——升卦的终极义理，不是一蹴而就，而是「顺德，积小以高大」——
> 地中生木是最深的生长哲学；柔以时升，巽而顺，刚中而应——升要与时俱进；
> 允升，孚升，顺升，贞升——是升卦最完整的升进之道；冥升，是升卦最深的警示：升要知止。
