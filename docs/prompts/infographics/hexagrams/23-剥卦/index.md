# 剥卦（䷖）· 六爻提示词

> 卦序：第 23 卦 | 卦象：艮上坤下 | 山地剥 | 卦德：剥

剥卦是易经最「凛冽」的一卦——山附地上，山被剥蚀，终将崩落。「君子尚消息盈虚，天行也」——剥不是终点，剥极则复，硕果不食，种子在最后一刻保存了生机。六爻：**剥床足→剥床辨→剥之→剥床肤→贯鱼→硕果不食**。

## 剥卦「消息盈虚」的深层结构

> 初六（灭下凶↓）→六二（无援凶↓）→六三（独立无咎◎）→六四（切近凶↓）→六五（顺从无不利↑）→上九（硕果生机☀️）
> 剥极则复——最后一颗果实，是复卦的种子。

## 贲与剥的深层对照

| | 贲卦 | 剥卦 |
|--|------|------|
| **核心** | 文明以止，人文化成 | 顺而止之，消息盈虚 |
| **最深一句** | 观乎人文，以化成天下 | 君子尚消息盈虚，天行也 |

白贲归真，是文采的终点；硕果保全，是生机的起点。

>「硕果不食，君子得舆，民所载也」——剥蚀到了极处，还有一颗果实没有被吃掉。这是复卦的种子，是阳气重新萌发的起点。剥极则复，是天道最深的承诺。
