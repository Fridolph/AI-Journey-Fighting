# 剥卦（䷖）· 六爻提示词

> 卦序：第 23 卦 &nbsp;|&nbsp; 卦象：艮上坤下 &nbsp;|&nbsp; 山地剥 &nbsp;|&nbsp; 卦德：落

剥——柔变刚也。五阴剥一阳，山附地上，剥落之象。「君子尚消息盈虚，天行也」——这是易经中最「顺势待时」的一卦。

## 剥卦「消息盈虚天行也」的深层结构

> 剥足灭下（初六）→剥辨无与（六二）→独处无咎（六三）→剥肤切近（六四）→贯鱼顺从（六五）→硕果不食剥极必复（上九◯）
> 五阴剥落唯一阳留存——硕果不食剥极必复；消息盈虚天行也。

## 贲与剥的深层对照

| | 贲卦 | 剥卦 |
|--|------|------|
| **核心** | 观乎人文以化成天下，白贲归真 | 消息盈虚天行也，硕果不食 |
| **最深一句** | 白贲无咎，上得志也 | 君子尚消息盈虚，天行也 |

贲是文采至极，剥是剥落至极。文采至极归于素白，剥落至极必复阳刚——二卦皆蕴含「极必反」的天道智慧。

## 初六 · 剥床以足，蔑贞凶

```text
易经爻辞知识图鉴
剥卦 · 初六 · 剥床以足，蔑贞凶

【画面主体意象】剥床以足，以灭下也——初六是剥卦最「根基剥落」的一爻，「剥床以足」——剥落床的足腿，
「蔑贞凶」——蔑视正道则凶；「以灭下也」——灭其根基。剥落从根基开始，蔑贞则凶。

【几何爻位结构】剥卦六爻从下到上（艮上坤下）：初六 ━ ━  高亮深赭剥足 ← 当前爻 六二 ━ ━ 六三 ━ ━ 六四 ━ ━ 六五 ━ ━ 上九 ━━
标注：「山附地上，剥；上以厚下，安宅」

【嵌入式大字】「初六」嵌入剥床以足几何结构，字色用深赭，被灭下蔑贞的厚重部分遮挡——
「剥床以足，蔑贞凶」，字，剥足，灭下，有剥卦最初的根基剥落警示。

【信息模块】① 剥卦 · 初六（右上角）② 爻辞：「剥床以足，蔑贞凶。」③ 小象传：「剥床以足，以灭下也。」
④ 彖传节录：「君子尚消息盈虚，天行也。」⑤ 核心义理：剥道起点——剥落从根基开始；蔑贞则凶。
⑨ 收尾精神之语：「剥落床足——蔑贞凶；以灭下也——根基既灭则凶。」

【配色】深赭·剥足为主，剥床以足用深赭，蔑贞用暗灰，凶用暗红，背景用深蓝山地。
16:9横幅，宋代剥落警示意境，现代平面设计结构。
```

## 六三 · 剥之，无咎

```text
易经爻辞知识图鉴
剥卦 · 六三 · 剥之，无咎

【画面主体意象】剥之无咎，失上下也——六三是剥卦最「独处」的一爻，「剥之」——剥落，「无咎」——无咎；
「失上下也」——失去上下依附独处剥中。五阴中唯六三无咎，因为失上下独处自守不随波逐流。

【几何爻位结构】六爻：初六 ━ ━（淡）六二 ━ ━（淡）六三 ━ ━  高亮暗金独处 ← 当前爻 六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━（淡）

【嵌入式大字】「六三」嵌入剥之无咎几何结构，字色用暗金，被失上下独处的孤寂部分遮挡——
「剥之，无咎」，字，剥之，独处，有剥卦最深的失上下无咎。

【信息模块】① 剥卦· 六三（右上角）② 爻辞：「剥之，无咎。」③ 小象传：「剥之无咎，失上下也。」
④ 核心义理：剥道的独处之智——五阴中唯六三无咎；失上下独处不随波逐流。独处是剥卦最深的自守智慧。
⑧ 收尾精神之语：「剥落——无咎；失上下也——独处自守不随波逐流。」
```

## 六四 · 剥床以肤，凶

```text
易经爻辞知识图鉴

【画面主体意象】剥床以肤，切近灾也——六四是剥卦最「切近灾」的一爻，「剥床以肤」——剥落到床皮面，「凶」；「切近灾也」。

【几何爻位结构】六爻：初六 ━ ━（淡）六二 ━ ━（淡）六三 ━ ━（淡）六四 ━ ━  高亮暗红切近 ← 当前爻 六五 ━ ━（淡）上九 ━━（淡）
```

## 上九 · 硕果不食，君子得舆，小人剥庐

```text
易经爻辞知识图鉴
剥卦 · 上九 · 硕果不食，君子得舆，小人剥庐

【画面主体意象】君子得舆民所载也，小人剥庐终不可用也——上九是剥卦最「硕果」的一爻，
「硕果不食」——五阴剥落后唯一阳留存，天道留存最后阳刚；
「君子得舆」——君子顺应天道民众承载；「小人剥庐」——小人逆天自食其果。
剥极必复消息盈虚天行也——剥卦直接预告复卦「一阳来复」。

【几何爻位结构】六爻：初六 ━ ━（淡）六二 ━ ━（淡）六三 ━ ━（淡）六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━  高亮深金硕果 ← 当前爻
标注：「君子得舆，民所载也；小人剥庐，终不可用也」

【嵌入式大字】「上九」嵌入硕果不食几何结构，字色用深金，被硕果留存的光芒四方普照——
「硕果不食，君子得舆，小人剥庐」，字，硕果，得舆，有剥卦最终的天道留存；剥极必复。

【信息模块】① 剥卦 · 上九（右上角）② 爻辞：「硕果不食，君子得舆，小人剥庐。」
③ 小象传：「君子得舆，民所载也。小人剥庐，终不可用也。」④ 彖传节录：「君子尚消息盈虚，天行也。」
⑤ 核心义理：剥道的终极对比——硕果不食天道留存；君子得舆民心所向；小人剥庐自食其果。剥极必复是天行也。
⑨ 收尾精神之语：「硕果不食——剥极必复；君子得舆民所载小人剥庐终不可用；天行也。」

【配色】深金·硕果为主，硕果不食用深金，君子得舆用纯金，小人剥庐用暗黑，民所载用暖金，背景用深蓝山地。
16:9横幅，宋代硕果天道意境，现代平面设计结构。
```
