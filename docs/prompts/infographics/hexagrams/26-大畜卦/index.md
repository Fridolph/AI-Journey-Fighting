# 大畜卦（䷙）· 六爻提示词

> 卦序：第 26 卦 | 卦象：艮上乾下 | 山天大畜 | 卦德：蓄

大畜是易经最「蓄养」的一卦——天在山中，乾刚被艮止蓄。「刚健笃实辉光，日新其德」——蓄养到辉光，每天更新德行。六爻：**有厉利已→舆说辐→良马逐→童豕牿→豶豕牙→何天之衢**。

## 大畜「日新其德」的深层结构

> 止危（初九）→自止（九二）→奔驰（九三）→早制（六四外）→深制（六五内）→通天（上九☀️）
> 止，是为了蓄；蓄，是为了行；行，是道大行。

## 无妄与大畜的深层对照

| | 无妄卦 | 大畜卦 |
|--|--------|--------|
| **核心** | 动而健，天之命 | 能止健，大正也 |
| **最深一句** | 大亨以正，天之命也 | 刚健笃实辉光，日新其德 |

无妄是真，大畜是蓄。真实是蓄养的前提，蓄养是真实的积累。动要真实，止要蓄养。

> 「多识前言往行，以畜其德」——读史识人，积累前人言行，不是为了博学，是为了畜德。止，是为了通；蓄，是为了行。
