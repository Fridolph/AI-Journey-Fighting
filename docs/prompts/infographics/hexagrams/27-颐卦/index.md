# 颐卦（䷚）· 六爻提示词

> 卦序：第 27 卦 &nbsp;|&nbsp; 卦象：艮上震下 &nbsp;|&nbsp; 山雷颐 &nbsp;|&nbsp; 卦德：养

颐——养也。山下有雷，颐养之象。「天地养万物，圣人养贤，以及万民；颐之时义大矣哉！」——这是易经中最「养正」的一卦。

## 颐卦「养正则吉」的深层结构

> 舍龟观颐（初九）→颠颐拂经（六二）→拂颐大悖（六三）→虎视施光（六四）→居贞顺上（六五）→由颐大有庆（上九◯）
> 从自养到养贤到养万民——层层扩展的圆满历程；六四虎视眈眈是颐卦最生动的养贤意象。

## 大畜与颐的深层对照

| | 大畜卦 | 颐卦 |
|--|------|------|
| **核心** | 刚健笃实辉光，日新其德，多识前言往行 | 养正则吉，天地养万物圣人养贤以及万民 |
| **最深一句** | 何天之衢，道大行也 | 由颐厉吉，大有庆也 |

大畜是蓄德养贤，颐是养正方民。大畜而后颐——蓄德已成而后养正施光，二卦是蓄与养的递进。

## 初九 · 舍尔灵龟，观我朵颐，凶

```text
易经爻辞知识图鉴
颐卦 · 初九 · 舍尔灵龟，观我朵颐，凶

【画面主体意象】观我朵颐，亦不足贵也——初九是「舍正求邪」的一爻，「舍尔灵龟」——舍弃自养能力，「观我朵颐」——羡慕外养，
「凶」；「亦不足贵也」。自养是颐道最深的根基；舍弃自养羡慕外养不足贵凶。

【几何爻位结构】颐卦六爻从下到上（艮上震下）：初九 ━━  高亮暗褐舍正 ← 当前爻 六二 ━ ━ 六三 ━ ━ 六四 ━ ━ 六五 ━ ━ 上九 ━━
标注：「山下有雷，颐；君子以慎言语，节饮食」

【嵌入式大字】「初九」嵌入舍龟观颐几何结构，字色用暗褐，被不足贵凶的沉重部分遮挡——
「舍尔灵龟，观我朵颐，凶」，字，舍龟，朵颐，有颐卦最初的舍正求邪警示。

【信息模块】① 颐卦 · 初九（右上角）② 爻辞：「舍尔灵龟，观我朵颐，凶。」③ 小象传：「观我朵颐，亦不足贵也。」
④ 彖传节录：「颐之时义大矣哉！」⑤ 核心义理：颐道起点警示——自养是根基；舍自养求外养不足贵凶。
⑨ 收尾精神之语：「舍弃灵龟——羡慕咀嚼凶；亦不足贵也——自养是颐道最深的根基。」

【配色】暗褐·舍正为主，舍龟用暗褐，观我朵颐用深赭，凶用暗红，不足贵用深灰，背景用深蓝山雷。
16:9横幅，宋代舍正求邪警示意境，现代平面设计结构。
```

```text
【配色】深赭·失类为主，颠颐用深赭，拂经用暗褐，征凶用暗红，行失类用深灰，背景用深蓝山雷。
```

```text
【配色】深金·施光为主，颠颐吉用深金，虎视眈眈用暗金，其欲逐逐用暖金，上施光用纯金，背景用深蓝山雷。
```

## 上九 · 由颐，厉吉，利涉大川

```text
易经爻辞知识图鉴

【画面主体意象】由颐厉吉，大有庆也——上九是「大有庆」的一爻，「由颐」——众人由此求养，「厉吉」——危厉而吉，
「利涉大川」——利于涉大川；「大有庆也」。由颐厉吉大有庆是颐道最终圆满。天地养万物圣人养贤以及万民。

【几何爻位结构】六爻：初九 ━━（淡）六二 ━ ━（淡）六三 ━ ━（淡）六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━  高亮纯金大庆 ← 当前爻
标注：「由颐厉吉，大有庆也」

【嵌入式大字】「上九」嵌入由颐大庆几何结构，字色用纯金，被大有庆的光芒四方普照天下——
「由颐，厉吉，利涉大川」，字，由颐，大庆，有颐卦最终大有庆通达；众人由此求养大有庆。

【信息模块】① 颐卦 · 上九（右上角）② 爻辞：「由颐，厉吉，利涉大川。」③ 小象传：「由颐厉吉，大有庆也。」

⑨ 收尾精神之语：「由颐厉吉——大有庆利涉大川；颐之时义大矣哉。」
```
