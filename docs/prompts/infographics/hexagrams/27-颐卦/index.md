# 颐卦（䷚）· 六爻提示词

> 卦序：第 27 卦 | 卦象：艮上震下 | 山雷颐 | 卦德：养

颐卦是易经最「养」的一卦——山下有雷，形如口颐。「天地养万物，圣人养贤，以及万民；颐之时义大矣哉」——从口腹到天下。六爻：**舍龟朵颐→颠颐拂经→拂颐贞凶→颠颐虎视→拂经居贞→由颐厉吉**。

## 颐卦「养正则吉」的深层结构

> 失养（初九舍龟）→颠养（六二失类）→悖道（六三十年凶）→转折↑虎视（六四上施光）→居贞（六五顺上）→养源（上九大有庆☀️）
> 养，要正；正，在方向；方向，在内求，在顺上，在成为来源。

## 大畜与颐的深层对照

| | 大畜卦 | 颐卦 |
|--|--------|------|
| **核心** | 刚健笃实辉光，日新其德 | 养正则吉，颐之时义大矣 |
| **最深一句** | 何天之衢，道大行也 | 天地养万物，圣人养贤，以及万民 |

蓄而后养。大畜蓄德至天衢，颐养正至万民。

> 「天地养万物，圣人养贤，以及万民」——养，不只是口腹。慎言语是养德，节饮食是养身，养贤是养天下。从舍龟失养到由颐大庆——养，从内求开始，到成为来源结束。
