# 节卦（䷻）· 六爻提示词

> 卦序：第 60 卦 &nbsp;|&nbsp; 卦象：坎上兑下 &nbsp;|&nbsp; 水泽节 &nbsp;|&nbsp; 卦德：制

节卦，是易经最「节制」的一卦——泽上有水，是水在泽中有节制的卦象。「节，亨」——节，是亨通。「天地节而四时成，节以制度，不伤财，不害民」——节，是天地最深的运行法则。「苦节不可贞，其道穷也」——节，不可过苦，过苦则道穷。「君子以制数度，议德行」——节，是制数度、议德行最深的精神。六爻是节制的六种面貌：**不出户庭→不出门庭→不节若嗟→安节→甘节→苦节**，是一部从知通塞到道穷凶的完整节制图鉴。

## 节卦「天地节而四时成」的深层结构

> 不出户庭知通塞（初九）→不出门庭失时极（九二）→不节自嗟（六三）→安节承上（六四）→甘节居中（九五）→苦节道穷（上六◯）
> 甘节·安节·苦节——三种节制三种境界；苦节不可贞其道穷也；知通塞是节制最初的智慧；制数度议德行是节卦最深的君子之道。

## 涣与节的深层对照

| | 涣卦 | 节卦 |
|--|------|------|
| **核心** | 涣有丘匪夷所思，王假有庙 | 天地节而四时成，甘节居中 |
| **最深一句** | 涣有丘，匪夷所思 | 苦节不可贞，其道穷也 |

涣是涣散消解，节是节制有度。涣散而后知节制——涣与节，是散与制的两种天道运行。

## 初九 · 不出户庭，无咎

```text
易经爻辞知识图鉴
节卦 · 初九 · 不出户庭，无咎

【画面主体意象】知通塞，慎节无咎——画面以「泽上有水」为整体背景，水在泽中有节有制是节卦最根本的气息。

初九，「不出户庭，无咎」——不出户庭是不出内庭；「知通塞也」——知道通与塞。

画面主体：不出户庭知通塞——「不出户庭」是慎重节制最深的表现；「知通塞也」——知道何时通何时塞；
知通塞则无咎——知通塞是节制最深的根基。

【几何爻位结构】节卦六爻从下到上（坎上兑下）：初九 ━━  高亮深赭户庭 ← 当前爻 九二 ━━ 六三 ━ ━ 六四 ━ ━ 九五 ━━ 上六 ━ ━
标注：「坎上兑下，泽上有水，节」

【嵌入式大字】「初九」嵌入户庭知通塞几何结构，字色用深赭，被泽上有水的节制部分遮挡——
「不出户庭，无咎」，字，户庭，知通塞，有节卦最初的慎节无咎。

【信息模块】① 节卦 · 初九（右上角）② 爻辞：「不出户庭，无咎。」③ 小象传：「不出户庭，知通塞也。」
④ 彖传节录：「天地节而四时成，节以制度，不伤财，不害民。」
⑤ 核心义理：节的起点是「知通塞」——不出户庭是慎重节制；「知通塞也」——知道通与塞则无咎。
⑨ 收尾精神之语：「不出内庭——无过咎；知通塞也——知道通与塞是节制最深的智慧。」

【配色】深赭·户庭为主，不出户庭用深赭，知通塞用暗金，无咎用素白，背景用深蓝水泽。
```
