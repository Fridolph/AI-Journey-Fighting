# 临卦（䷒）· 六爻提示词

> 卦序：第 19 卦 | 卦象：坤上兑下 | 地泽临 | 卦德：临

临卦是易经最「亲近」的一卦——泽上有地，大地临于泽上，居高临下而亲近万物。「刚浸而长」——阳刚渐渐增长，泰运将至的前兆。六爻：**咸临→咸临→甘临→至临→知临→敦临**。

## 临卦「六临」的深层结构

> 咸临（初九）感应之临 → 咸临（九二）刚中之临 → 甘临（六三）⚠️甘言之临 → 至临（六四）亲至之临 → 知临（六五）☀️智慧之临 → 敦临（上六）◯敦厚之临

## 蛊与临的深层对照

| | 蛊卦 | 临卦 |
|--|------|------|
| **核心** | 终则有始 | 刚浸而长 |
| **君子之道** | 振民育德 | 教思无穷，容保民无疆 |
| **最深一句** | 先甲三日，后甲三日 | 至于八月有凶，消不久也 |

整治之后，刚阳渐长。临，是蛊之后最自然的生机。

> 象传：「教思无穷，容保民无疆」——教，无穷；容，无疆。大地临泽，不是管制，是教化，是包容。
