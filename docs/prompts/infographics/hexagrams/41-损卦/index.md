# 损卦（䷨）· 六爻提示词

> 卦序：第 41 卦 &nbsp;|&nbsp; 卦象：艮上兑下 &nbsp;|&nbsp; 山泽损 &nbsp;|&nbsp; 卦德：损

损——损下益上，其道上行。山下有泽，泽深山高，损下以益上，是为损。然「损益盈虚，与时偕行」——损之有时，二簋可享。

## 卦辞总图 · 二簋可用享

```text
易经爻辞知识图鉴
损卦 · 卦辞总图 · 损益盈虚与时偕行

【画面主体意象】山下有泽损，君子以惩忿窒欲——

画面主体：高山居画面右侧山势巍峨是艮卦之象（上卦受益方）。山脚之下一片泽水居左水面开阔却正在缓缓下降——水位降低是兑泽之象（下卦损减方）。
泽水下降之处水流沿山根向上渗透滋养山体——损下以益上。画面中央一张简朴祭台上只有两只竹簋——「二簋可用享」的意象。
祭台前一个人影恭敬而立不因简朴而轻慢——有孚是那个人影最深的气质。

【彖传节录】「二簋应有时，损刚益柔有时，损益盈虚，与时偕行」

【构图】横版 16:9，高山居右下降泽水居左，二簋祭台居中。山体用深青灰墨，泽水用深蓝冷色，祭台用深赭暖木，二簋用暗金，人影用纯白。
```

## 初九 · 已事遄往，酌损之，无咎

```text
易经爻辞知识图鉴
损卦 · 初九 · 已事遄往，酌损之，无咎

【画面主体意象】已事遄往，尚合志也——初九是「合志速往」的一爻。

「已事遄往」——事情办完迅速前往上位。初九阳爻与六四相应，不拖延速往是损下益上的第一步。
「酌损之」——斟酌着损减。酌是斟酌是适度，损不是无限损而是斟酌着损——损之有度是初九最深的损道智慧。

画面主体：一条向上延伸的山路，旅人事已毕正迅速向上攀登步伐轻快方向明确。手中捧着一个简朴器物是「酌损之」的象征——不是空手而是带着斟酌过的适度贡献。
路边有一个量器是「酌」的意象——斟酌是损有度最深的体现。山顶有光透出是六四等待的方向。

【几何爻位结构】损卦从下到上（艮上兑下）：初九 ━━  高亮冷青合志 ← 当前爻 九二 ━━ 六三 ━ ━ 六四 ━ ━ 六五 ━ ━ 上九 ━━
标注：「山下有泽，损；君子以惩忿窒欲」

【信息模块】① 损卦 · 初九 ② 爻辞：「已事遄往无咎酌损之。」③ 小象传：「已事遄往尚合志也。」
④ 核心义理：损之初——事毕速往斟酌损减合志上行。⑨ 收尾之语：「事毕速往——酌损有度；尚合志也——损之初最深的智慧。」

【配色】冷青·合志为主。16:9横幅。
```

## 九二 · 利贞，征凶，弗损益之

```text
易经爻辞知识图鉴
损卦 · 九二 · 利贞，征凶，弗损益之

【画面主体意象】九二利贞，中以为志也——九二是「守中」的一爻。

「利贞」——守正则利。九二居中守正是最深的根基。「征凶」——轻易前进离开中位则凶。守中才是九二的正道。
「弗损益之」——不损减反而是益。九二守中不动不轻易损减自身反而是对上位的真正增益。
「中以为志也」——以中正为志向。损不是无条件损——守中才是真正的益；有时候不损才是最大的损道。

画面主体：画面中央一根笔直柱子深深扎根于地稳立不动——是「守中」的象征。左右有力量拉扯削减，但柱子纹丝不动居中而立。
根部深入大地是「中以为志」的根基，顶端有光是守中之后自然而来的增益。

【几何爻位结构】初九 ━━（淡）九二 ━━  高亮深赭守中 ← 当前爻 六三 ━ ━（淡）六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━（淡）

【信息模块】① 损卦 · 九二 ② 爻辞：「利贞，征凶，弗损益之。」③ 小象传：「中以为志也。」
④ 核心义理：损中守中——不轻易前进不轻易损减以中正为志弗损反益。⑧ 收尾之语：「弗损益之——中以为志；守中是最深的损道。」

【配色】深赭·守中为主，中柱用深赭，拉扯用冷蓝，削减用暗红，顶端光用暖金。16:9横幅。
```

## 六三 · 三人行损一人，一人行得友

```text
易经爻辞知识图鉴
损卦 · 六三 · 三人行损一，一人行得友

【画面主体意象】一人行，三则疑也——六三是「专一得友」的一爻。

「三人行则损一人」——三人同行人多则疑志向分散，必须损去一人才能专一。
「一人行则得其友」——一人独行志向专一反而能感召真正志同道合的朋友。
「一人行三则疑也」——损去多余专一而行才能得到真正同道。

画面主体：左半三人行——三人方向各异志散「三则疑」；右半一人独行步伐坚定方向明确，前方真友从远处走来方向完全一致。相遇处有一道光。
从三到一是一个「损」的转化。

【几何爻位结构】初九 ━━（淡）九二 ━━（淡）六三 ━ ━  高亮暗金专一 ← 当前爻 六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━（淡）

【信息模块】① 损卦 · 六三 ② 爻辞：「三人行损一人；一人行得其友。」③ 小象传：「一人行三则疑也。」
④ 核心义理：损中专一——损去多余关系专一行反得真友。⑥ 收尾之语：「三则疑损——一得友；专一是损卦最深人际智慧。」

【配色】暗金·专一为主。16:9横幅。
```

## 六四 · 损其疾，使遄有喜，无咎

```text
易经爻辞知识图鉴
损卦 · 六四 · 损其疾，使遄有喜，无咎

【画面主体意象】损其疾，亦可喜也——六四是「损疾有喜」的一爻。

「损其疾」——主动减损自身缺陷。疾是六四自身不足，损其疾是一种自我修正。
「使遄有喜」——损去疾患迅速带来喜庆。损在这里不是失去而是通过减损缺陷获得喜庆。
「亦可喜也」——主动损疾是最值得喜悦的损——损去缺陷光从内部透出。

画面主体：左人影身上有阴影（疾）正被双手轻轻剥离。阴影剥离处露出光——「有喜」是损去疾患光从内透出。
右同影疾患已除身姿更正面容有喜——「遄有喜」。损疾到有喜是一个迅速转化。

【几何爻位结构】初九 ━━（淡）九二 ━━（淡）六三 ━ ━（淡）六四 ━ ━  高亮深蓝损疾 ← 当前爻 六五 ━ ━（淡）上九 ━━（淡）

【信息模块】① 损卦 · 六四 ② 爻辞：「损其疾，使遄有喜，无咎。」③ 小象传：「损其疾亦可喜也。」
④ 核心义理：损中得喜——主动损去自身缺陷迅速带来喜庆损疾即是得喜。⑥ 收尾之语：「损其疾——遄有喜；损去缺陷是最深喜。」

【配色】深蓝·损疾为主。16:9横幅。
```

## 六五 · 或益之，十朋之龟弗克违，元吉

```text
易经爻辞知识图鉴
损卦 · 六五 · 或益之，十朋之龟弗克违，元吉

【画面主体意象】六五元吉，自上佑也——六五是「天佑元吉」的一爻。

「或益之」——有人来增益他。六五以柔居尊损而有孚天下自然来益。
「十朋之龟弗克违」——最珍贵的占卜之龟也无法违背，连神龟占卜都显示吉兆——是上天的佑护无可置疑的吉。
「元吉」——大吉。「自上佑也」——来自上天的佑护。六五之所以元吉不是因为强取而是因为损而有孚天自佑之。

画面主体：人影居中而立姿态谦逊。光从天而降「自上佑」。光中隐约可见神龟轮廓「十朋龟」。四野众人向中央汇聚「或益之」——损己有孚天下自来益。

【几何爻位结构】初九 ━━（淡）九二 ━━（淡）六三 ━ ━（淡）六四 ━ ━（淡）六五 ━ ━  高亮烟金天佑 ← 当前爻 上九 ━━（淡）

【信息模块】① 损卦 · 六五 ② 爻辞：「或益之，十朋之龟弗克违，元吉。」③ 小象传：「六五元吉自上佑也。」
④ 核心义理：损中天佑——损己有孚天自佑之元吉是损卦最深天道回应。⑧ 收尾之语：「或益之——十朋龟弗克违；自上佑也元吉。」

【配色】烟金·天佑为主，天光用纯金烟金，神龟轮廓用冷银，众人来益用暖色。16:9横幅。
```

## 上九 · 弗损益之，得臣无家，大得志

```text
易经爻辞知识图鉴
损卦 · 上九 · 弗损益之，得臣无家，大得志也

【画面主体意象】弗损益之，大得志也——上九是损卦「大得志」的终局。

「弗损益之」——损极反益。道终不再损减而是自然增益。
「得臣无家」——得到天下归附却没有私心没有私家。损去私欲得到天下——是损道最高境界。
「大得志也」——大大实现了志向。损之极是无私是无家是得天下是大得志。

画面主体：人影立于高山之巅身姿开阔双手展开如拥抱天下。身后没有家没有围墙——「无家」天地就是他的家。
面前四面八方人影汇聚——「得臣」是损去私欲自然结果。山下泽水已充盛——损道终益道启。天空辽阔星辰璀璨「大得志」。「损益盈虚与时偕行」

【几何爻位结构】初九 ━━（淡）九二 ━━（淡）六三 ━ ━（淡）六四 ━ ━（淡）六五 ━ ━（淡）上九 ━━  高亮纯金大志 ← 当前爻

【信息模块】① 损卦 · 上九 ② 爻辞：「弗损益之无咎贞吉利有攸往得臣无家。」③ 小象传：「大得志也。」
④ 核心义理：损之终局——损极反益损去私欲得臣无家天下归附大得志。惩忿窒欲最终圆满境界。
⑨ 收尾之语：「弗损益之——得臣无家；大得志也；损益盈虚与时偕行。」

【配色】纯金·大志为主，山巅用深青灰，人影用纯金光晕，归附用暖色，泽水用深蓝，星光用冷银。16:9横幅。
```
