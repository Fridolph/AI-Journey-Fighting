# 损卦（䷨）· 六爻提示词

> 卦序：第 41 卦 | 卦象：艮上兑下 | 山泽损 | 卦德：损

损卦是易经最「深刻」的一卦——山下有泽，损下益上。「损益盈虚，与时偕行」——减损是为了更深的充盈。六爻：**已事遄往→利贞弗损→三人行损一人→损其疾→十朋之龟→弗损益之**。

## 损卦「与时偕行」的深层结构

> 酌损合志（初九）→守中不损（九二）→损多得友（六三）→损疾得喜（六四）→天佑元吉（六五）→不损大志（上九◯）
> 最深的损，是不损；最深的益，是得臣无家。

## 解与损的深层对照

| | 解卦 | 损卦 |
|--|------|------|
| **核心** | 险以动，动而免乎险 | 损益盈虚，与时偕行 |
| **最深一句** | 解之时义大矣哉 | 损而有孚，元吉 |

解是舒展释放，损是减损蓄养。二簋可用享——简约，是损卦最深的精神。
