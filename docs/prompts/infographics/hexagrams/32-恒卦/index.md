# 恒卦（䷟）· 六爻提示词

> 卦序：第 32 卦 | 卦象：震上巽下 | 雷风恒 | 卦德：久

恒卦是易经最「久」的一卦——雷风相与，刚柔皆应。「天地之道，恒久而不已也」——日月以久照为德，四时以久成为功，圣人以久于其道化成天下。六爻：**浚恒始深→悔亡久中→不恒承羞→田无禽→恒德从一→振恒无功**。

## 恒卦「立不易方」的深层结构

> 急深凶（初六↓）→久中悔亡（九二◎）→德不恒失（九三↓）→久非其位（九四）→因位而恒（六五辩）→振动大无功（上六↓）
> 恒要正位正德。久非其位是误，德不恒是失，振动是终极的失恒。

## 咸与恒的深层对照

| | 咸卦 | 恒卦 |
|--|------|------|
| **核心** | 感也，二气感应 | 久也，恒久不已 |
| **最深一句** | 君子以虚受人 | 君子以立不易方 |

咸是感的起点，恒是久的根本。感要虚，恒要正。

> 「日月得天而能久照，四时变化而能久成，圣人久于其道而天下化成」——久，不是固执，是恒于正道。立不易方，是恒卦给君子最深的嘱托。
