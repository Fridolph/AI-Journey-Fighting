# 否卦（䷋）· 六爻提示词

> 卦序：第 12 卦 &nbsp;|&nbsp; 卦象：乾上坤下 &nbsp;|&nbsp; 天地否 &nbsp;|&nbsp; 卦德：塞

否卦与泰卦是易经最经典的一对——泰是天地交，否是天地不交。乾上坤下，天气上升，地气下降，各走各路，万物不通。但否卦不是绝望，六爻藏着君子在否运中的生存智慧：**守贞→不乱群→包羞→志行→系苞桑→倾否**，是一部在闭塞中守正待时的图鉴。

## 初六 · 拔茅茹，以其夤，贞吉亨

```
易经爻辞知识图鉴
否卦 · 初六 · 拔茅茹，以其夤，贞吉亨

【画面主体意象】
同类守贞，志在君也——
否卦初六与泰卦初九
同样是「拔茅茹，以其夤」，
但气质截然不同——
泰卦是「征吉」，
志在外，出发；
否卦是「贞吉」，
志在君，守正。

画面主体：
同样是茅草，
根系相连，
但这一次，
茅草不是被拔起出发，
而是根扎得更深——
「贞吉」，
守正，
是否运中
最重要的动作。

泰否对比的意象：
泰卦的茅草，
被拔起，向上，
生机勃勃；
否卦的茅草，
根扎更深，
向下，
在闭塞的天地里，
守住自己。

「志在君」：
不是出征，
是守望——
守望着君，
守望着正道，
在否运中，
这是最难，
也最重要的事。

背景：
天地不交——
天，往上走；
地，往下走；
两者之间，
是闭塞的空间，
但茅草的根，
在这片土地里，
依然相连。

天色：
比泰卦初九暗，
没有春日的生机，
但茅草，
依然在。

【几何爻位结构】
否卦六爻从下到上
（乾上坤下）：
初六 ━ ━  高亮深赭内守 ← 当前爻
六二 ━ ━
六三 ━ ━
九四 ━━
九五 ━━
上九 ━━
标注：「坤下乾上，天地不交」

【嵌入式大字】
「初六」嵌入茅草根系
几何结构，
字色用深赭，
根系向下扎，
字，也向下沉——
「贞」，
是向下扎根的力量，
不是向上生长的冲动。

【笔刷动线】
一条向下扎根
而非向上拔起的笔刷，
模拟「贞」的方向——
泰卦是向上，
否卦是向下，
同样的茅草，
两个方向，
两种智慧。

【信息模块】
① 否卦 · 初六（右上角）
② 卦象标注：地天否 · 乾上坤下
③ 爻辞：「拔茅茹，以其夤，贞吉亨。」
④ 小象传：「拔茅贞吉，志在君也。」
⑤ 象传节录：「天地不交，否；
   君子以俭德辟难，
   不可荣以禄。」
⑥ 彖传节录：「天地不交，
   而万物不通也；
   上下不交，而天下无邦也。」
⑦ 爻位说明：阴爻居初，
   坤卦之始，
   否运之初，
   同类相守，
   志在君，
   贞吉。
⑧ 核心义理：否运的开始，
   不是出发，
   是守正。
   同样的茅草，
   泰时征吉，
   否时贞吉——
   时不同，
   道不同。
⑨ 角落批注：
   「志在君——
    否运中，
    守望，
    是最有力量的行动。」
⑩ 收尾精神之语：
   「泰时，拔茅出征；
    否时，扎根守贞。
    同样的根，
    知道什么时候
    该往哪里用力，
    这是智慧。」

【配色】
深赭·内守为主，
茅草用深赭，
根系向下用深褐，
天色用闭塞深灰蓝，
土地用沉厚深赭，
整体内敛、沉稳、向下扎根，
与泰卦初九的嫩绿向上
形成鲜明对比，
是否卦六爻中
最「守」的一张，
守，是否运的第一课。

现代平面设计结构。
```

## 六二 · 包承，大人否亨

```
易经爻辞知识图鉴
否卦 · 六二 · 包承，大人否亨

【画面主体意象】
不乱群，否中自亨——
六二是否卦最「清醒」的一爻，
「包承」——
包容承受，
小人在否运中得吉，
但「大人否亨」——
大人，
在否运中，
不随波逐流，
「不乱群也」——
不与小人混同，
自有其亨。

画面主体：
一片混浊的水流，
代表否运中
小人得势的环境——
但画面中心，
有一块清澈的石，
在混浊中
保持自身的清洁——
「大人否亨」，
不是逃离，
是在混浊中
守住清澈。

「不乱群」的意象：
石，在水中，
水流过石，
石不随水走——
「不乱群」，
就是这块石
在水中的姿态。

「包承」：
承受，
是否运中
必须面对的现实——
但承受，
不等于同流，
大人，
承受了，
但没有乱。

画面光线：
极暗，
但石的表面，
有一点自己的光——
「否亨」，
亨，是内在的，
不依赖外部的光。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  高亮深墨持守 ← 当前爻
六三 ━ ━  （淡）
九四 ━━   （淡）
九五 ━━   （淡）
上九 ━━   （淡）
标注：「大人否亨，不乱群也」

【嵌入式大字】
「六二」嵌入清石
几何结构，
字色用深墨，
在混浊水流中
清晰可见——
「不乱群」，
字，在混浊里，
依然是自己的。

【笔刷动线】
混浊的水流笔刷
从四面涌来，
但在中心的石处
自然分开，
绕石而过——
「不乱群」，
就是这块石
在水流中
保持自身的样子。

【信息模块】
① 否卦 · 六二（右上角）
② 爻辞：「包承。小人吉，大人否亨。」
③ 小象传：「大人否亨，不乱群也。」
④ 爻位说明：阴爻居二，
   得中，
   否运中守正，
   不随小人，
   自有其亨。
⑤ 核心义理：否运中，
   最难的不是承受，
   是在承受中
   不失去自己。
   「不乱群」——
   三个字，
   是否卦最难的功课。
⑥ 角落批注：
   「不乱群——
    混浊的水，
    流过清澈的石，
    石，还是石。」
⑦ 收尾精神之语：
   「小人在否运中得吉，
    大人在否运中自亨——
    不是因为大人更强，
    是因为大人
    知道自己是谁，
    不乱。」

【配色】
深墨·持守为主，
混浊水流用深灰蓝，
清石用深墨，
石上自光用极淡冷白，
背景用压抑深灰，
整体沉郁、清醒、有内守感，
是否卦六爻中
最「清」的一张，
清，在混浊中，
才显出价值。

现代平面设计结构。
```

## 六三 · 包羞

```
易经爻辞知识图鉴
否卦 · 六三 · 包羞

【画面主体意象】
位不当，包藏羞耻——
六三是否卦最沉郁的一爻，
「包羞」——
只有两个字，
包藏着羞耻，
「位不当也」——
位置不对，
所做的事，
只能包藏羞耻。

画面主体：
一个紧闭的容器，
或一个被遮盖的东西——
「包」，
是包藏，
是遮掩，
「羞」，
是羞耻，
是不能见人的东西。

「包羞」的意象：
不直接画羞耻，
用一个
被紧紧包裹的包袱，
或一个
被遮盖的器物——
里面是什么，
不说，
但「包」这个动作，
本身就说明了一切。

位不当的意象：
包袱，
放在了不该放的地方，
或器物，
用在了不该用的场合——
「位不当」，
是这一切的根本原因。

画面整体：
极简，
极压抑，
没有出路，
没有光——
「包羞」，
是否卦六爻中
最沉默的两个字。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  高亮暗灰沉郁 ← 当前爻
九四 ━━   （淡）
九五 ━━   （淡）
上九 ━━   （淡）
标注：「包羞，位不当也」

【嵌入式大字】
「六三」嵌入包裹
几何暗块，
字色用暗灰，
被包裹的遮盖
深度遮挡，
字，几乎看不见——
「包羞」，
连字，都想藏起来。

【笔刷动线】
一条向内收缩、
向内包裹的笔刷，
不向外延伸，
只是收，
收，
收——
「包」，
就是这个
不断向内收缩的动作。

【信息模块】
① 否卦 · 六三（右上角）
② 爻辞：「包羞。」
③ 小象传：「包羞，位不当也。」
④ 爻位说明：阴爻居三，
   失位不中，
   以柔居刚，
   否运中位不当，
   所为可羞，
   只能包藏。
⑤ 核心义理：否卦最短的爻辞，
   也是最沉重的——
   「包羞」，
   不是因为做了坏事，
   是因为位不当，
   做了不该做的事，
   只能包藏。
⑥ 角落批注：
   「位不当——
    位，是一切的根本。
    位不当，
    做什么都是包羞。」
⑦ 收尾精神之语：
   「包羞，
    不是终点，
    是一个提醒——
    位不当，
    先正位，
    再行事。」

【配色】
暗灰·沉郁为主，
包裹用深暗灰，
遮盖用深墨，
背景用最压抑的深灰，
无光，无出路，
整体沉默、压抑、内收，
是否卦六爻中
最「暗」的一张，
暗，是位不当的颜色。

现代平面设计结构。

## 九四 · 有命无咎，畴离祉

```
易经爻辞知识图鉴
否卦 · 九四 · 有命无咎，畴离祉

【画面主体意象】
志行得命，同类得福——
九四是否卦第一道转机，
从坤入乾，
阴转阳，
「有命无咎」——
有天命，无咎，
「畴离祉」——
同类，
都得到了福祉，
「志行也」——
志，开始行了。

画面主体：
一道破云而出的光——
不是满天的光，
是第一道，
从否运的厚云中
透出的光——
「有命」，
就是这道光，
是天命的信号。

「畴离祉」的意象：
几个身影，
在这道光下，
聚在一起——
「畴」，是同类，
「离祉」，是附丽于福祉，
同类，
在这道光下，
找到了彼此。

转机的质感：
不是突然的大晴天，
是第一道光——
否，还没有结束，
但转机，
已经开始了。

画面构图：
上方是厚重的否云，
中间是那道光，
下方是聚集的同类——
「志行」，
志，终于开始走了。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  （淡）
九四 ━━   高亮深青转暖 ← 当前爻
九五 ━━   （淡）
上九 ━━   （淡）
标注：「有命无咎，志行也」

【嵌入式大字】
「九四」嵌入破云光柱
几何结构，
字色用深青，
被那道光
正中穿过——
「有命」，
字，被光照亮了
一半。

【笔刷动线】
一条从厚云中
破出而来的笔刷，
从上方深暗处
向下透出，
在聚集的同类处
展开——
「畴离祉」，
就是这道光
照到同类的那一刻。

【信息模块】
① 否卦 · 九四（右上角）
② 爻辞：「有命无咎，畴离祉。」
③ 小象传：「有命无咎，志行也。」
④ 爻位说明：阳爻居四，
   由坤入乾，
   否运转机，
   有天命，
   同类得福，
   志行。
⑤ 核心义理：否运的转机，
   不是突然的，
   是「有命」——
   天命来了，
   志，才能行。
   「畴离祉」——
   一个人的转机，
   带动了同类的福祉。
⑥ 角落批注：
   「志行——
    否运中守了那么久，
    志，终于可以走了。」
⑦ 收尾精神之语：
   「那道光，
    不是终点，
    是开始——
    否，还没完，
    但志，
    已经在走了。」

【配色】
深青·转暖为主，
厚云用深暗灰蓝，
破云之光用冷白金，
同类身影用深墨，
聚集处用暖赭，
整体从暗到有光，
有明显的转机感，
是否卦六爻中
第一张有「光」的，
光，是志行的颜色。

## 九五 · 休否，大人吉，其亡其亡，系于苞桑

```
易经爻辞知识图鉴
否卦 · 九五 · 休否，大人吉，其亡其亡，系于苞桑

【画面主体意象】
系于苞桑，警醒中的大吉——
九五是否卦最深刻的一爻，
「休否」——
否运，休止了，
「大人吉」——
大人，吉，
但最重要的是：
「其亡其亡，系于苞桑」——
时时警惕「要亡了，要亡了」，
把自己系在苞桑（丛生的桑树）上，
才能真正稳固。

画面主体：
一棵丛生的桑树——
「苞桑」，
根系极深，
枝干丛生，
是最稳固的植物意象。

「系于苞桑」的意象：
一根绳，
或一条藤，
把某个重要的东西
系在桑树的根部——
不是束缚，
是「系」——
主动地把自己
系在最稳固的地方。

「其亡其亡」：
画面一角，
有极淡的否运阴云，
还没有完全散去——
大人，
在吉的时候，
依然警惕着
「要亡了」——
这种警惕，
才是真正的吉。

「休否」：
否，在休止，
但不是消失——
桑树周围，
否云在退，
光，在进，
但大人，
依然系着桑树，
不放松。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  （淡）
九四 ━━   （淡）
九五 ━━   高亮苍青桑绿 ← 当前爻
上九 ━━   （淡）
标注：「位正当也，系于苞桑」

【嵌入式大字】
「九五」嵌入苞桑
几何根系结构，
字色用苍青，
被桑树根系
部分遮挡——
「系于苞桑」，
字，也系在根上，
稳，是字的气质。

【笔刷动线】
一条从桑树根系
向外延伸，
但在根处
有一个明显的
「系」的动作——
向外延伸，
但根，
牢牢系住——
「其亡其亡，系于苞桑」，
就是这个
延伸而不离根的形状。

【信息模块】
① 否卦 · 九五（右上角）
② 爻辞：「休否，大人吉。
   其亡其亡，系于苞桑。」
③ 小象传：「大人之吉，位正当也。」
④ 彖传节录：「内小人而外君子，
   小人道长，君子道消也。」
⑤ 爻位说明：阳爻居五，
   刚健中正，
   否运将休，
   大人居正位，
   警惕自守，
   系于苞桑，
   大吉。
⑥ 核心义理：「其亡其亡」——
   在最吉的时候，
   依然警惕着「要亡了」，
   这是否卦送给大人
   最重要的功课。
   系于苞桑，
   不是悲观，
   是清醒。
⑦ 角落批注：
   「其亡其亡——
    越是吉的时候，
    越要说这四个字。
    这是易经
    最深的警醒。」
⑧ 收尾精神之语：
   「否，休了，
    大人，吉了，
    但大人说：
    其亡其亡——
    系在桑树上，
    不是因为怕，
    是因为，
    知道吉，
    有多脆弱。」

【配色】
苍青·桑绿为主，

## 上九 · 倾否，先否后喜

```
易经爻辞知识图鉴
否卦 · 上九 · 倾否，先否后喜

【画面主体意象】
否终则倾，先否后喜——
上九是否卦最有希望的结尾，
「倾否」——
否运，倾覆了，
「先否后喜」——
先是否，
后是喜，
「否终则倾，何可长也」——
否，不会永远，
终究会倾覆。

画面主体：
一个倾覆的意象——
否运的象征物，
正在倾倒——
不是被外力打倒，
是「否终则倾」——
否，走到了尽头，
自然倾覆。

「倾」的方向：
向下，向外，
否运的重量，
在这一刻，
倒了——
倒，是解脱，
不是灾难。

「先否后喜」：
画面构图，
左侧是否运的深暗，
右侧是喜的光——
不是突然的大晴天，
是否倾之后，
光，自然透进来。

破晓的意象：
否云倾倒，
露出了后面的天——
「后喜」，
就是那片天，
一直在否云后面，
等着。

画面整体：
有一种
「终于」的感觉——
不是胜利的喧嚣，
是「终于」的轻——
否，走完了，
喜，来了。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
六二 ━ ━  （淡）
六三 ━ ━  （淡）
九四 ━━   （淡）
九五 ━━   （淡）
上九 ━━   高亮深墨破晓 ← 当前爻
标注：「否终则倾，何可长也」

【嵌入式大字】
「上九」嵌入倾覆
几何结构，
字的左半在否运的深暗里，
右半在破晓的光里——
「先否后喜」，
字，也经历了这个
从暗到明的过程。

【笔刷动线】
一条从左侧深暗处
向右侧倾倒，
在倾覆处
散开，
露出右侧的光——
「倾否」，
就是这个
倒下去，
光进来的动作。

【信息模块】
① 否卦 · 上九（右上角）
② 爻辞：「倾否，先否后喜。」
③ 小象传：「否终则倾，何可长也。」
④ 彖传节录：「大往小来，
   则是天地不交，
   而万物不通也。」
⑤ 爻位说明：阳爻居极上，
   否之终，
   否极必倾，
   先否后喜，
   天地规律。
⑥ 核心义理：否卦最美的结尾——
   「何可长也」，
   否，不会永远，
   这是易经对所有
   在否运中守正的人
   最后的承诺。
⑦ 角落批注：
   「何可长——
    否，不会永远。
    这四个字，
    是否卦
    最温暖的话。」
⑧ 收尾精神之语：
   「否，倒了，
    不是因为你打倒了它，
    是因为它，
    走到了尽头。
    先否后喜——
    喜，一直在那里，
    等否，走完。」

【配色】
深墨·破晓为主，
左侧否运用最深玄墨，
倾覆处用深灰，
右侧破晓用冷白渐暖，
喜的光用极淡暖金，
整体从最深的暗
到第一道破晓之光，
是否卦六爻中
最有「希望」的一张——
希望，在倾覆之后，
自然透进来。
