# 大壮卦（䷡）· 六爻提示词

> 卦序：第 34 卦 | 卦象：震上乾下 | 雷天大壮 | 卦德：壮

大壮是易经最「壮」的一卦——雷在天上，刚以动，故壮。「正大而天地之情可见矣」——壮要正，正大才能见天地之情。六爻：**壮于趾→贞吉中→羝羊触藩→藩决壮舆→丧羊于易→不能退遂**。

## 大壮「正大」的深层结构

> 趾壮急进（初九↓）→守中贞吉（九二◎）→触藩羸角（九三↓）→藩决壮进（九四☀️）→丧羊位失（六五）→两难艰吉（上六↑）
> 壮要正，正才能大。非礼勿履是壮的约束，守中是壮的方向。

## 遁与大壮的深层对照

| | 遁卦 | 大壮卦 |
|--|------|--------|
| **核心** | 遁而亨，与时行 | 大者正也，正大 |
| **最深一句** | 遁之时义大矣哉 | 正大而天地之情可见矣 |

遁是退，大壮是进。退得自在，进得正大。

> 「君子以非礼勿履」——壮若无礼的约束，就会触藩羸角。非礼勿履，守中，正大才能见天地之情。
