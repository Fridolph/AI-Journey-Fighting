# 大壮卦（䷡）· 六爻提示词

> 卦序：第 34 卦 &nbsp;|&nbsp; 卦象：震上乾下 &nbsp;|&nbsp; 雷天大壮 &nbsp;|&nbsp; 卦德：壮

大壮——大者壮也。刚以动，故壮。「正大而天地之情可见矣！」——这是易经中最「刚健壮大」的一卦。

## 大壮卦「正大而天地之情可见矣」的深层结构

> 壮趾征凶孚穷（初九）→居中贞吉（九二）→触藩羸角君子罔（九三）→藩决不羸尚往（九四）→丧羊无悔位不当（六五）→触藩艰则吉咎不长（上六◯）
> 羝羊触藩意象贯穿九三与上六——九三触藩羸角壮而受阻，上六触藩不退遂艰则吉，同一意象截然不同结局。

## 遁与大壮的深层对照

| | 遁卦 | 大壮卦 |
|--|------|------|
| **核心** | 遁而亨，与时行也，肥遁无不利 | 刚以动故壮，正大而天地之情可见矣 |
| **最深一句** | 肥遁，无不利 | 贞吉，以中也 |

遁是隐退从容，大壮是刚健壮大。遁而后可知大壮——知退而后知进，遁与大壮是进与退的终极对照。

## 初九 · 壮于趾，征凶，有孚

```text
易经爻辞知识图鉴
大壮卦 · 初九 · 壮于趾，征凶，有孚

【画面主体意象】壮于趾，其孚穷也——初九是「趾壮」的一爻，「壮于趾」——壮盛于脚趾，「征凶」——前往则凶，
「有孚」——有诚信；「其孚穷也」——诚信穷尽。壮于趾不可轻易前往；大壮之道从知止开始。

【几何爻位结构】大壮卦从下到上（震上乾下）：初九 ━━  高亮深赭趾壮 ← 当前爻 九二 ━━ 九三 ━━ 九四 ━━ 六五 ━ ━ 上六 ━ ━
标注：「雷在天上，大壮；君子以非礼勿履」

【嵌入式大字】「初九」嵌入壮趾征凶几何结构，字色用深赭，被趾壮孚穷的沉重部分遮挡——
「壮于趾，征凶，有孚」，字，壮趾，孚穷，有大壮卦最初趾壮征凶孚穷警示。

【信息模块】① 大壮卦 · 初九（右上角）② 爻辞：「壮于趾，征凶，有孚。」③ 小象传：「壮于趾，其孚穷也。」
④ 核心义理：大壮起点警示——壮于趾不可轻易前往；大壮之道从知止开始。⑨ 收尾精神之语：「壮盛于脚趾——征凶孚穷；其孚穷也——壮趾不可轻往。」

【配色】深赭·趾壮为主，壮趾用深赭，征凶用暗红，有孚用暖金，孚穷用深灰，背景用深蓝雷天。
16:9横幅，宋代趾壮孚穷意境，现代平面设计结构。
```

## 九二 · 贞吉

```text
易经爻辞知识图鉴
大壮卦 · 九二 · 贞吉

【画面主体意象】九二贞吉，以中也——九二是「居中」的一爻，「贞吉」——守正则吉；「以中也」——以居中。
居中守正则吉——居中是大壮最深智慧。短短二字却是整卦最精华的刚中智慧。

【几何爻位结构】六爻：初九 ━━（淡）九二 ━━  高亮纯金居中 ← 当前爻 九三 ━━（淡）九四 ━━（淡）六五 ━ ━（淡）上六 ━ ━（淡）
标注：「九二贞吉，以中也」

【嵌入式大字】「九二」嵌入贞吉以中几何结构，字色用纯金，被居中贞吉的光芒四方普照——
「贞吉」，字，贞吉，以中，有大壮卦最深的居中贞吉。

【信息模块】① 大壮卦 · 九二（右上角）② 爻辞：「贞吉。」③ 小象传：「九二贞吉，以中也。」
④ 核心义理：大壮核心智慧——居中守正则吉。⑥ 收尾精神之语：「贞吉——以中也；居中守正是大壮最深智慧。」

【配色】纯金·居中为主，贞吉用纯金，以中用明金，背景用深蓝雷天。
16:9横幅，宋代居中贞吉意境，现代平面设计结构。
```

## 九三 · 小人用壮，君子用罔，羝羊触藩，羸其角

```text
易经爻辞知识图鉴
大壮卦 · 九三 · 小人用壮，君子用罔，羝羊触藩，羸其角

【画面主体意象】小人用壮，君子罔也——九三是「触藩」的一爻，「小人用壮」——小人使用壮力，「君子用罔」——君子不用壮力，
「羝羊触藩」——公羊触篱，「羸其角」——角被缠住。与上六同一意象「羝羊触藩」——九三触藩羸角壮而受阻。

【几何爻位结构】六爻：初九 ━━（淡）九二 ━━（淡）九三 ━━  高亮暗褐触藩 ← 当前爻 九四 ━━（淡）六五 ━ ━（淡）上六 ━ ━（淡）
标注：「小人用壮，君子罔也」

【嵌入式大字】「九三」嵌入触藩羸角几何结构，字色用暗褐，被触藩羸角的沉重部分遮挡——
「小人用壮，君子用罔，羝羊触藩，羸其角」，字，触藩，羸角，有大壮卦最深触藩羸角警示。

【信息模块】① 大壮卦 · 九三（右上角）② 爻辞：「羝羊触藩，羸其角。小人用壮，君子用罔。」
③ 小象传：「小人用壮，君子罔也。」④ 核心义理：大壮触藩之警——小人用壮则触藩羸角；君子用罔是大壮最深智慧。
⑧ 收尾精神之语：「小人用壮触藩羸角——君子用罔；羝羊触篱角被缠住是大壮最生动意象。」

【配色】暗褐·触藩为主，小人用壮用暗褐，君子用罔用深金，触藩羸角用暗红，背景用深蓝雷天。
16:9横幅，宋代触藩羸角意境，现代平面设计结构。
```

## 九四 · 贞吉悔亡，藩决不羸，壮于大舆之輹

```text
易经爻辞知识图鉴
大壮卦 · 九四 · 贞吉悔亡，藩决不羸，壮于大舆之輹

【画面主体意象】藩决不羸，尚往也——九四是「藩决」的一爻，「藩决不羸」——篱笆决开不被缠住，
「壮于大舆之輹」——壮盛于大车车轴，「尚往也」——尚可前往。与九三触藩羸角对比——九三壮而受阻，九四壮而通达。

【几何爻位结构】六爻：初九 ━━（淡）九二 ━━（淡）九三 ━━（淡）九四 ━━  高亮深金藩决 ← 当前爻 六五 ━ ━（淡）上六 ━ ━（淡）
标注：「藩决不羸，尚往也」

【嵌入式大字】「九四」嵌入藩决尚往几何结构，字色用深金，被藩决不羸尚往的刚健部分遮挡——
「贞吉悔亡，藩决不羸，壮于大舆之輹」，字，藩决，尚往，有大壮卦最深藩决不羸通达智慧。

【信息模块】① 大壮卦 · 九四（右上角）② 爻辞：「贞吉悔亡，藩决不羸，壮于大舆之輹。」
③ 小象传：「藩决不羸，尚往也。」④ 核心义理：大壮藩决之通——藩决不羸壮于大舆之輹尚往。与九三壮而受阻对比。
⑧ 收尾精神之语：「藩决不羸——壮于大舆之輹尚往；触藩有阻藩决则通——大壮最深的通达。」

【配色】深金·藩决为主，藩决不羸用深金，壮舆輹用明金，尚往用暖金，背景用深蓝雷天。
16:9横幅，宋代藩决通达意境，现代平面设计结构。
```

## 六五 · 丧羊于易，无悔

```text
易经爻辞知识图鉴
大壮卦 · 六五 · 丧羊于易，无悔

【画面主体意象】丧羊于易，位不当也——六五是「位不当」的一爻，「丧羊于易」——在易地失去羊，
「无悔」——无悔恨；「位不当也」——位置不当。位不当丧羊于易然而无悔——无悔是大壮最深的德性。

【几何爻位结构】六爻：初九 ━━（淡）九二 ━━（淡）九三 ━━（淡）九四 ━━（淡）六五 ━ ━  高亮暗金位当 ← 当前爻 上六 ━ ━（淡）
标注：「丧羊于易，位不当也」

【嵌入式大字】「六五」嵌入丧羊无悔几何结构，字色用暗金，被位不当无悔的暗淡部分遮挡——
「丧羊于易，无悔」，字，丧羊，无悔，有大壮卦最深丧羊位不当无悔。

【信息模块】① 大壮卦 · 六五（右上角）② 爻辞：「丧羊于易，无悔。」③ 小象传：「丧羊于易，位不当也。」
④ 核心义理：大壮位当之省——位不当丧羊于易然而无悔。⑧ 收尾精神之语：「丧羊于易——无悔；位不当也——位不当然而无悔是大壮最深的省。」

【配色】暗金·位当为主，丧羊用暗金，位不当用深灰，无悔用暖金，背景用深蓝雷天。
16:9横幅，宋代丧羊无悔意境，现代平面设计结构。
```

## 上六 · 羝羊触藩，不能退，不能遂，艰则吉

```text
易经爻辞知识图鉴
大壮卦 · 上六 · 羝羊触藩，不能退，不能遂，艰则吉

【画面主体意象】艰则吉，咎不长也——上六是「艰则吉」的一爻，「羝羊触藩」——公羊触篱进退两难，
「不能退不能遂」——不能后退不能前进，「艰则吉」——艰难则吉；「咎不长也」——咎害不长久。
与九三同一「羝羊触藩」——九三触藩羸角壮而受阻，上六触藩艰则吉咎不长，同一意象截然不同结局。

【几何爻位结构】六爻：初九 ━━（淡）九二 ━━（淡）九三 ━━（淡）九四 ━━（淡）六五 ━ ━（淡）上六 ━ ━  高亮暗红艰吉 ← 当前爻
标注：「不能退不能遂，艰则吉咎不长也」

【嵌入式大字】「上六」嵌入触藩艰吉几何结构，字色用暗红，被艰则吉咎不长的深意四方弥漫——
「羝羊触藩，不能退，不能遂，艰则吉」，字，触藩，艰吉，有大壮卦最终触藩艰则吉咎不长。

【信息模块】① 大壮卦 · 上六（右上角）② 爻辞：「羝羊触藩，不能退，不能遂，无攸利，艰则吉。」
③ 小象传：「艰则吉，咎不长也。」
④ 核心义理：大壮终极智慧——进退两难时艰则吉咎不长。与九三同一意象截然结局——大壮最精彩呼应。
⑨ 收尾精神之语：「触藩不退不遂——艰则吉咎不长；正大而天地之情可见矣。」

【配色】暗红·艰吉为主，触藩用暗红，不退遂用深灰，艰则吉用暖金，咎不长用纯金，背景用深蓝雷天。
16:9横幅，宋代触藩艰则吉意境，现代平面设计结构。
```

## 大壮卦六爻 · 完整刚健图鉴

```
初九 · 壮于趾      ——  征凶，孚穷
九二 · 贞吉        ——  以中，刚中核心
九三 · 触藩羸角    ——  用壮，君子罔
九四 · 藩决不羸    ——  尚往，壮舆輹
六五 · 丧羊无悔    ——  位不当，省察
上六 · 触藩艰吉    ——  不退遂，咎不长
```

> **「正大而天地之情可见矣！」**
> ——从初九「壮趾征凶孚穷」到上六「触藩艰则吉咎不长」，
> 大壮六爻是完整刚健壮大图鉴——羝羊触藩意象贯穿九三与上六：
> 九三触藩羸角壮而受阻，上六触藩艰则吉同一意象截然结局；
> 雷在天上大壮君子以非礼勿履。
