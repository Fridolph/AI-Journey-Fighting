# 蒙卦（䷃）· 六爻提示词

> 卦序：第 4 卦 &nbsp;|&nbsp; 卦象：艮上坎下 &nbsp;|&nbsp; 山水蒙 &nbsp;|&nbsp; 卦德：昧

蒙卦的气质很特别——不是难，是**昧**。屯是难中求生，蒙是昧中求明。山下有泉，泉刚出来，方向未定，需要引导。六爻是启蒙的六种面貌：**开蒙、包蒙、迷蒙、困蒙、童蒙、击蒙**，是一部师与生、明与昧之间的关系史。

## 初六 · 发蒙，利用刑人，用说桎梏

```
易经爻辞知识图鉴
蒙卦 · 初六 · 发蒙，利用刑人，用说桎梏

【画面主体意象】
开蒙之始，立规正法——
画面以山下泉水初涌
为整体背景意象，
泉水刚从山石缝隙中流出，
方向未定，
需要引导。

画面中心：
一副桎梏（脚镣手铐），
被打开，
放在地面上——
不是惩罚的终点，
是「用说桎梏」——
解开，
是为了让人知道
规矩的存在，
从而不再需要桎梏。

桎梏旁：
一块石碑或木牌，
刻着极简的法度符号，
「以正法也」——
不是暴力，
是让规矩可见，
让边界清晰。

地面：
山石纹理，
坚实，
暗示艮卦（山）的止与正，
泉水在石缝间初涌，
暗示坎卦（水）的险与动——
山下有险，
蒙之象。

画面光线：
从山顶透下，
是开蒙之光，
照在桎梏被解开的那一刻，
有一种「发」的感觉——
蒙，开始被打开了。

【几何爻位结构】
蒙卦六爻从下到上
（艮上坎下）：
初六 ━ ━  高亮石青 ← 当前爻
九二 ━━
六三 ━ ━
六四 ━ ━
六五 ━ ━
上九 ━━
标注：「坎下艮上，山下有险」

【嵌入式大字】
「初六」嵌入山石几何块面，
字体有石刻感、碑铭感，
被山石纹理部分切割，
字色用深石青，
与山石融为一体，
局部可见，整体可读。

【笔刷动线】
一条从山石缝隙
向下流出的笔刷，
模拟泉水初涌之势，
在桎梏处短暂停顿，
然后继续向下流动——
「发蒙」，
就是这股水，
开始找到方向的那一刻。

【信息模块】
① 蒙卦 · 初六（右上角）
② 卦象标注：山水蒙 · 艮上坎下
③ 爻辞：「发蒙，利用刑人，
   用说桎梏，以往吝。」
④ 小象传：「利用刑人，以正法也。」
⑤ 彖传节录：「蒙以养正，圣功也。」
⑥ 爻位说明：阴爻居初，
   蒙之始，
   发蒙须立规，
   规立则蒙可开，
   往而无规则吝。
⑦ 核心义理：开蒙的第一步，
   不是灌输知识，
   是让人知道边界在哪里。
   桎梏解开的那一刻，
   才是真正自由的开始。
⑧ 角落批注：「以正法也」
⑨ 收尾精神之语：
   「解开桎梏，
    不是因为没有规矩，
    是因为规矩，
    已经在心里了。」

【配色】
深墨石青为主，
山石用深灰青，
桎梏用冷铁灰，
泉水用极淡冷银，
开蒙之光用冷白，
整体严正、清醒、有力度，
是蒙卦六爻中
最「立规」的一张，
严，是为了之后的自由。

16:9横幅，宋代山石意境，
现代平面设计结构。
```

## 九二 · 包蒙吉，纳妇吉，子克家

```
易经爻辞知识图鉴
蒙卦 · 九二 · 包蒙吉，纳妇吉，子克家

【画面主体意象】
包容引导的温厚——
九二是蒙卦最温暖的一爻，
「包蒙」——包容蒙昧，
「纳妇」——接纳柔顺，
「子克家」——刚柔相接，
能成其家。

画面主体：
一棵大树，
树冠宽阔，
枝叶覆盖下方，
树下有几株幼苗——
大树不压幼苗，
是庇护，是包容，
「刚柔接也」。

幼苗的状态：
各有不同，
有的已经长出叶子，
有的还是嫩芽，
有的歪着，
大树的根系在地下
悄悄支撑着它们——
这是「包蒙」的视觉化，
不强求整齐，
但给予支撑。

树的周围：
温暖的光，
从侧面照入，
是午后的光，
不是正午的直射，
是包容的光——
有温度，不刺眼。

地面：
厚实的土，
暗示坤德与艮止，
根扎得深，
才能包容得广。

画面一角：
一个简单的家的轮廓，
极淡，极远，
「子克家」——
能包容，能接纳，
才能成家。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
九二 ━━   高亮暖赭 ← 当前爻
六三 ━ ━  （淡）
六四 ━ ━  （淡）
六五 ━ ━  （淡）
上九 ━━   （淡）
标注：「刚中，刚柔接也」

【嵌入式大字】
「九二」嵌入树冠的
几何圆弧结构，
字色用暖赭金，
被枝叶部分遮挡，
有一种「包」的感觉——
字在树冠里，
被包容着。

【笔刷动线】
一条从树干向外
自然扩散的弧形笔刷，
模拟树冠生长之势，
笔触温润，
有轻微晕染，
如包容本身——
不是直线，
是向四周自然延伸的弧。

【信息模块】
① 蒙卦 · 九二（右上角）
② 爻辞：「包蒙吉；纳妇吉；子克家。」
③ 小象传：「子克家，刚柔接也。」
④ 爻位说明：阳爻居二，
   刚中，
   蒙卦之主，
   以刚包柔，
   以中接众，
   是启蒙者的最佳姿态。
⑤ 核心义理：真正的启蒙，
   不是强迫，是包容。
   包蒙，是让蒙者
   在安全中慢慢开化。
⑥ 角落批注：
   「匪我求童蒙，
    童蒙求我——
    九二，就是那个
    值得被求的人。」
⑦ 收尾精神之语：
   「包容，
    不是纵容，
    是给蒙昧的人
    一个可以生长的空间。」

【配色】
暖赭厚土为主，
大树用深墨褐，
幼苗用嫩绿，
光线用温暖侧光赭黄，
地面用厚实深赭，
整体温暖、包容、有生机，
是蒙卦六爻中
最「暖」的一张，
刚而不硬，
柔而有根。

16:9横幅，宋代园林意境，
现代平面设计结构。
```

## 六三 · 勿用娶女，见金夫，不有躬，无攸利

```
易经爻辞知识图鉴
蒙卦 · 六三 · 勿用娶女，见金夫，不有躬，无攸利

【画面主体意象】
见利忘义的迷失——
六三是蒙卦最具警示性的一爻，
「见金夫，不有躬」——
见到有钱有势的人，
连自身都顾不上了，
「行不顺也」。

画面主体：
一条岔路，
一条路通向远处朴素的家园，
路面平实，有光；
另一条路通向
金光闪闪的方向，
路面华丽，但模糊——

画面中心：
一个身影，
正在向金光方向倾斜，
脚步已经迈出，
但身后的路，
已经开始模糊。

「金夫」的意象：
不画具体人物，
用一堆金器、金光，
或一面反光的铜镜，
代表那种
让人忘记自身的诱惑——
光太亮，
反而看不见自己。

地面：
在岔路口，
有一个脚印，
深深陷入泥中——
「不有躬」，
连自己的立足点，
都开始不稳了。

画面背景：
山水蒙的整体意象，
泉水在乱石中流淌，
方向混乱，
暗示「行不顺也」——
水，失去了方向。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
九二 ━━   （淡）
六三 ━ ━  高亮暗金 ← 当前爻
六四 ━ ━  （淡）
六五 ━ ━  （淡）
上九 ━━   （淡）
标注：「行不顺也，无攸利」

【嵌入式大字】
「六三」嵌入金光几何块，
字色用浮华的暗金，
有一种刺眼但不踏实的感觉——
金，但不正，
亮，但不实。

【笔刷动线】
一条从正道方向
急速转向金光方向的笔刷，
转折突然，
方向混乱，
在金光处散开，
没有落点——
「无攸利」，
冲过去，
什么都没抓到。

【信息模块】
① 蒙卦 · 六三（右上角）
② 爻辞：「勿用娶女；见金夫，
   不有躬，无攸利。」
③ 小象传：「勿用娶女，行不顺也。」
④ 爻位说明：阴爻居三，
   失位不中，
   见利忘义，
   不能自守，
   是蒙而不知蒙的状态。
⑤ 核心义理：最危险的蒙，
   不是不知道，
   是以为自己知道，
   却被表面的光迷住了。
⑥ 角落批注：
   「不有躬——
    连自己在哪里，
    都忘了。」
⑦ 收尾精神之语：
   「金光，
    不是方向，
    是让你忘记方向的东西。」

【配色】
暗金·浮华为主，
金光方向用刺眼的暗金黄，
正道方向用朴素的深赭，
身影用深墨，
地面用混乱的深褐，
整体有一种
华而不实的浮躁感，
是蒙卦六爻中
最「乱」的一张，
乱，是因为被光迷住了。

16:9横幅，宋代人物叙事意境，
现代平面设计结构。
```

## 六四 · 困蒙，吝

```
易经爻辞知识图鉴
蒙卦 · 六四 · 困蒙，吝

【画面主体意象】
孤处远实的困昧——
六四是蒙卦最孤寂的一爻，
「困蒙」——困在蒙昧里，
「独远实也」——
远离了实（九二的刚实），
孤立无援，
困而不得开。

画面主体：
一个封闭的空间——
四面是山石，
或四面是高墙，
中间是一个人的背影，
坐着，
四周没有窗，
没有光源，
只有极微弱的、
从石缝渗入的一点点光。

那点光：
极细，极淡，
照不亮整个空间，
只照亮了地面的一小块——
「独远实也」，
实（真知、明师）在远处，
这里，只有困。

空间的质感：
不是监狱，
是自己围起来的——
墙是自己的蒙昧，
门，其实有，
但被自己的困蒙挡住了，
看不见。

画面角落：
极淡的九二大树的轮廓，
极远，极小，
暗示「实」在那里，
但六四，
离得太远了。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
九二 ━━   （淡，但标注「实」）
六三 ━ ━  （淡）
六四 ━ ━  高亮深灰 ← 当前爻
六五 ━ ━  （淡）
上九 ━━   （淡）
标注：「独远实也，困于蒙」

【嵌入式大字】
「六四」嵌入封闭空间的
几何暗块，
字色与背景几乎同色，
只有那一缕石缝光
隐约照出字的轮廓——
困，连字，都快看不见了。

【笔刷动线】
一条极短的笔刷，
在封闭空间中心
按下，
四面都是墙，
笔触无处延伸，
只能在原地
形成一个小小的墨晕——
「困」，就是这个形状。

【信息模块】
① 蒙卦 · 六四（右上角）
② 爻辞：「困蒙，吝。」
③ 小象传：「困蒙之吝，独远实也。」
④ 爻位说明：阴爻居四，
   上下皆阴，
   远离九二之刚实，
   无援无应，
   困于蒙昧而不自知。
⑤ 核心义理：困蒙最难，
   不是没有出路，
   是不知道出路在哪里，
   甚至不知道自己困了。
⑥ 角落批注：
   「独远实——
    实，一直在那里，
    是自己走远了。」
⑦ 收尾精神之语：
   「困，
    不是因为没有光，
    是因为离光，
    太远了。」

【配色】
深灰·封闭为主，
四壁用最深的石灰墨，
石缝光用极细冷白，
人影用深墨，
远处九二轮廓用极淡暖赭，
整体是蒙卦六爻中
最「封闭」的一张，
留白最少，
压迫感最强，
困，是画面本身。

16:9横幅，宋代山洞意境，
现代平面设计结构。
```

## 六五 · 童蒙，吉

```
易经爻辞知识图鉴
蒙卦 · 六五 · 童蒙，吉

【画面主体意象】
虚心受教的纯真——
六五是蒙卦最「吉」的一爻，
「童蒙」——像孩子一样的蒙，
不是愚昧，
是虚心，是谦顺，
「顺以巽也」——
顺，是这一爻最美的品质。

画面主体：
山下的泉水，
刚刚从石缝中涌出，
清澈，
方向未定，
但流动的姿态，
是顺的——
顺着地势，
顺着引导，
自然流向该去的地方。

泉水旁：
一株幼苗，
或一个孩子的手印，
极简，极纯，
「童蒙」的意象——
不是无知，
是还没有被污染的
那种干净。

画面光线：
清晨的光，
柔和，
从山顶透下，
照在泉水上，
泉水反光，
整个画面因此明亮——
「童蒙之吉」，
吉，是因为顺，
顺，所以光能进来。

背景：
山（艮）的轮廓，
清晰，稳定，
是引导的存在，
泉水在山下，
顺着山的方向流——
这是「顺以巽」的
最准确的自然意象。

【几何爻位结构】
六爻从下到上：
初六 ━ ━  （淡）
九二 ━━   （淡）
六三 ━ ━  （淡）
六四 ━ ━  （淡）
六五 ━ ━  高亮清白 ← 当前爻
上九 ━━   （淡）
标注：「顺以巽也，童蒙之吉」

【嵌入式大字】
「六五」嵌入晨光几何层，
字色用清透的冷白，
被泉水和晨光部分遮挡，
整体明亮可读，
有一种「童」的纯净感——
字，是干净的。

【笔刷动线】
一条顺势而下的笔刷，
模拟泉水流动之势，
笔触轻盈，
有自然的弯曲，
不是直线，
是顺着地势的弧——
「顺以巽」，
就是这条线的样子。

【信息模块】
① 蒙卦 · 六五（右上角）
② 爻辞：「童蒙，吉。」
③ 小象传：「童蒙之吉，顺以巽也。」
④ 彖传节录：「蒙以养正，圣功也。」
⑤ 爻位说明：阴爻居五，
   以柔居尊，
   虚心下求九二，
   顺而能受，
   是蒙卦最吉之象。
⑥ 核心义理：童蒙之吉，
   不是因为无知，
   是因为知道自己不知道，
   并且愿意去学。
   谦顺，是开蒙最短的路。
⑦ 角落批注：
   「匪我求童蒙，
    童蒙求我——
    六五，就是那个
    主动来求的人。」
⑧ 收尾精神之语：
   「最聪明的蒙，
    是知道自己蒙，
    然后，顺着光，
    走过去。」

【配色】
清白·晨光为主，
泉水用极清透的冷蓝白，
山石用淡青灰，
晨光用温柔冷白，
幼苗用极嫩的浅绿，
整体是蒙卦六爻中
最「明亮」、最「纯净」的一张，
吉，是画面本身的感觉——
看见这张图，
就知道什么是「童蒙之吉」。

现代平面设计结构。

## 上九 · 击蒙，不利为寇，利御寇

```
易经爻辞知识图鉴
蒙卦 · 上九 · 击蒙，不利为寇，利御寇

【画面主体意象】
刚正击昧，上下顺也——
上九是蒙卦的收尾，
「击蒙」——以刚击昧，
不是暴力，
是在蒙昧走向极端时，
必要的纠正与阻止。

画面主体：
