# 离卦（䷝）· 六爻提示词

> 卦序：第 30 卦 &nbsp;|&nbsp; 卦象：离上离下 &nbsp;|&nbsp; 离为火 &nbsp;|&nbsp; 卦德：丽

离——丽也。日月丽乎天，百谷草木丽乎土。「重明以丽乎正，乃化成天下」——这是易经中最「光明附丽」的一卦。

## 离卦「重明以丽乎正乃化成天下」的深层结构

> 履错敬慎辟咎（初九）→黄离中道元吉（六二）→日昃鼓缶随遇（九三）→突如焚死弃无容（九四）→忧戚离王公吉（六五）→出征折首正邦（上九◯）
> 敬慎是光明起点，正邦是光明终极；六二黄离中道是离卦最圆满的光明。

## 坎与离的深层对照

| | 坎卦 | 离卦 |
|--|------|------|
| **核心** | 行险不失其信，水流而不盈 | 重明以丽乎正，日月丽乎天 |
| **最深一句** | 樽酒簋贰，用缶，纳约自牖 | 黄离，元吉 |

坎是重险叠加的守信，离是重明附丽的光明。涉险守信而后光明附丽——坎与离是易经中水火的终极对照，上经至此圆满。

## 初九 · 履错然，敬之无咎

```text
易经爻辞知识图鉴
离卦 · 初九 · 履错然，敬之无咎

【画面主体意象】履错之敬，以辟咎也——初九是「敬慎」的一爻，「履错然」——行步错杂纷乱，「敬之无咎」——敬慎则无咎；
「以辟咎也」。敬慎是离卦最初的处世智慧；光明从敬慎开始。

【几何爻位结构】离卦六爻从下到上（离上离下）：初九 ━━  高亮深赭敬慎 ← 当前爻 六二 ━ ━ 九三 ━━ 九四 ━━ 六五 ━ ━ 上九 ━━
标注：「明两作离；大人以继明照于四方」

【嵌入式大字】「初九」嵌入履错敬慎几何结构，字色用深赭，被敬慎辟咎的厚重部分遮挡——
「履错然，敬之无咎」，字，履错，敬之，有离卦最初的敬慎辟咎无咎。

【信息模块】① 离卦 · 初九（右上角）② 爻辞：「履错然，敬之无咎。」③ 小象传：「履错之敬，以辟咎也。」
④ 彖传节录：「重明以丽乎正，乃化成天下。」
⑤ 核心义理：离道起点——行步错乱敬慎则无咎；光明从敬慎开始。
⑨ 收尾精神之语：「行步错杂纷乱——敬之无咎；以辟咎也——敬慎是辟咎最深的智慧。」

【配色】深赭·敬慎为主，履错然用深赭，敬之用暗金，辟咎用暖金，无咎用纯金，背景用深红火明。
16:9横幅，宋代敬慎辟咎意境，现代平面设计结构。
```

## 六二 · 黄离，元吉

```text
易经爻辞知识图鉴
离卦 · 六二 · 黄离，元吉

【画面主体意象】黄离元吉，得中道也——六二是「中道」的一爻，「黄离」——黄色的附丽中央之色，「元吉」——大吉；「得中道也」。
柔丽乎中正是离卦最圆满的光明。黄离中道是易经对「中道」最简洁有力的赞美。

【几何爻位结构】六爻：初九 ━━（淡）六二 ━ ━  高亮纯金中道 ← 当前爻 九三 ━━（淡）九四 ━━（淡）六五 ━ ━（淡）上九 ━━（淡）
标注：「黄离元吉，得中道也」

【嵌入式大字】「六二」嵌入黄离中道几何结构，字色用纯金，被中道元吉的光芒四方普照——
「黄离，元吉」，字，黄离，中道，有离卦最深的黄离中道元吉。

【信息模块】① 离卦 · 六二（右上角）② 爻辞：「黄离，元吉。」③ 小象传：「黄离元吉，得中道也。」
④ 核心义理：离道中道之吉——黄离是中央之色柔丽乎中正；得中道则元吉。
⑧ 收尾精神之语：「黄色附丽——元吉；得中道也——柔丽乎中正是最圆满的光明。」

【配色】纯金·中道为主，黄离用纯金，中道用明金，元吉用暖金，背景用深红火明。
16:9横幅，宋代黄离中道意境，现代平面设计结构。
```

## 九三 · 日昃之离，不鼓缶而歌，则大耋之嗟，凶

```text
易经爻辞知识图鉴
离卦 · 九三 · 日昃之离，不鼓缶而歌，则大耋之嗟，凶

【画面主体意象】日昃之离，何可久也——九三是「日昃」的一爻，「日昃之离」——日偏西的光明将逝，
「不鼓缶而歌」——不能随遇而安，「则大耋之嗟凶」——老年悲嗟则凶；「何可久也」。日昃之时应鼓缶而歌随遇而安。

【几何爻位结构】六爻：初九 ━━（淡）六二 ━ ━（淡）九三 ━━  高亮暗金日昃 ← 当前爻 九四 ━━（淡）六五 ━ ━（淡）上九 ━━（淡）
标注：「日昃之离，何可久也」

【嵌入式大字】「九三」嵌入日昃鼓缶几何结构，字色用暗金，被日昃将逝的暗淡部分遮挡——
「日昃之离，不鼓缶而歌，则大耋之嗟，凶」，字，日昃，大耋，有离卦最深日昃将逝警示与随遇安智慧。

【信息模块】① 离卦 · 九三（右上角）② 爻辞：「日昃之离，不鼓缶而歌，则大耋之嗟，凶。」
③ 小象传：「日昃之离，何可久也。」
④ 核心义理：离道日昃之警——日偏西时鼓缶而歌随遇而安；不随遇则老年悲嗟凶。
⑥ 收尾精神之语：「日偏西光明将逝——不击缶而歌则老年悲嗟凶；何可久也——鼓缶而歌最豁达。」

【配色】暗金·日昃为主，日昃离用暗金，不鼓缶用深赭，大耋嗟用深灰，凶用暗红，背景用深红火明渐暗。
16:9横幅，宋代日昃将逝意境，现代平面设计结构。
```

「出涕沱若，戚嗟若，吉」，字，出涕，离王公，有离卦最深的忧戚悲嗟离王公吉。

【信息模块】① 离卦 · 六五（右上角）② 爻辞：「出涕沱若，戚嗟若，吉。」③ 小象传：「六五之吉，离王公也。」
