# 信息图

将知识、数据、历史、建筑等信息以图形化方式呈现，兼具知识性和审美性。本分类下的提示词专注于"信息组织 + 视觉呈现"的双重能力。

## 包含提示词

| 编号 | 名称 | 核心能力 |
|------|------|---------|
| 01 | [进化史信息图](./evolution-history) | 任意主题的进程演化信息图 |
| 02 | [博物馆国宝/文物解读图鉴](./museum-artifact) | 文物知识海报与展览级单页 |
| 03 | [科学百科信息图](./science-encyclopedia) | 经典科学百科全书风格插画 |
| 04 | [景点建筑信息图](./landmark-architecture) | 建筑照片 + 技术蓝图叠加 |

## 设计要点

- **智能补全内容**：根据主题自动延展阶段/知识点，不需用户提供全部细节
- **强组织性结构**：采用台阶、地层、谱系等空间化推进路径
- **配色因题而变**：根据主题语义、时代属性灵活匹配色彩系统
- **图文深度融合**：不是背景图+贴字，而是完整知识可视化系统
